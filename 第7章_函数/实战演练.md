## 六、程序设计题（共25分）

### 题目1：计算平均值（10分）

编写一个C程序，定义一个函数 `average`，接收一个整型数组和数组长度，计算并返回数组元素的平均值。

**要求：**
- 使用数组名作函数参数
- 在 `main` 函数中输入数组元素
- 输出平均值，保留2位小数

**输入样例：**
```
请输入元素个数：5
请输入5个整数：10 20 30 40 50
```

**输出样例：**
```
平均值为：30.00
```

**参考代码：**

```c
#include <stdio.h>

float average(int arr[], int n)
{
    int i;
    float sum = 0;
    for (i = 0; i < n; i++)
    {
        sum += arr[i];
    }
    return sum / n;
}

int main()
{
    int n, i;
    int arr[100];
    printf("请输入元素个数：");
    scanf("%d", &n);
    printf("请输入%d个整数：", n);
    for (i = 0; i < n; i++)
    {
        scanf("%d", &arr[i]);
    }
    printf("平均值为：%.2f\n", average(arr, n));
    return 0;
}
```


### 题目2：递归求斐波那契数列（15分）

编写一个C程序，使用递归函数求斐波那契数列的第 n 项。

**要求：**
- 斐波那契数列：第1项=1，第2项=1，从第3项起每项等于前两项之和
- 使用递归函数实现
- 输入项数 n，输出第 n 项的值

**输入样例：**
```
请输入项数：7
```

**输出样例：**
```
第7项 = 13
```

**参考代码：**

```c
#include <stdio.h>

int fib(int n)
{
    if (n == 1 || n == 2)
        return 1;
    else
        return fib(n - 1) + fib(n - 2);
}

int main()
{
    int n;
    printf("请输入项数：");
    scanf("%d", &n);
    printf("第%d项 = %d\n", n, fib(n));
    return 0;
}
```

### 五、程序改错题

| 错误序号 | 错误代码位置 | 错误原因 | 正确写法 |
| :---: | :--- | :--- | :--- |
| 1 | `int max = getmax(a, b);` | 调用函数时getmax未声明 | 在main前添加函数声明或把getmax定义移到main前 |
