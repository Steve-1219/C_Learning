# 7.3 数组作为函数参数

## 🎯 学习目标

- 掌握数组元素作为函数实参的使用方法
- 掌握一维数组名作为函数参数的使用方法
- 掌握多维数组名作为函数参数的使用方法
- 理解数组名作函数参数与变量作函数参数的区别


## 📖 知识梳理

### 7.3.1 数组元素作函数实参 

#### 一、基本概念

实参可以是**常量、变量或表达式**。数组元素也可以用作函数实参，其用法与变量相同，将数组元素的值传递给形参。

> 💡 **名师点金**：数组元素可以用作函数**实参**，但不能用作函数**形参**。因为形参是在函数调用时临时分配存储单元，而数组元素是数组的一部分，数组在内存中占连续的一段存储单元。

**数据传递方式：**

用数组元素作函数实参时，把实参的值传给形参，是**"值传递"**方式，数据传递的方向是**从实参传到形参**，**单向传递**。

#### 二、应用实例

**【例6】输入10个数，要求输出其中值最大的元素和该数的位置。**

**算法分析：**
1. 定义一个数组 `a[10]`，存放输入的10个数
2. 设计一个函数 `max`，用来求两个数中的较大者
3. 在主函数中定义变量 `m`，初值为 `a[0]`
4. 依次将 `a[1]` 到 `a[9]` 与 `m` 比较，每次调用 `max` 函数更新 `m` 的值
5. 最后得到的 `m` 值就是10个数中的最大者

**程序代码：**

```c
#include <stdio.h>

int main()
{
    int max(int x, int y);    // 函数声明
    int a[10], m, n, i;
    
    printf("请输入10个数：");
    for (i = 0; i < 10; i++)
    {
        scanf("%d", &a[i]);
    }
    
    for (i = 1, m = a[0], n = 0; i < 10; i++)
    {
        if (max(m, a[i]) > m)
        {
            m = max(m, a[i]);    // 用数组元素a[i]作实参
            n = i;
        }
    }
    
    printf("最大的数是%d，这是第%d个数。\n", m, n + 1);
    return 0;
}

int max(int x, int y)
{
    return (x > y ? x : y);
}
```

**运行结果：**

```
请输入10个数：12 34 56 23 67 45 18 29 38 47
最大的数是67，这是第5个数。
```

**程序分析：**

| 步骤 | 操作 | 说明 |
| :---: | :--- | :--- |
| ① | 输入10个数存入数组 `a[0]` ~ `a[9]` | 使用循环为数组赋值 |
| ② | `m = a[0]`，`n = 0` | m记录最大值，n记录最大值的位置 |
| ③ | 循环比较 `a[1]` ~ `a[9]` | 依次与当前最大值m比较 |
| ④ | `max(m, a[i])` | 用数组元素作实参调用max函数 |
| ⑤ | 更新最大值和位置 | 如果 `a[i] > m`，更新m和n |
| ⑥ | 输出结果 | 输出最大值和对应的位置（n+1） |


### 7.3.2 数组名作函数参数 

#### 一、一维数组名作函数参数

> 💡 **拿分笔记**：用数组名作函数参数时，不是把数组元素的值传递给形参，而是把**实参数组的首元素的地址**传递给形参数组名（或指针变量），传递的是数组首元素的地址。数组名既可作函数**实参**，也可作函数**形参**。

**重要区别：**

| 对比项 | 数组元素作实参 | 数组名作实参 |
| :--- | :--- | :--- |
| **传递内容** | 数组元素的值 | 数组首元素的地址 |
| **传递方式** | 值传递（单向） | 地址传递 |
| **形参类型** | 普通变量 | 数组名或指针变量 |
| **形参改变是否影响实参** | 不影响 | **影响**（共占同一存储单元） |

**【例7】有一个一维数组score内放置10个学生的成绩，求平均成绩。**

用数组名作为函数实参，在average函数中引用各数组元素，求平均成绩并返回main函数。

**程序代码：**

```c
#include <stdio.h>

int main()
{
    float average(float array[]);    // 函数声明
    float score[10], aver;
    int i;
    
    printf("输入这10个学生的成绩：\n");
    for (i = 0; i < 10; i++)
    {
        scanf("%f", &score[i]);
    }
    
    aver = average(score);           // 以数组名score为实参
    printf("平均分是%5.2f\n", aver);
    return 0;
}

float average(float array[])         // 形参数组不指定大小
{
    int i;
    float aver, sum = array[0];
    for (i = 1; i < 10; i++)
    {
        sum = sum + array[i];
    }
    aver = sum / 10;
    return (aver);
}
```

**运行结果：**

```
输入这10个学生的成绩：
100 56 78 98 67.5 99 54 88.5 76 58
平均分是77.50
```

**程序分析：**

| 要点 | 说明 |
| :--- | :--- |
| ① | 实参数组与形参数组**类型一致**，否则结果出错 |
| ② | 定义average函数时声明数组大小**不起作用**，编译系统不检查形参数组大小 |
| ③ | 形参数组名获得实参数组的**首元素地址**，两者**共占同一存储单元** |
| ④ | `score[n]` 和 `array[n]` 指向同一单元，具有**相同的值** |
| ⑤ | 形参数组可以**不指定大小**，使用空方括号即可 |

**数据传递示意图：**

```
┌─────────────────────────────────────────────────────────────┐
│                    数据传递示意图                           │
│                                                             │
│   实参数组 score           形参数组 array                   │
│                                                             │
│   score[0] ←─────────────→ array[0]   (同一存储单元)       │
│   score[1] ←─────────────→ array[1]   (同一存储单元)       │
│   score[2] ←─────────────→ array[2]   (同一存储单元)       │
│      ...                           ...                       │
│                                                             │
│   ⚠️ 形参数组元素改变 → 实参数组元素同时改变               │
│   ⚠️ 传递的是地址，而非值                                   │
└─────────────────────────────────────────────────────────────┘
```

**【例8】求两个不同长度的数组的平均值。**

定义average函数时**不指定数组的长度**，增加一个整型变量 `n` 从实参传递数组长度。

**程序代码：**

```c
#include <stdio.h>

int main()
{
    float average(float array[], int n);    // 函数声明
    float score1[5] = {98.5, 97, 91, 51.6, 55};
    float score2[10] = {67.5, 89.5, 99, 69.5, 77, 89.5, 76.5, 54, 60, 99.5};
    
    printf("A班的平均分是%.2f\n", average(score1, 5));
    printf("B班的平均分是%.2f\n", average(score2, 10));
    return 0;
}

float average(float array[], int n)
{
    int i;
    float aver, sum = array[0];
    for (i = 1; i < n; i++)
    {
        sum = sum + array[i];
    }
    aver = sum / n;
    return (aver);
}
```

**运行结果：**

```
A班的平均分是78.62
B班的平均分是77.20
```

**程序分析：**

| 要点 | 说明 |
| :--- | :--- |
| ① | 形参增加 `int n` 用于传递数组长度 |
| ② | 第1次调用：实参 `score1` 和 `5`，循环4次，求5个数的平均 |
| ③ | 第2次调用：实参 `score2` 和 `10`，循环9次，求10个数的平均 |
| ④ | **同一个函数**可以处理**不同长度**的数组 |


### 7.3.3 多维数组名作函数参数

可以用多维数组名作为函数的实参和形参，在被调用函数中对形参数组声明时可以**指定每一维的大小**，也可以**省略第一维的大小说明**。

**合法声明：**

```c
int array[3][10];     // 指定所有维的大小
int array[][10];      // 省略第一维大小（合法）
```

**非法声明：**

```c
int array[][];        // 错误：不能省略第二维大小
int array[3][];       // 错误：不能省略第二维大小
```

> ⚠️ **重要**：**第二维以及其他高维的大小说明不能省略**。因为系统需要知道每行有多少列才能正确计算元素的位置。


### 7.3.4 数组名作函数参数总结

| 对比项 | 数组元素作实参 | 数组名作实参 |
| :--- | :--- | :--- |
| **传递内容** | 数组元素的值 | 数组首元素的**地址** |
| **传递方式** | 值传递（单向） | 地址传递 |
| **形参** | 普通变量 | 数组名或指针 |
| **实参与形参是否共占存储单元** | 否 | **是** |
| **形参改变是否影响实参** | 否 | **是** |
| **适用场景** | 逐个传递数组元素 | 传递整个数组 |


## 💡 核心考点

1. **数组元素作实参**：传递的是数组元素的**值**，是值传递
2. **数组名作实参**：传递的是数组**首元素的地址**，是地址传递
3. **数组名作形参**：可以不指定数组大小（一维数组），用空方括号
4. **多维数组形参**：可以省略第一维大小，但**不能省略第二维大小**
5. **地址传递特点**：形参数组的改变会影响实参数组
6. **长度传递**：使用 `int n` 参数传递数组长度，使函数通用性更强
