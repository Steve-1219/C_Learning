# 7.5 内部函数和外部函数

## 🎯 学习目标

- 掌握内部函数（static函数）的定义和使用
- 掌握外部函数（extern函数）的定义和使用
- 理解内部函数和外部函数的区别
- 能够正确使用函数声明在多文件程序中共享函数


## 📖 知识梳理

### 7.5.1 内部函数 

#### 一、内部函数的概念

如果一个函数**只能被本文件中的其他函数调用**，而不能被其他文件调用，则称为**内部函数**（又称静态函数）。

**定义格式：**

```c
static 类型标识符 函数名(形参表列)
```

**示例：**

```c
static int fun(int a, int b)    // static表示fun是一个内部函数
{
    return a + b;
}
```

#### 二、内部函数的特点

| 特点 | 说明 |
| :--- | :--- |
| ① 作用范围 | 只限于**本文件**内部 |
| ② 文件保护 | 不同文件中可以使用相同名字的内部函数，互不干扰 |
| ③ 模块化 | 有助于实现信息隐藏，将函数限制在模块内部使用 |


### 7.5.2 外部函数 

#### 一、外部函数的概念

如果在定义函数时，在函数首部添加关键字 **`extern`**，则此函数是**外部函数**，可供**其他文件**调用。

**定义格式：**

```c
extern 类型标识符 函数名(形参表列)
```

**示例：**

```c
extern int fun(int a, int b)   // extern表示fun是外部函数
{
    return a + b;
}
```

> 💡 **注意**：C语言规定，如果在定义函数时**省略 `extern`**，则**默认为外部函数**。

#### 二、外部函数的使用

在需要调用外部函数的其他文件中，需要用关键字 `extern` 对函数作**声明**，表示该函数是在其他文件中定义的外部函数。

**【例】用外部函数实现删除字符串中的指定字符**

**问题描述**：有一个字符串，内有若干个字符，现输入一个字符，要求将字符串中该字符删去。

分别定义3个函数：
- `enter_string`：输入字符串
- `delete_string`：删除指定字符
- `print_string`：输出字符串

将以上3个函数分别放在3个文件中，main函数在另一个文件中调用这3个函数。

**file1.c（主文件）**

```c
// file1.c
#include <stdio.h>

int main()
{
    extern void enter_string(char str[]);
    extern void delete_string(char str[], char ch);
    extern void print_string(char str[]);
    
    char str[80];
    char ch;
    
    enter_string(str);
    printf("请输入要删除的字符：");
    scanf("%c", &ch);
    delete_string(str, ch);
    print_string(str);
    
    return 0;
}
```

**file2.c（输入函数）**

```c
// file2.c
#include <stdio.h>

void enter_string(char str[])
{
    printf("请输入字符串：");
    gets(str);
}
```

**file3.c（删除函数）**

```c
// file3.c
void delete_string(char str[], char ch)
{
    int i, j;
    for (i = 0, j = 0; str[i] != '\0'; i++)
    {
        if (str[i] != ch)
        {
            str[j] = str[i];
            j++;
        }
    }
    str[j] = '\0';
}
```

**file4.c（输出函数）**

```c
// file4.c
#include <stdio.h>

void print_string(char str[])
{
    printf("删除后的字符串：%s\n", str);
}
```

#### 三、内部函数与外部函数的对比

| 对比项 | 内部函数（static） | 外部函数（extern） |
| :--- | :--- | :--- |
| **作用范围** | 只限于**本文件** | **所有文件** |
| **关键字** | `static` | `extern` |
| **默认情况** | 需要显式指定 | 省略时默认 |
| **文件间同名** | 可以同名，互不干扰 | 不能同名（会冲突） |
| **适用场景** | 模块内部使用的辅助函数 | 供其他文件调用的公共函数 |


## 💡 核心考点

1. **内部函数**：`static`修饰，只能在本文件内使用
2. **外部函数**：`extern`修饰（默认），可被其他文件调用
3. **函数声明**：在其他文件中调用外部函数时，需要用 `extern` 声明
4. **文件保护**：使用 `static` 可以防止函数被其他文件误用
