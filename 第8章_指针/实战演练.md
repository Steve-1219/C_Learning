## 六、程序设计题（共25分）

### 题目1：指针交换两个数（10分）

编写一个C程序，定义一个函数 `swap`，使用指针参数交换两个整数的值。在主函数中输入两个整数，调用 `swap` 函数交换它们的值，并输出交换后的结果。

**输入样例：**
```
请输入两个整数：5 10
```

**输出样例：**
```
交换前：a = 5, b = 10
交换后：a = 10, b = 5
```

**参考代码：**

```c
#include <stdio.h>

void swap(int *p1, int *p2)
{
    int temp;
    temp = *p1;
    *p1 = *p2;
    *p2 = temp;
}

int main()
{
    int a, b;
    printf("请输入两个整数：");
    scanf("%d%d", &a, &b);
    printf("交换前：a = %d, b = %d\n", a, b);
    swap(&a, &b);
    printf("交换后：a = %d, b = %d\n", a, b);
    return 0;
}
```


### 题目2：使用指针查找数组最大值（15分）

编写一个C程序，定义一个函数 `find_max`，使用指针查找数组中的最大值及其位置。函数返回最大值的地址，在主函数中输出最大值和位置。

**要求：**
- 使用指针遍历数组
- 函数返回最大值的地址
- 在主函数中通过指针输出最大值及其位置

**输入样例：**
```
请输入5个整数：3 7 1 9 5
```

**输出样例：**
```
最大值为：9，位置为：3
```

**参考代码：**

```c
#include <stdio.h>

int *find_max(int *arr, int n)
{
    int *max = arr;
    int *p;
    for (p = arr + 1; p < arr + n; p++)
    {
        if (*p > *max)
            max = p;
    }
    return max;
}

int main()
{
    int a[100], n, i;
    int *p;
    printf("请输入元素个数：");
    scanf("%d", &n);
    printf("请输入%d个整数：", n);
    for (i = 0; i < n; i++)
        scanf("%d", &a[i]);
    p = find_max(a, n);
    printf("最大值为：%d，位置为：%d\n", *p, p - a);
    return 0;
}
```

### 五、程序改错题

| 错误序号 | 错误代码位置 | 错误原因 | 正确写法 |
| :---: | :--- | :--- | :--- |
| 1 | `int *p;` | p 未初始化 | `int *p;` 后应指向变量 |
