# 8.7 指针数组和多重指针

## 🎯 学习目标

- 掌握指针数组的定义和使用
- 理解指向指针的指针（多重指针）的概念
- 掌握指针数组处理字符串的方法
- 了解指针数组作 main 函数形参

## 📖 知识梳理

### 8.7.1 指针数组 

一个数组，若其元素均为指针类型数据，称为**指针数组**。指针数组中的每一个元素都存储一个地址，相当于一个指针变量。

**定义一维指针数组的一般形式：**

```c
类型名 *数组名[数组长度];
```

**示例：**

```c
int *p[4];
```

**优先级理解：**

由于 `[]` 比 `*` 优先级高，因此：

```c
int *p[4];    // p 先与 [4] 结合，表示 p 有4个元素，再与 * 结合，表示元素是指针类型
```

> ⚠️ **注意区别**：
> - `int *p[4];` → **指针数组**，有4个指针元素
> - `int (*p)[4];` → **指向一维数组的指针变量**，p 指向包含4个元素的一维数组

**指针数组的图示：**

```
┌─────────────────────────────────────────────────────┐
│                   指针数组 p                        │
│  ┌──────────┐  ┌──────────┐  ┌──────────┐  ┌──────────┐ │
│  │  p[0]    │  │  p[1]    │  │  p[2]    │  │  p[3]    │ │
│  │ 指向 int │  │ 指向 int │  │ 指向 int │  │ 指向 int │ │
│  └────┬─────┘  └────┬─────┘  └────┬─────┘  └────┬─────┘ │
│       └─────────────┴─────────────┴─────────────┘       │
│                            ↓                            │
│                   各指向一个整型变量                     │
└─────────────────────────────────────────────────────┘
```


### 8.7.2 指向指针的指针（多重指针）

指向指针数据的指针变量，简称为**指向指针的指针**。

**定义形式：**

```c
char **p;    // p 是指向 char 类型指针的指针变量
```

> 💡 **理解**：`char **p` 等价于 `char *(*p)`，表示 p 指向一个 `char *` 类型的数据。

**示意图：**

```
┌─────────────────────────────────────────────────────────────┐
│                   指向指针的指针示意图                       │
│                                                             │
│   p          *p           **p                              │
│   ↓           ↓            ↓                              │
│ ┌───────┐  ┌───────┐  ┌─────────┐                        │
│ │ p的值  │→ │指针值  │→ │ 数据    │                        │
│ │ 地址   │  │ 地址   │  │ 值      │                        │
│ └───────┘  └───────┘  └─────────┘                        │
│   指向      指向         指向                              │
│  指针数组   变量         数据                              │
└─────────────────────────────────────────────────────────────┘
```

**【例16】用指向指针的指针输出整型数组各元素的值。**

```c
#include <stdio.h>
int main()
{
    int a[5] = {1, 3, 5, 7, 9};
    int *num[5] = {&a[0], &a[1], &a[2], &a[3], &a[4]};
    int **p, i;
    p = num;    // p 指向指针数组 num 的首元素
    
    for (i = 0; i < 5; i++)
    {
        printf("%2d", **p);
        p++;    // p 指向下一个指针数组元素
    }
    printf("\n");
    return 0;
}
```

**运行结果：**

```
1 3 5 7 9
```

**程序分析：**

| 步骤 | 操作 | 说明 |
| :---: | :--- | :--- |
| ① | `int *num[5] = {...};` | 定义指针数组，每个元素指向 `a` 的一个元素 |
| ② | `int **p;` | 定义指向指针的指针变量 |
| ③ | `p = num;` | p 指向 `num[0]`，即 `&a[0]` 的地址 |
| ④ | `**p` | 第一次输出 `a[0]` 的值 1 |
| ⑤ | `p++` | p 指向 `num[1]`，即 `&a[1]` 的地址 |
| ⑥ | 循环输出 `a[0]` ~ `a[4]` 的值 |


### 8.7.3 指针数组处理字符串

指针数组比较适合用来指向若干个字符串，使字符串处理更加方便灵活。

**优点：**

| 优点 | 说明 |
| :--- | :--- |
| ① 长度灵活 | 每个字符串的长度可以不同 |
| ② 排序高效 | 只需移动指针，不必移动字符串 |
| ③ 节省空间 | 不需要为等长字符串分配固定空间 |

**示意图：**

```
┌─────────────────────────────────────────────────────────────┐
│              指针数组指向字符串                             │
│                                                             │
│   name[0] ─────────→ "Follow me"                          │
│   name[1] ─────────→ "BASIC"                              │
│   name[2] ─────────→ "Great Wall"                         │
│   name[3] ─────────→ "FORTRAN"                            │
│   name[4] ─────────→ "Computer design"                    │
│                                                             │
│   ⚠️ 排序时只需交换指针数组元素的值，字符串本身不动         │
└─────────────────────────────────────────────────────────────┘
```

**【例15】将若干字符串按字母顺序（由小到大）输出。**

```c
#include <stdio.h>
#include <string.h>

int main()
{
    void sort(char *name[], int n);
    void print(char *name[], int n);
    char *name[] = {"Follow me", "BASIC", "Great Wall", "FORTRAN", "Computer design"};
    int n = 5;
    
    sort(name, n);
    print(name, n);
    return 0;
}

void sort(char *name[], int n)
{
    char *temp;
    int i, j, k;
    for (i = 0; i < n - 1; i++)
    {
        k = i;
        for (j = i + 1; j < n; j++)
        {
            if (strcmp(name[k], name[j]) > 0)
                k = j;
        }
        if (k != i)
        {
            temp = name[i];
            name[i] = name[k];
            name[k] = temp;
        }
    }
}

void print(char *name[], int n)
{
    int i;
    for (i = 0; i < n; i++)
        printf("%s\n", name[i]);
}
```

**运行结果：**

```
BASIC
Computer design
FORTRAN
Follow me
Great Wall
```

**程序分析：**

| 行号 | 代码 | 说明 |
| :---: | :--- | :--- |
| 第8行 | `char *name[] = {...};` | 定义指针数组，指向5个字符串 |
| 第18行 | `strcmp(name[k], name[j]) > 0` | 比较两个字符串，若前者大则交换指针 |
| 第24-27行 | 交换 `name[i]` 和 `name[k]` | 只交换指针，不交换字符串本身 |
| 第32行 | `printf("%s\n", name[i]);` | 按排序后的顺序输出字符串 |


### 8.7.4 指针数组作 main 函数的形参 

main 函数可以有参数：

```c
int main(int argc, char *argv[])
```

| 参数 | 全称 | 说明 |
| :--- | :--- | :--- |
| `argc` | argument count | 命令行参数的**个数** |
| `argv` | argument vector | 指向命令行参数字符串的**指针数组** |

> 💡 **注意**：main 函数的形参名不一定为 `argc` 和 `argv`，可以任意命名，只是习惯用这两个名字。

**合法的 main 函数参数形式：**

```c
int main(int a, char *c[])    // 合法
int main(int arc, char **argv)   // 合法
```

**不合法形式：**

```c
int main(int argc, char *argv)   // 错误：argv 应为指针数组，不是指针变量
```


## 💡 核心考点

1. **指针数组定义**：`类型名 *数组名[数组长度];`
2. **优先级**：`[]` 高于 `*`，`int *p[4]` 是指针数组，`int (*p)[4]` 是行指针
3. **指向指针的指针**：`char **p;`
4. **指针数组处理字符串**：排序时只交换指针，不移动字符串
5. **main 函数参数**：`int main(int argc, char *argv[])`
