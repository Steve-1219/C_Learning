# 8.6 返回指针值的函数

## 🎯 学习目标

- 掌握返回指针值的函数的定义方法
- 理解返回指针的函数与普通函数的区别
- 能够使用返回指针的函数解决实际问题


## 📖 知识梳理

### 8.6.1 定义返回指针的函数 

一个函数可以返回一个整型值、字符值、实型值等，也可以返回**指针型的数据**，即**地址**。

**定义返回指针的函数的一般形式：**

```c
类型名 *函数名(参数表列);
```

**示例：**

```c
int *a(int x, int y);
```

> 💡 **理解**：`a` 是函数名，调用它以后能得到一个指向整型数据的指针，即整型数据的地址。

**运算符优先级说明：**

```c
int *a(int x, int y);
```

由于 `()` 的优先级高于 `*`，因此 `int *a(int x, int y)` 等价于：

```c
int *(a(int x, int y))
```

- `a` 是一个函数
- 函数的返回值是 `int *` 类型（指向整型数据的指针）
- 最前面的 `int` 表示返回的指针指向整型变量


### 8.6.2 使用返回指针的函数 

**【例14】有 a 个学生，每个学生有 b 门课程的成绩。要求在用户输入学生序号以后，输出该学生的全部成绩。用指针函数来实现。**

**问题分析：**
- 学生数 a = 3，课程数 b = 4
- 定义一个二维数组 `score`，用来存放学生成绩
- 定义一个查询学生成绩的函数 `search`，它是一个**返回指针的函数**
- `search` 函数的返回值是 `&score[k][0]`（第 k 个学生第 0 门课程的地址）
- 在主函数中通过返回的指针输出该学生的全部成绩

**程序代码：**

```c
#include <stdio.h>

int main()
{
    float score[][4] = {
        {60, 70, 80, 90},
        {56, 89, 67, 88},
        {34, 78, 90, 66}
    };
    float *search(float (*pointer)[4], int n);    // 函数声明
    float *p;
    int i, k;
    
    printf("请输入学生的序号（从0开始）：");
    scanf("%d", &k);
    
    p = search(score, k);    // 调用 search 函数，返回 score[k][0] 的地址
    
    printf("序号为 %d 的学生的成绩：\n", k);
    for (i = 0; i < 4; i++)
    {
        printf("%5.2f ", *(p + i));
    }
    printf("\n");
    return 0;
}

// 定义返回指针的函数 search
float *search(float (*pointer)[4], int n)
{
    return *(pointer + n);    // 返回第 n 行第 0 列元素的地址
}
```

**运行结果：**

```
请输入学生的序号（从0开始）：2
序号为 2 的学生的成绩：
34.00 78.00 90.00 66.00
```

**程序分析：**

| 行号 | 代码 | 说明 |
| :---: | :--- | :--- |
| 第8行 | `float *search(float (*pointer)[4], int n);` | 声明 search 函数返回 `float *` 类型 |
| 第12行 | `p = search(score, k);` | 调用函数，返回第 k 行首地址 |
| 第15行 | `*(p + i)` | 通过指针输出该行第 i 列的元素 |
| 第21行 | `return *(pointer + n);` | 返回第 n 行第 0 列元素的地址 |

**数据传递过程：**

```
┌─────────────────────────────────────────────────────────────┐
│                    数据传递示意图                           │
│                                                             │
│   score 二维数组                    search 函数             │
│   ┌─────────────────────┐          ┌──────────────────┐    │
│   │ 第0行: 60 70 80 90  │ ←────── │ pointer 指向第0行 │    │
│   │ 第1行: 56 89 67 88  │ ←────── │ pointer+1 指向第1行│    │
│   │ 第2行: 34 78 90 66  │ ←────── │ pointer+2 指向第2行│    │
│   └─────────────────────┘          └──────────────────┘    │
│                                                             │
│   当 n = 2 时：                                             │
│   return *(pointer + 2) = 第2行首地址 = &score[2][0]       │
│                                                             │
│   主函数中：                                                │
│   p = &score[2][0]                                         │
│   *(p+0) = score[2][0] = 34.00                             │
│   *(p+1) = score[2][1] = 78.00                             │
│   *(p+2) = score[2][2] = 90.00                             │
│   *(p+3) = score[2][3] = 66.00                             │
└─────────────────────────────────────────────────────────────┘
```


### 8.6.3 返回指针的函数与普通函数的对比

| 对比项 | 普通函数 | 返回指针的函数 |
| :--- | :--- | :--- |
| **定义形式** | `类型名 函数名(参数表列)` | `类型名 *函数名(参数表列)` |
| **返回值类型** | 普通数据（int、float、char等） | **地址**（指针） |
| **return语句** | `return 变量名;` | `return &变量名;` 或 `return 指针变量;` |
| **调用结果** | 得到一个值 | 得到一个**地址** |


## 💡 核心考点

1. **返回指针的函数定义**：`类型名 *函数名(参数表列);`
2. **运算符优先级**：`()` 优先级高于 `*`，所以 `int *a()` 表示返回指针的函数
3. **return语句**：返回的是**地址**（指针）
4. **指针函数的典型应用**：返回数组某行的首地址或某个元素的地址
