# 8.4 通过指针引用字符串

## 🎯 学习目标

- 掌握字符数组表示法引用字符串
- 掌握字符指针变量表示法引用字符串
- 掌握字符指针作为函数参数的使用
- 理解字符指针变量和字符数组的区别


## 📖 知识梳理

### 8.4.1 字符串的引用方式

在C程序中，想引用一个字符串，可以用以下两种方法。

#### 一、字符数组表示法

用字符数组存放一个字符串，可以通过数组名和下标引用字符串数组中的元素，也可以通过数组名和字符串常量 `%s` 输出该字符串。

**【例8】定义一个字符数组，在其中存放字符串 `"I love study!"`，输出该字符串和第8个字符。**

**程序代码：**

```c
#include <stdio.h>
int main()
{
    char string[] = "I love study!";    // 定义字符数组 string
    printf("string 数组存放的字符串是：%s\n", string);    // 用 %s 输出整个字符串
    printf("该字符串的第8个字符是：%c\n", string[7]);      // 用 %c 输出一个字符
    return 0;
}
```

**运行结果：**

```
string 数组存放的字符串是：I love study!
该字符串的第8个字符是：s
```

> 💡 **注意**：`string[7]` 对应字符串 `"I love study!"` 中的第8个字符（从0开始计数）。

#### 二、字符指针变量表示法

用字符指针变量指向一个字符串常量，通过字符指针变量引用字符串常量。

**【例9】通过字符指针变量输出一个字符串。**

可以不定义字符数组，只定义一个字符指针变量，用它指向字符串常量中的字符。通过字符指针变量输出该字符串。

**程序代码：**

```c
#include <stdio.h>
int main()
{
    char *string = "I love study!";    // 定义字符指针变量 string 并初始化
    printf("%s\n", string);            // 输出字符串
    return 0;
}
```

**程序分析：**

| 要点 | 说明 |
| :--- | :--- |
| ① | 没有定义字符数组，只定义了一个字符指针变量 `string` |
| ② | C语言对字符串常量是按**字符数组**处理的，在内存中开辟了一个字符数组存放该字符串常量，但这个数组是**无名**的 |
| ③ | 对字符指针变量初始化，实际上是把字符串第1个元素的地址赋给指针变量 |
| ④ | `char *string = "I love study!";` 等价于：`char *string; string = "I love study!";` |
| ⑤ | `printf("%s\n", string);` 会从 `string` 指向的字符开始输出，直到遇到 `'\0'` 结束 |

#### 三、字符串的复制

**【例10】将字符串 `a` 复制给字符串 `b`，然后输出字符串 `b`。**

**方法一：使用数组名和地址法**

```c
#include <stdio.h>
int main()
{
    char a[] = "I am Chinese", b[20];    // 定义字符数组
    int i;
    
    for (i = 0; *(a + i) != '\0'; i++)    // 用地址法访问数组元素
        *(b + i) = *(a + i);               // 复制字符
    *(b + i) = '\0';                       // 添加字符串结束标志
    
    printf("string a is: %s\n", a);
    printf("string b is: ");
    for (i = 0; b[i] != '\0'; i++)
        printf("%c", b[i]);                // 用下标法输出
    printf("\n");
    return 0;
}
```

**运行结果：**

```
string a is: I am Chinese
string b is: I am Chinese
```

**方法二：使用指针变量**

```c
#include <stdio.h>
int main()
{
    char a[] = "I am Chinese", b[20], *p1, *p2;
    p1 = a; p2 = b;    // p1, p2 分别指向数组 a 和数组 b 的第一个元素
    
    for (; *p1 != '\0'; p1++, p2++)    // p1, p2 每次自增1
        *p2 = *p1;                     // 将 p1 指向的字符赋给 p2 指向的元素
    *p2 = '\0';                        // 添加字符串结束标志
    
    printf("string a is: %s\n", a);
    printf("string b is: %s\n", b);
    return 0;
}
```

**运行结果：** 与例10相同。

**程序分析：**

| 行号 | 代码 | 说明 |
| :---: | :--- | :--- |
| 第3行 | `p1 = a; p2 = b;` | p1指向a[0]，p2指向b[0] |
| 第5行 | `*p2 = *p1;` | 将p1指向的字符赋给p2指向的元素 |
| 第5行 | `p1++, p2++` | 同步后移，指向下一个字符 |
| 第6行 | `*p2 = '\0';` | 复制完成后添加结束标志 |


### 8.4.2 字符指针作函数参数 

如果想把一个字符串从一个函数"传递"到另一个函数，可以用地址传递的办法，即用字符数组名作参数，也可以用字符指针变量作参数。在被调用的函数中可以改变字符串的内容，在主调函数中可以得到改变后的字符串。

**【例2】用函数调用实现字符串的复制。**

**（1）用字符数组名作函数形参和实参**

```c
#include <stdio.h>

int main()
{
    void copy_string(char from[], char to[]);    // 函数声明
    char a[] = "I am array a";
    char b[] = "I am array b";
    
    printf("string a = %s\nstring b = %s\n", a, b);
    copy_string(a, b);    // 用数组名作实参
    printf("复制后：\n");
    printf("string a = %s\nstring b = %s\n", a, b);
    return 0;
}

void copy_string(char from[], char to[])
{
    int i = 0;
    while (from[i] != '\0')
    {
        to[i] = from[i];
        i++;
    }
    to[i] = '\0';
}
```

**运行结果：**

```
string a = I am array a
string b = I am array b
复制后：
string a = I am array a
string b = I am array a
```

**（2）用字符指针变量作实参**

```c
#include <stdio.h>

int main()
{
    void copy_string(char from[], char to[]);
    char a[] = "I am array a";
    char b[] = "I am array b";
    char *from = a, *to = b;    // from 指向 a，to 指向 b
    
    printf("string a = %s\nstring b = %s\n", from, to);
    copy_string(from, to);    // 实参为指针变量
    printf("复制后：\n");
    printf("string a = %s\nstring b = %s\n", from, to);
    return 0;
}

void copy_string(char from[], char to[])
{
    int i = 0;
    while (from[i] != '\0')
    {
        to[i] = from[i];
        i++;
    }
    to[i] = '\0';
}
```

**（3）用字符指针变量作形参和实参**

```c
#include <stdio.h>

int main()
{
    void copy_string(char *from, char *to);
    char *a = "I am array a";    // a 是字符指针变量
    char b[] = "I am array b";
    char *p = b;                 // p 指向 b 数组首元素
    
    printf("string a = %s\n", a);
    printf("string b = %s\n", p);
    copy_string(a, p);           // 实参为指针变量
    printf("复制后：\n");
    printf("string a = %s\n", a);
    printf("string b = %s\n", b);
    return 0;
}

void copy_string(char *from, char *to)
{
    for (; *from != '\0'; from++, to++)
        *to = *from;
    *to = '\0';
}
```

**调用函数时实参与形参的对应关系：**

| 序号 | 实参 | 形参 |
| :---: | :--- | :--- |
| ① | 字符数组名 | 字符数组名 |
| ② | 字符数组名 | 字符指针变量 |
| ③ | 字符指针变量 | 字符数组名 |
| ④ | 字符指针变量 | 字符指针变量 |


### 8.4.3 使用字符指针变量和字符数组的区别 

用字符数组和字符指针变量都能实现字符串的存储和运算，但它们之间有重要的区别。

| 对比项 | 字符数组 | 字符指针变量 |
| :--- | :--- | :--- |
| **存储内容** | 存放字符数据 | 存放**地址** |
| **赋值方式** | 只能在定义时整体赋值，不能用赋值语句整体赋值 | 可以用赋值语句指向另一个字符串 |
| **内存分配** | 编译时分配若干存储单元 | 只分配一个存储单元（4字节） |
| **值可变性** | 数组名是常量，**不能改变** | 指针变量的值**可以改变** |
| **内容可变性** | 数组元素的值**可以改变** | 指向的字符串常量内容**不能改变** |
| **引用方式** | 下标法、地址法 | 下标法、地址法 |

**详细说明：**

**1. 存储内容不同**

```c
char str[14] = "I love study!";    // str 存放的是字符数据
char *p = "I love study!";         // p 存放的是地址（字符串首字符的地址）
```

**2. 赋值方式不同**

```c
// 字符指针变量可以赋值
char *a;
a = "I love study!";    // 合法

// 字符数组名不能赋值
char str[14];
str = "I love study!";    // 错误！数组名是常量
```

**3. 初始化含义不同**

```c
// 字符指针变量初始化
char *a = "I love study!";    // 将字符串首地址赋给 a

// 字符数组初始化
char str[14] = "I love study!";    // 将字符数据存入数组各元素
```

**4. 存储单元分配不同**

```c
char str[14];    // 分配14个字节
char *p;         // 分配4个字节（Visual C++）
```

**5. 指针值可变性不同**

```c
char *p = "I love study!";
p = "Hello";    // 合法：p 可以指向不同的字符串

char str[14] = "I love study!";
// str = "Hello";    // 错误：数组名是常量，不能赋值
```

**6. 内容可变性不同**

```c
char str[14] = "I love study!";
str[0] = 'Y';    // 合法：可以修改数组元素

char *p = "I love study!";
// p[0] = 'Y';    // 非法：不能修改字符串常量
```

> ⚠️ **重要提醒**：字符指针变量指向的**字符串常量**的内容是**不可修改**的。如果想让内容可修改，应使用字符数组。

**7. 引用数组元素**

```c
char *a = "I love study!";
printf("%c\n", a[5]);    // 合法：输出 'e'（字符串常量以数组形式存放）
```


## 💡 核心考点

1. **字符串的两种引用方式**：字符数组、字符指针变量
2. **字符指针变量初始化**：`char *p = "字符串";`
3. **`%s` 格式输出**：输出字符串，遇到 `'\0'` 结束
4. **字符串复制**：逐个字符复制，最后加 `'\0'`
5. **实参与形参的对应关系**：数组名或指针变量可互换
6. **字符数组与指针变量的区别**：存储内容、赋值、可变性等
