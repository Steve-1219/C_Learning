# 8.5 指向函数的指针

## 🎯 学习目标

- 理解函数指针的概念
- 掌握指向函数的指针变量的定义方法
- 掌握通过函数指针变量调用函数的方法
- 掌握用指向函数的指针作函数参数的方法


## 📖 知识梳理

### 8.5.1 函数指针的概念 

如果在程序中定义了一个函数，在编译时，编译系统为函数代码分配一段存储空间，这段存储空间的**起始地址**（又称**入口地址**）称为这个函数的**指针**。

可以定义一个指向函数的指针变量，用来存放某一函数的起始地址，这就意味着此指针变量指向该函数。

```c
int (*p)(int, int);    // p 是一个指向函数的指针变量
```


### 8.5.2 定义和使用指向函数的指针变量

#### 一、定义指向函数的指针变量

定义指向函数的指针变量的一般形式为：

```c
类型名 (*指针变量名)(函数参数表列);
```

**示例：**

```c
int (*p)(int, int);    // p 是一个指向函数的指针变量
```

**各部分说明：**

| 部分 | 说明 |
| :--- | :--- |
| `int` | 函数返回值的类型 |
| `(*p)` | p 是一个指针变量（括号不能省略） |
| `(int, int)` | 函数的参数表列 |

> ⚠️ **注意**：括号 `(*p)` 不能省略。如果写成 `int *p(int, int)`，则表示声明一个函数 `p`，其返回值是 `int` 型指针。

#### 二、使用指向函数的指针变量的注意事项

| 序号 | 注意事项 | 说明 |
| :---: | :--- | :--- |
| ① | **类型匹配** | 指针变量只能指向**同类型**的函数（返回值类型相同，参数个数和类型相同） |
| ② | **赋值方式** | 给函数指针变量赋值时，只需给出**函数名**，不必给出参数 |
| ③ | **调用方式** | 用函数指针变量调用函数时，使用 `(*p)(参数表列)` |
| ④ | **不能运算** | 对指向函数的指针变量**不能进行算术运算**（如 `p+n`、`p+p` 无意义） |
| ⑤ | **灵活切换** | 一个指针变量可以先后指向同类型的**不同函数** |

**正确与错误示例：**

```c
int max(int x, int y);    // 定义函数 max
int (*p)(int, int);       // 定义函数指针变量

p = max;                  // 正确：将 max 函数的地址赋给 p
// p = max(a, b);         // 错误：不能将函数调用的结果赋给 p
```

> 💡 **理解**：`p = max;` 是将函数的**入口地址**赋给 p，而不是调用函数。


### 8.5.3 通过函数指针变量调用函数 

除了可以通过函数名调用函数，还可以通过指向函数的指针变量来调用该函数。

**【例1】用函数求整数 a 和 b 中的大者。**

**（1）通过函数名调用函数**

```c
#include <stdio.h>
int main()
{
    int max(int, int);    // 函数声明
    int a, b, c;
    printf("请输入 a 和 b 的值：");
    scanf("%d,%d", &a, &b);
    c = max(a, b);        // 通过函数名调用 max 函数
    printf("%d 和 %d 中的较大者是：%d\n", a, b, c);
    return 0;
}

int max(int x, int y)
{
    int z;
    if (x > y) z = x;
    else z = y;
    return z;
}
```

**运行结果：**

```
请输入 a 和 b 的值：85,13
85 和 13 中的较大者是：85
```

**（2）通过函数指针变量调用函数**

```c
#include <stdio.h>
int main()
{
    int max(int, int);           // 函数声明
    int (*p)(int, int);          // 定义指向函数的指针变量 p
    int a, b, c;
    p = max;                     // 使 p 指向 max 函数
    printf("请输入 a 和 b 的值：");
    scanf("%d,%d", &a, &b);
    c = (*p)(a, b);              // 通过指针变量调用 max 函数
    printf("%d 和 %d 中的较大者是：%d\n", a, b, c);
    return 0;
}

int max(int x, int y)
{
    int z;
    if (x > y) z = x;
    else z = y;
    return z;
}
```

**程序分析：**

| 行号 | 代码 | 说明 |
| :---: | :--- | :--- |
| 第4行 | `int (*p)(int, int);` | 定义 p 是指向函数的指针变量，函数返回值为 int，有两个 int 参数 |
| 第6行 | `p = max;` | 将函数 max 的入口地址赋给 p，p 指向 max |
| 第9行 | `c = (*p)(a, b);` | 通过指针变量调用 max 函数，等价于 `c = max(a, b);` |

> 💡 **核心理解**：`(*p)(a, b)` 表示调用 p 所指向的函数，实参为 a 和 b。


### 8.5.4 用指向函数的指针作函数参数 

指向函数的指针变量的一个重要用途是把**函数的地址作为参数**传递到其他函数。

#### 一、基本原理

指向函数的指针可以作为函数参数，把函数的入口地址传递给形参，这样就能够在被调用的函数中调用其他函数。

```
实参函数名    f1    f2
    ↓         ↓     ↓
void fun(int (*x1)(int), int (*x2)(int, int))    // 形参是指向函数的指针变量
{
    int a, b, i = 3, j = 5;
    a = (*x1)(i);          // 调用 f1 函数
    b = (*x2)(i, j);       // 调用 f2 函数
}
```

#### 二、执行过程

```
┌─────────────────────────────────────────────────────────────┐
│                      函数指针传递过程                       │
│                                                             │
│   main 函数                    fun 函数                    │
│                                                             │
│   实参传递函数地址             形参接收函数地址              │
│                                                             │
│   fun(f1, f2)  ──────────→  x1 = f1 的地址                 │
│   (f1, f2 是函数名)         x2 = f2 的地址                 │
│                                                             │
│   在 fun 函数中：                                           │
│   (*x1)(i) 相当于调用 f1(i)                                │
│   (*x2)(i, j) 相当于调用 f2(i, j)                          │
│                                                             │
│   ⚠️ 调用不同的函数时，只需改变实参即可，fun 函数无需修改   │
└─────────────────────────────────────────────────────────────┘
```

#### 三、应用场景

如果在每次调用 `fun` 函数时，要调用的函数不是固定的：
- 这次调用 `f1` 和 `f2`
- 下次调用 `f3` 和 `f4`
- 第3次调用 `f5` 和 `f6`

通过指针变量，只要在每次调用 `fun` 函数时给出不同的函数名作为实参即可，`fun` 函数本身**不必做任何修改**。

> 💡 **优点**：这种方法符合结构化程序设计方法原则，是程序设计中常用的技巧。


## 💡 核心考点

1. **函数指针**：函数的入口地址
2. **函数指针变量定义**：`类型名 (*指针变量名)(参数表列);`
3. **括号不能省略**：`(*p)` 不能写成 `*p`
4. **赋值**：`p = 函数名;`（不能带括号和参数）
5. **调用**：`(*p)(实参表列);`
6. **不能运算**：函数指针不能进行算术运算
7. **函数指针作参数**：传递函数地址，实现灵活调用

2. 若有函数 `int max(int a, int b);` 和函数指针变量 `int (*p)(int, int);`，以下赋值正确的是（ ）
   A. `p = max(a, b);`  B. `p = max;`  C. `*p = max;`  D. `p = &max;`

3. 通过函数指针变量 `p` 调用函数，正确的写法是（ ）

4. 以下关于函数指针变量的说法，错误的是（ ）
   A. 函数指针变量可以指向同类型的不同函数  B. 函数指针变量可以进行算术运算
   C. 函数指针变量存放的是函数的入口地址  D. 函数指针变量可以作为函数参数

### 一、选择题

| 题号 | 答案 | 解析 |
| :---: | :---: | :--- |
| 1 | **B** | `int (*p)(int, int);` 是正确的函数指针定义，括号不能省略 |
| 2 | **B** | `p = max;` 将函数 max 的入口地址赋给 p |
