# 8.1 指针的概念

## 🎯 学习目标

- 理解地址和指针的基本概念
- 掌握直接访问与间接访问的区别
- 理解指针变量的概念
- 能够区分指针与指针变量


## 📖 知识梳理

### 8.1.1 地址与指针

#### 一、内存与地址

如果在程序中定义了一个变量，在对程序进行编译时，系统就会根据变量类型给这个变量分配内存单元。

| 变量类型 | Visual C++分配字节数 |
| :--- | :---: |
| 整型变量（int） | 4个字节 |
| 单精度浮点型变量（float） | 4个字节 |
| 字符型变量（char） | 1个字节 |

内存区的每一个字节有一个编号，这就是**地址**。通过地址能找到所需的变量单元。

> 💡 **核心概念**：在C语言中将地址形象化地称为**"指针"**，通过它能找到以它为地址的内存单元。

#### 二、存储单元的地址和内容

假设程序已定义了3个整型变量 `i`、`j`、`k`：
- 系统分配地址 `2000~2003` 给变量 `i`
- 地址 `2004~2007` 给变量 `j`
- 地址 `2008~2011` 给变量 `k`

```
内存地址      存储单元内容
 2000  ┌─────────────┐
 2001  │     3       │  ← 变量 i（地址2000~2003）
 2002  │             │
 2003  ├─────────────┤
 2004  │     6       │  ← 变量 j（地址2004~2007）
 2005  │             │
 2006  │             │
 2007  ├─────────────┤
 2008  │     9       │  ← 变量 k（地址2008~2011）
 2009  │             │
 2010  │             │
 2011  └─────────────┘
```

> ⚠️ **重要**：存储单元的**地址**和存储单元的**内容**是两个完全不同的概念。


### 8.1.2 直接访问与间接访问

#### 一、直接访问

在程序中一般是通过**变量名**来引用变量的值。实际上，是通过变量名找到存储单元的地址，从而对存储单元进行存取操作的。程序经过编译以后已经将变量名转换为变量的地址，对变量的值存取都是通过地址进行的。

**示例：**

```c
int i = 3;
int j = 6;
int k;
k = i + j;
```

**执行过程：**
1. 从 `2000~2003` 字节取出 `i` 的值（3）
2. 从 `2004~2007` 字节取出 `j` 的值（6）
3. 将 3 + 6 的和（9）送到 `k` 所占用的 `2008~2011` 字节单元中

> 💡 **定义**：这种直接按变量名进行的访问，称为**"直接访问"**方式。

```
直接访问示意图：

    ┌─────────┐
    │ 变量名 i │  ← 直接通过变量名
    └────┬────┘
         ↓
    ┌─────────┐
    │ 地址2000 │  ← 找到对应的地址
    └────┬────┘
         ↓
    ┌─────────┐
    │ 值 3    │  ← 取出或存入值
    └─────────┘
```

#### 二、间接访问

除了"直接访问"方式，还可以采用另一种称之为**"间接访问"**的方式，即将变量 `i` 的地址存放在另一个变量中，然后通过该变量来找到变量 `i` 的地址，从而访问变量 `i`。

**示例：**

```c
int i = 3;
int *i_pointer;      // 定义指针变量
i_pointer = &i;      // 将i的地址存放到i_pointer中
```

此时，`i_pointer` 的值就是 `2000`（变量 `i` 所占用的起始地址）。

**执行过程：**
1. 先找到存放"变量 `i` 的地址"的变量 `i_pointer`
2. 从中取出 `i` 的地址（2000）
3. 从 `2000` 字节开始的存储单元中取出 `i` 的值（3）

> 💡 **定义**：这种通过另一个变量间接找到数据的方式，称为**"间接访问"**方式。

```
间接访问示意图：

    ┌─────────────┐
    │  i_pointer   │  ← 指针变量存放地址
    │   2000       │
    └──────┬───────┘
           ↓
    ┌─────────────┐
    │   地址2000   │  ← 找到地址
    └──────┬───────┘
           ↓
    ┌─────────────┐
    │   值 3       │  ← 取出或存入值
    └─────────────┘
```

#### 三、直接访问与间接访问的对比

| 对比项 | 直接访问 | 间接访问 |
| :--- | :--- | :--- |
| **访问方式** | 通过变量名直接访问 | 通过指针变量间接访问 |
| **示例** | `i = 3;` | `*i_pointer = 3;` |
| **特点** | 直接操作存储单元 | 先找地址，再操作存储单元 |

**两种表达方式：**

| 方式 | 代码 | 说明 |
| :--- | :--- | :--- |
| 直接访问 | `i = 3;` | 将数值3直接送到变量i所标志的单元中 |
| 间接访问 | `*i_pointer = 3;` | 将数值3送到i_pointer所指向的单元（即变量i的存储单元） |


### 8.1.3 指针与指针变量

#### 一、指针

一个变量的地址称为该变量的**指针**。例如，地址 `2000` 是变量 `i` 的指针。

| 术语 | 含义 |
| :--- | :--- |
| **指针** | 变量的地址（如 `2000`） |
| **指向** | 通过地址能找到变量单元，称为"指向" |

#### 二、指针变量

如果有一个变量专门用来**存放另一个变量的地址**（即指针），则它称为**指针变量**。

**示例：**

```c
int i = 3;
int *i_pointer;      // 定义指针变量i_pointer
i_pointer = &i;      // i_pointer存放i的地址
```

- `i_pointer` 是**指针变量**
- `2000`（i的地址）是**指针**

> 💡 **重要区分**：

| 概念 | 含义 | 示例 |
| :--- | :--- | :--- |
| **指针** | 变量的地址 | 地址 `2000` 是变量i的指针 |
| **指针变量** | 存放地址的变量 | `i_pointer` 是指针变量 |


### 8.1.4 指针变量定义的基本概念

在C语言中，指针变量用来存放地址。定义指针变量的一般形式将在8.2节详细介绍，但需要先理解以下概念：

| 概念 | 说明 |
| :--- | :--- |
| **指针变量的类型** | 指针变量有类型之分，不同类型的指针变量存放不同类型变量的地址 |
| **指针变量的值** | 指针变量的值是某个变量的地址 |
| **指针变量指向的对象** | 指针变量所存放的地址对应的存储单元 |


## 💡 核心考点

1. **地址**：内存单元的编号，通过地址可以找到对应的存储单元
2. **指针**：变量的地址，即"指针 = 地址"
3. **指针变量**：存放地址的变量
4. **直接访问**：通过变量名直接访问存储单元
5. **间接访问**：通过指针变量间接访问存储单元
6. **指针与指针变量的区别**：指针是地址，指针变量是存放地址的变量
