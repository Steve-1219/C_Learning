# 8.2 指针变量

## 🎯 学习目标

- 掌握指针变量的定义方法
- 理解指针变量的基类型概念
- 掌握指针变量的引用方法
- 理解指针变量作函数参数的使用


## 📖 知识梳理

### 8.2.1 指针变量的定义

#### 一、指针变量定义的一般形式

定义指针变量的一般形式为：

```c
基类型 *指针变量名;
```

**示例：**

```c
int *pointer_1, *pointer_2;    // 定义两个指向整型变量的指针变量
```

**各部分说明：**

| 部分 | 说明 |
| :--- | :--- |
| **基类型** | 指定此指针变量可以指向的变量的类型（如 `int`、`float`、`char` 等） |
| `*` | 表示该变量是指针类型变量 |
| **指针变量名** | 指针变量的标识符（如 `pointer_1`） |

> 💡 **重要理解**：指针变量是**从基本数据类型派生出来的类型**，它不能离开基本类型而独立存在。

**合法定义示例：**

```c
float *pointer_3;      // pointer_3 是指向 float 型变量的指针变量
char *pointer_4;       // pointer_4 是指向字符型变量的指针变量
```

#### 二、指针变量的初始化

可以在定义指针变量的同时对它进行初始化：

```c
int a = 3, b = 5;
int *pointer_1 = &a, *pointer_2 = &b;    // 定义指针变量并分别指向 a 和 b
```

#### 三、定义指针变量时的注意事项

| 序号 | 注意事项 | 说明 |
| :---: | :--- | :--- |
| ① | `*` 的含义 | 指针变量前面的 `*` 表示该变量的类型是指针型变量。指针变量名是 `pointer_1` 和 `pointer_2`，而不是 `*pointer_1` 和 `*pointer_2` |
| ② | 必须指定基类型 | 知道地址还不够，必须知道该数据的**类型**，才能按存储单元的长度以及数据的存储形式正确地取出数据 |
| ③ | 指针移动的单位 | 指针移动1个位置时，移动的字节数取决于基类型。如 `int` 指针移动4个字节，`char` 指针移动1个字节 |
| ④ | 类型匹配 | 一个指针变量**只能指向同一类型的变量**，不能指向不同类型的变量 |
| ⑤ | 只能存放地址 | 指针变量中**只能存放地址（指针）**，不能存放数据 |

**错误示例：**

```c
int *pointer_1;
pointer_1 = 100;    // 错误！不能将整数赋给指针变量
```

> 💡 **即学即练**：
> （单选）已知 p 是指针变量，则不能初始化为 p 的值是（ ）
> A. 地址  B. NULL  C. 0  D. 1
>
> 【答案】**D**
>
> 【解析】指针变量中只能存放地址，0 和 NULL 都代表空指针，1 是整数，不能赋给指针变量。


### 8.2.2 指针变量的引用 

在引用指针变量时，可能有以下3种情况：

#### 一、给指针变量赋值

```c
int a = 3;
int *p;
p = &a;              // 把 a 的地址赋给指针变量 p
```

#### 二、引用指针变量指向的变量

```c
printf("%d", *p);    // 输出 p 指向的变量的值
```

> 💡 **易混辨析**：

| 符号 | 含义 | 说明 |
| :--- | :--- | :--- |
| `&a` | 取地址运算符 | 代表变量 `a` 的地址 |
| `*p` | 指针运算符 | 代表 `p` 指向的变量的值 |

#### 三、引用指针变量的值

```c
printf("%o", p);     // 以八进制形式输出指针变量 p 的值（即地址）
```

如果 `p` 指向了 `a`，就是输出 `a` 的地址，即 `&a`。

**示例总结：**

```c
int a = 3;
int *p = &a;         // p 指向 a

printf("%d\n", a);   // 输出 3（a 的值）
printf("%d\n", *p);  // 输出 3（p 指向的变量的值，即 a 的值）
printf("%p\n", p);   // 输出 a 的地址
printf("%p\n", &a);  // 输出 a 的地址
```

> 💡 **即学即练**：
> 若有 `int a = 5; int *p = &a;` 则 `*p = 1;` 相当于执行了（ ）
> A. `a = 1`  B. `p = 1`  C. `&a = 1`  D. `*a = 1`
>
> 【答案】**A**
>
> 【解析】`*p` 代表 p 指向的变量，即变量 a，所以 `*p = 1;` 相当于 `a = 1;`。


### 8.2.3 指针变量作函数参数 

函数的参数不仅可以是整型、浮点型、字符型等数据，还可以是**指针类型**。它的作用是将一个变量的**地址**传送到另一个函数中。

**【例1】输入 a 和 b 两个整数，按先大后小的顺序输出 a 和 b。**

**算法分析：**
定义一个函数 `swap`，将两个指向整型变量的指针变量作为形参，在函数中通过指针交换两个变量的值。

**程序代码：**

```c
#include <stdio.h>

int main()
{
    void swap(int *p1, int *p2);    // 对 swap 函数的声明
    int a, b;
    int *pointer_1, *pointer_2;
    
    printf("请输入两个整数：");
    scanf("%d,%d", &a, &b);
    
    pointer_1 = &a;    // pointer_1 指向 a
    pointer_2 = &b;    // pointer_2 指向 b
    
    if (a < b)
        swap(pointer_1, pointer_2);    // 交换 a 和 b 的值
    
    printf("a = %d, b = %d\n", a, b);
    return 0;
}

void swap(int *p1, int *p2)    // 形参为指针变量
{
    int temp;
    temp = *p1;        // temp = a 的值
    *p1 = *p2;         // a = b 的值
    *p2 = temp;        // b = temp 的值
}
```

**运行结果：**

```
请输入两个整数：3,5
a = 5, b = 3
```

**数据传递过程：**

```
主函数 main                     swap 函数
┌─────────────────┐            ┌─────────────────┐
│  int a = 3      │            │  int *p1        │
│  int b = 5      │            │  int *p2        │
│  pointer_1 = &a │ ────────→ │  p1 = &a        │
│  pointer_2 = &b │ ────────→ │  p2 = &b        │
└─────────────────┘            └─────────────────┘
                                      ↓
                               ┌─────────────────┐
                               │ temp = *p1 (3)  │
                               │ *p1 = *p2 (5)   │
                               │ *p2 = temp (3)  │
                               └─────────────────┘
```

**程序分析：**

| 要点 | 说明 |
| :--- | :--- |
| ① | `pointer_1` 和 `pointer_2` 存放的是 `a` 和 `b` 的地址 |
| ② | 调用 `swap` 时，将 `pointer_1` 和 `pointer_2` 的值（即 `a` 和 `b` 的地址）传递给形参 `p1` 和 `p2` |
| ③ | 在 `swap` 函数中，`*p1` 和 `*p2` 分别代表 `a` 和 `b` |
| ④ | 交换 `*p1` 和 `*p2` 的值，就是交换 `a` 和 `b` 的值 |
| ⑤ | **注意**：`temp` 不能写成 `*temp`，因为 `*temp` 是指针变量 `temp` 所指向的变量，而 `temp` 未初始化 |

> 💡 **即学即练**：
> （判断）指针变量不可以作为函数的参数。（ ）
>
> 【答案】**✖**
>
> 【解析】函数的参数可以是整型、浮点型、字符型等，也可以是**指针类型**。指针变量作函数参数的作用是将一个变量的地址传送到另一个函数中。


## 💡 核心考点

1. **指针变量定义**：`基类型 *指针变量名;`
2. **基类型**：指针变量必须指定基类型，决定指针移动的单位
3. **`*` 的含义**：定义时表示指针类型，使用时表示取指向的内容
4. **`&` 的含义**：取变量的地址
5. **指针变量初始化**：`int *p = &a;`
6. **指针变量只能存放地址**：不能存放普通数据
7. **指针变量作函数参数**：传递地址，可修改实参的值

## ✅ 参考答案

### 一、选择题

| 题号 | 答案 | 解析 |
| :---: | :---: | :--- |
| 1 | **B** | `p` 是指针变量名，`*p` 表示 p 指向的内容 |
| 2 | **A** | `*p` 代表 p 指向的变量 a 的值，即 3 |
| 3 | **B** | 指针变量只能存放地址，不能存放任意值 |
| 4 | **B** | 指针变量作函数参数时，传递的是变量的**地址** |

### 二、填空题

| 题号 | 答案 |
| :---: | :--- |
| 1 | 基类型 |
| 2 | a |
| 3 | 地址（指针） |
| 4 | `&`、`*` |

### 三、判断题

| 题号 | 答案 | 解析 |
| :---: | :---: | :--- |
| 1 | **✖** | 一个指针变量只能指向**同类型**的变量 |
| 2 | **✔** | 指针变量中只能存放地址（指针） |
| 3 | **✖** | `p` 是指针变量的值（地址），`*p` 是 p 指向的变量的值 |
