## 七、程序设计题（共10分）

编写一个C程序，从键盘输入一个整数，判断该数是否为偶数，并输出判断结果。

**要求：**
- 从键盘输入一个整数
- 判断该数是否为偶数（用取余运算符 `%` 判断）
- 输出结果

**输入样例：**
```
请输入一个整数：8
```

**输出样例：**
```
8 是偶数
```

**参考代码：**

```c
#include <stdio.h>

int main() {
    int num;
    printf("请输入一个整数：");
    scanf("%d", &num);
    if (num % 2 == 0) {
        printf("%d 是偶数\n", num);
    } else {
        printf("%d 是奇数\n", num);
    }
    return 0;
}
```

### 四、简答题

**1. 简述算法的5个特性。**

| 特性   | 说明                                         |
| :----- | :------------------------------------------- |
| 输入   | 可以有零个或多个输入                         |
| 输出   | 至少有一个或多个输出                         |
| 有穷性 | 算法在执行有限步后必须结束                   |
| 确定性 | 算法的每一步都具有确定的含义，不能出现二义性 |
| 可行性 | 算法的每一步都必须是可以执行的               |


**2. 简述C语言程序的三种基本结构，并各举一个对应的C语言关键字。**

| 结构     | 说明                               | C语言关键字                    |
| :------- | :--------------------------------- | :----------------------------- |
| 顺序结构 | 按照书写顺序依次执行各语句         | 无特定关键字                   |
| 选择结构 | 根据条件判断选择执行不同的语句分支 | `if`、`else`、`switch`、`case` |
| 循环结构 | 在满足条件时重复执行某段代码       | `while`、`do-while`、`for`     |


**3. 简述C语言程序的结构特点（至少写出4点）。**

① 一个程序由一个或多个源程序文件组成，包含预处理指令、全局声明和函数定义

② C程序从 `main` 函数开始执行，一个程序有且仅有一个 `main` 函数

③ C语言本身没有输入输出语句，输入输出由库函数 `printf` 和 `scanf` 实现

④ 每个语句以分号 `;` 结尾

⑤ 书写格式自由，一行内可写多个语句，一个语句可写在多行上

⑥ C语言支持单行注释 `//` 和块式注释 `/* */`


**4. 简述C程序从编写到运行的完整过程，并说明各阶段的作用。**

| 步骤 | 作用                                 | 生成文件    |
| :--- | :----------------------------------- | :---------- |
| 编辑 | 编写C语言源代码                      | `.c` 文件   |
| 编译 | 将源程序翻译为目标代码，检查语法错误 | `.obj` 文件 |
| 链接 | 将目标文件与库文件合并为可执行文件   | `.exe` 文件 |
| 运行 | 执行可执行程序，得到运行结果         | 输出结果    |
