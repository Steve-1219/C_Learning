## 📖 知识梳理

### 1.2.1 C语言程序的结构 

C语言程序的结构有以下特点：

（一）**一个程序由一个或多个源程序文件组成。** 在一个源程序文件中可以包括以下3个部分：

#### 1.预处理指令

如 `#include <stdio.h>`（还有一些其他预处理指令，如 `#define` 等）。

C编译系统在对源程序进行"翻译"以前，先由一个**预处理程序**（也称预处理器、预编译器）对预处理指令进行预处理。对于 `#include <stdio.h>` 指令来说，就是将 `stdio.h` 头文件的内容读进来，放在 `#include` 指令行。由预处理得到的结果与程序其他部分一起，组成一个完整的、可以用来编译的最后的源程序，然后由编译程序对该源程序正式进行编译，才得到目标程序。

#### 2.全局声明

在函数之外进行的数据声明，如 `int a, b;`。

- 放到 main 函数的前面是**全局声明**
- 在函数外声明的变量是**全局变量**
- 在函数中声明的变量是**局部变量**

#### 3.函数定义



（二）C程序由一个或多个函数组成，其中必须包含且只能包含一个 `main` 函数（主函数）。函数用来实现特定的功能，程序的全部功能都是由各个函数分别完成的。编写C程序加上编写一个一个的函数，函数是C语言的主要组成成分。一个小程序只包含一个源程序文件，一个源程序文件包含若干个函数，其中只有一个main函数。在进行编译时是以源程序文件为对象进行的

---

### 1.2.2 C语言程序的结构特点

#### （1）main函数

- **main** 是函数的名字，表示"主函数"
- `int main()` 表示此函数的类型是 **int 类型（整型）**
- 一个C程序总是从 **main 函数开始执行**，而不论 main 函数在整个程序中的位置如何
- 在执行主函数后会得到一个值（即函数值），其值为整型

#### （2）函数体

函数体由一对花括号 `{ }` 括起来，包含声明部分和执行部分。

#### （3）输入输出函数

C语言本身**没有输入输出语句**。输入和输出的操作是由库函数 `scanf` 和 `printf` 等函数来完成的。

> 把输入输出操作用库函数实现，可以使C语言本身的规模较小，编译程序简单，很容易在各种机器上实现，程序具有**可移植性**。

#### （4）语句以分号结尾

每个语句和数据项的最后必须有一个**分号**。

```c
c = a + b;    // 分号不可缺少
```

#### （5）书写格式自由

一行内可以写几个语句，一个语句可以分写在多行上。

#### （6）注释

C语言允许使用两种注释方式：

| 注释方式     | 解释                                                       | 实例                 |
| :----------- | :--------------------------------------------------------- | :------------------- |
| **单行注释** | 可以单独占一行，也可以出现在一行中其他内容的右侧，不能跨行 | `// 这是单行注释`    |
| **块式注释** | 可以包含多行内容，以 `/*` 开始，以 `*/` 结束               | `/* 这是多行注释 */` |

> **注意**：注释内容不会被编译和运行。编译时无法发现注释中的拼写错误。

> **注意**：在字符串中的 `"/"` 和 `"*"` 都不作为注释的开始，而是作为字符串的一部分。
>
> ```c
> printf("//Hello World!\n");   // 输出：//Hello World!
> printf("/*Hello World!*/\n"); // 输出：/*Hello World!*/
> ```


### 1.2.3 简单的C语言程序举例

#### 例1：输出"Hello World!"

**程序代码：**

```c
#include <stdio.h>              // 第1行：编译预处理指令

int main() {                    // 第2行：定义主函数
                                // 第3行：函数开始的标志
    printf("Hello World!\n");   // 第4行：输出所指定的一行信息
    return 0;                   // 第5行：函数执行完毕时返回函数值0
}                               // 第6行：函数结束的标志
```

**运行结果：**

```
Hello World!
```

**程序分析：**

| 行号  | 代码                        | 说明                                                         |
| :---: | :-------------------------- | :----------------------------------------------------------- |
| 第1行 | `#include <stdio.h>`        | 用来提供有关输入输出函数的信息。`stdio.h` 是系统提供的头文件，`stdio` 是 "standard input & output" 的缩写，即"标准输入输出"。文件后缀 `.h` 表示头文件（header file）。如果在程序中要用到标准函数库中的输入函数 `scanf` 和输出函数 `printf`，应该在程序的开头写上：`#include <stdio.h>` |
| 第2行 | `int main()`                | `main` 是函数名，表示"主函数"，前面的 `int` 表示函数返回值类型为整型 |
| 第4行 | `printf("Hello World!\n");` | 输出语句，使用输出函数 `printf` 将双撇号中的字符串输出。`\n` 是换行符，将光标位置移到下一行的开头 |
| 第5行 | `return 0;`                 | 当主函数执行结束时将整型 `0` 作为函数的返回值                |


#### 例2：求两个整数之和

**程序代码：**

```c
#include <stdio.h>

int main() {
    int a, b, sum;              // 声明部分：定义a、b和sum为整型变量
    scanf("%d, %d", &a, &b);    // 输入变量a和b的值
    sum = a + b;                // 计算a与b之和，赋值给sum
    printf("sum = %d\n", sum);  // 输出结果
    return 0;
}
```

**运行结果：**

```
21, 32
sum = 53
```

**程序分析：**

| 行号  | 代码                         | 说明                                                         |
| :---: | :--------------------------- | :----------------------------------------------------------- |
| 第4行 | `int a, b, sum;`             | 声明部分，定义 `a`、`b` 和 `sum` 为整型（`int`）变量         |
| 第5行 | `scanf("%d, %d", &a, &b);`   | 输入函数。`%d` 指定输入的数据格式为十进制整数。`&a` 表示"变量a的地址"，`&b` 表示"变量b的地址"。执行时从键盘读入两个整数，送到变量a和b的地址处，然后分别赋给变量a和b |
| 第6行 | `sum = a + b;`               | 使 `sum` 的值为 `a` 与 `b` 之和                              |
| 第7行 | `printf("sum = %d\n", sum);` | 输出结果。`"sum = %d\n"` 是输出格式字符串，其中 `"sum="` 是用户希望输出的字符，`%d` 指定输出格式为十进制整数。执行时用 `sum` 变量的值（十进制整数）取代 `%d` |


### 1.2.4 return语句

`return` 语句在编程中起着非常重要的作用，特别是在函数中。其主要作用包括：

| 作用         | 说明                                                         |
| :----------- | :----------------------------------------------------------- |
| **终止函数** | 当函数执行到 `return` 语句时，函数的执行会立即终止，无论 `return` 语句后面的代码是否执行完。`return` 语句可以用来提前结束函数的运行 |
| **返回值**   | `return` 语句可以返回一个值，这个值可以被调用函数所使用。返回值可以是任何类型的数值，如整型、浮点型、字符型等 |

**示例：**

```c
int max(int x, int y) {
    if (x > y)
        return x;   // 返回较大值，函数立即结束
    else
        return y;   // 返回较大值，函数立即结束
}
```


## 💡 核心考点

1. **C程序结构**：一个C程序由一个或多个源程序文件组成，包含预处理指令、全局声明、函数定义
2. **main函数**：程序从 `main` 函数开始执行，一个程序有且仅有一个 `main` 函数
3. **输入输出**：C语言本身无输入输出语句，通过库函数 `printf` 和 `scanf` 实现
4. **注释方式**：单行注释 `//` 和块式注释 `/* */`
5. **分号**：每个语句以分号 `;` 结尾
6. **return语句**：终止函数并返回函数值
7. **格式输入输出**：`%d` 表示十进制整数格式
8. **地址符**：`&` 是地址符，用于获取变量的地址

## 🔧 上机实践

编写一个C程序，要求：
1. 定义两个整型变量 `a` 和 `b`
2. 从键盘输入两个整数
3. 计算两个数的和、差、积、商
4. 输出结果

**参考代码：**

```c
#include <stdio.h>

int main() {
    int a, b;
    int sum, diff, product;
    float quotient;

    printf("请输入两个整数（用空格分隔）：");
    scanf("%d %d", &a, &b);

    sum = a + b;
    diff = a - b;
    product = a * b;
    quotient = (float)a / b;

    printf("和 = %d\n", sum);
    printf("差 = %d\n", diff);
    printf("积 = %d\n", product);
    printf("商 = %.2f\n", quotient);

    return 0;
}
```
