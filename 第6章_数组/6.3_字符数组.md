# 6.3 字符数组

- 掌握字符数组的定义和初始化方法

- 掌握字符数组的输入输出方法
- 掌握常用字符串处理函数的使用


## 📖 知识梳理

### 6.3.1 定义与初始化字符数组 

#### 一、定义字符数组

字符数组用来存放字符数据，数组中的**每一个元素存放一个字符**。

**定义字符数组的方法**与定义数值型数组的方法类似：

```c
char c[10];
```

以上定义了字符数组 `c`，包含10个元素：`c[0]` ~ `c[9]`。

**赋值示例：**

```c
c[0] = 'T';
c[1] = 'e';
c[2] = 'a';
c[3] = 'm';
c[4] = 'h';
c[5] = 'a';
c[6] = 'p';
c[7] = 'p';
c[8] = 'y';
c[9] = 't';
```

赋值后数组的状态：

```
┌─────┬─────┬─────┬─────┬─────┬─────┬─────┬─────┬─────┬─────┐
│ c[0]│ c[1]│ c[2]│ c[3]│ c[4]│ c[5]│ c[6]│ c[7]│ c[8]│ c[9]│
├─────┼─────┼─────┼─────┼─────┼─────┼─────┼─────┼─────┼─────┤
│  T  │  e  │  a  │  m  │  h  │  a  │  p  │  p  │  y  │  t  │
└─────┴─────┴─────┴─────┴─────┴─────┴─────┴─────┴─────┴─────┘
```

> 💡 **注意**：由于字符型数据是以整数形式（ASCII代码）存放的，因此也可以用整型数组来存放字符数据，但会浪费存储空间。
> ```c
> int c[10];
> c[0] = 'a';   // 合法，但浪费存储空间
> ```

#### 二、字符数组的初始化

**1. 用单个字符初始化**

```c
char c[10] = {'T', 'e', 'a', 'm', 'h', 'a', 'p', 'p', 'y', 't'};
```

**2. 初值个数小于数组长度**

如果初值个数小于数组长度，则只将这部分元素赋给数组前面的元素，**其余元素自动变为空字符 `\0`**。

```c
char c[10] = {'T', 'e', 'a', 'm'};
```

数组状态：

```
┌─────┬─────┬─────┬─────┬─────┬─────┬─────┬─────┬─────┬─────┐
│ c[0]│ c[1]│ c[2]│ c[3]│ c[4]│ c[5]│ c[6]│ c[7]│ c[8]│ c[9]│
├─────┼─────┼─────┼─────┼─────┼─────┼─────┼─────┼─────┼─────┤
│  T  │  e  │  a  │  m  │  \0 │  \0 │  \0 │  \0 │  \0 │  \0 │
└─────┴─────┴─────┴─────┴─────┴─────┴─────┴─────┴─────┴─────┘
```

**3. 初值个数大于数组长度**

如果花括号中字符个数大于数组长度，则出现**语法错误**。

**4. 省略数组长度**

如果提供的初值个数与预定的数组长度相同，在定义时可以省略数组长度，系统会自动确定数组长度。

```c
char c[] = {'T', 'e', 'a', 'm', 'h', 'a', 'p', 'p', 'y', 't'};
// 系统自动确定数组长度为10
```


### 6.3.2 引用字符数组 

可以引用字符数组中的一个元素，得到一个字符。

**【例5】输出一个已知的字符串。**

```c
#include <stdio.h>
int main()
{
    int i;
    char c[15] = {'I', ' ', 'a', 'm', ' ', 'a', ' ', 's', 't', 'u', 'd', 'e', 'n', 't', '.'};
    
    for (i = 0; i < 15; i++)
    {
        printf("%c", c[i]);
    }
    printf("\n");
    
    return 0;
}
```

**运行结果：**

```
I am a student.
```

**【例6】输出一个三角形图案。**

```c
#include <stdio.h>
int main()
{
    int i, j;
    char diamond[2][3] = {
        {'*', ' ', '*'},
        {' ', '*', ' '}
    };
    
    for (i = 0; i < 2; i++)
    {
        for (j = 0; j < 3; j++)
        {
            printf("%c", diamond[i][j]);
        }
        printf("\n");
    }
    
    return 0;
}
```

**运行结果：**

```
* *
 *
```


### 6.3.3 字符串和字符串结束标志 

#### 一、字符串结束标志 `\0`

编译系统在用字符串数组存储字符串常量时会**自动加一个 `\0` 作为结束符**。

> 💡 `\0` 是ASCII码为0的字符，称为"空字符"或"字符串结束标志"。

**示例：**

```c
char c[] = {"c program."};
```

系统在最后一个字符后面自动添加 `\0`，存储情况如下：

```
┌───┬───┬───┬───┬───┬───┬───┬───┬───┬───┬────┐
│ c │   │ p │ r │ o │ g │ r │ a │ m │ . │ \0 │
└───┴───┴───┴───┴───┴───┴───┴───┴───┴───┴────┘
```

#### 二、用字符串常量初始化字符数组

**方式一：带花括号**

```c
char c[] = {"I am happy"};
```

**方式二：省略花括号（推荐）**

```c
char c[] = "I am happy";
```

> ⚠️ **重要**：此时数组 `c` 的长度不是10，而是**11**。因为字符串常量的最后一位是结束标志 `\0`。

上面的初始化与下面的初始化等价：

```c
char c[] = {'I', ' ', 'a', 'm', ' ', 'h', 'a', 'p', 'p', 'y', '\0'};
```

#### 三、字符数组与字符串的区别

| 对比项 | 字符数组 | 字符串 |
| :--- | :--- | :--- |
| **结束标志** | 不要求必须有 `\0` | 必须有 `\0` |
| **示例** | `char c[5] = {'C','h','i','n','a'};` | `char c[6] = "China";` |
| **长度** | 数组长度 = 5 | 数组长度 = 6（含 `\0`） |

> 💡 **注意**：字符数组并不要求它的最后一个字符为 `\0`，甚至可以不包含 `\0`。但在字符串处理中，为了方便，通常会在字符数组末尾加上 `\0`。

#### 四、字符串输出要点

1. 输出字符串时，**不包括结束符 `\0`**
2. 用 `%s` 格式输出字符串时，只输出到 `\0` 结束
3. 如果一个字符串中包含多个 `\0`，则遇到**第一个 `\0`** 时就会结束输出


### 6.3.4 字符数组的输入输出 

字符数组的输入输出有以下两种方法：

| 方法 | 格式声明 | 说明 |
| :--- | :---: | :--- |
| **逐个输入输出** | `%c` | 每次输入或输出一个字符 |
| **整个字符串输入输出** | `%s` | 一次性输入或输出整个字符串 |

#### 一、用 `%s` 格式输出字符串

```c
char c[] = "China";
printf("%s\n", c);
```

**输出结果：**

```
China
```

> 💡 **注意**：
> - 输出的字符中**不包括结束符 `\0`**
> - `printf` 函数中的输出项是**字符串数组名**，而不是数组元素名
> - 如果数组长度大于字符串的实际长度，也只输出到 `\0` 结束

#### 二、用 `%s` 格式输入字符串

```c
char c[10];
scanf("%s", c);    // 注意：c前面不加 &
```

> 💡 **拿分笔记**：`scanf` 函数中的输入项如果是**数组名**，**不要再加取地址符 `&`**，因为在C语言中数组名代表数组的首地址。

**错误写法：**

```c
scanf("%s", &c);    // ❌ 错误！c前面不应加 &
```

#### 三、输入多个字符串

```c
char str1[5], str2[5], str3[5];
scanf("%s%s%s", str1, str2, str3);
```

**输入数据：**

```
How are you?
```

由于有空格字符分隔，作为3个字符串输入。输入完后：

```
str1: H o w \0
str2: a r e \0
str3: y o u ? \0
```

> ⚠️ **注意**：如果使用一个字符串数组输入包含空格的字符串，空格会被视为分隔符：
> ```c
> char str[13];
> scanf("%s", str);
> ```
> 输入 `How are you?` 时，只将空格前的 `"How"` 存入数组。


### 6.3.5 字符串处理函数 

在使用字符串处理函数时，应当在程序文件的开头加上：

```c
#include <string.h>
```

#### 一、常用字符串处理函数

| 函数 | 函数形式 | 功能 |
| :--- | :--- | :--- |
| **puts** | `puts(字符串数组)` | 输出一个字符串（自动换行） |
| **gets** | `gets(字符串数组)` | 从终端输入一个字符串 |
| **strcat** | `strcat(str1, str2)` | 将str2连接到str1后面 |
| **strcpy** | `strcpy(str1, str2)` | 将str2复制到str1中 |
| **strcmp** | `strcmp(str1, str2)` | 比较两个字符串 |
| **strlen** | `strlen(字符串数组)` | 测试字符串的实际长度（**不含 `\0`**） |
| **strlwr** | `strlwr(字符串数组)` | 将大写字母转换为小写字母 |
| **strupr** | `strupr(字符串数组)` | 将小写字母转换为大写字母 |

#### 二、各函数详解

**1. puts函数（输出字符串）**

```c
char str[] = "China";
puts(str);    // 输出 China 并自动换行
```

**2. gets函数（输入字符串）**

```c
char str[20];
gets(str);    // 从键盘输入一个字符串
```

**3. strcat函数（字符串连接）**

```c
char str1[30] = "How are ";
char str2[] = "you";
printf("%s", strcat(str1, str2));
```

**输出：**

```
How are you
```

**4. strcpy函数（字符串复制）**

```c
char str1[20], str2[] = "Hello";
strcpy(str1, str2);    // 将str2复制到str1中
printf("%s", str1);    // 输出 Hello
```

> 💡 **名师点拨**：若要将一个字符串复制到另一个字符串数组中，**必须使用 `strcpy` 函数**，不能直接用赋值运算符 `=`。

**错误写法：**

```c
char str1[20], str2[] = "Hello";
str1 = str2;    // ❌ 错误！不能直接赋值
```

**5. strcmp函数（字符串比较）**

**比较规则：** 从左到右逐个比较两个字符串中字符的ASCII码值，直到出现不同的字符或遇到 `\0` 为止。

| 比较结果 | 函数返回值 |
| :--- | :---: |
| str1 == str2 | 0 |
| str1 > str2 | 正整数 |
| str1 < str2 | 负整数 |

**示例：**

```c
printf("%d", strcmp("abc", "abc"));    // 输出 0
printf("%d", strcmp("abc", "abd"));    // 输出 负整数
printf("%d", strcmp("abd", "abc"));    // 输出 正整数
```

**6. strlen函数（测试字符串长度）**

```c
char str[] = "China";
printf("%d", strlen(str));    // 输出 5（不包含 \0）
```

> ⚠️ **注意**：`strlen` 测试的是字符串的**实际长度**，不包含结束符 `\0`。

#### 三、即学即练

1. **（单选）字符串连接函数是（ ）**
   A. `strlen`  B. `strcat`  C. `strcpy`  D. `strcmp`

   **答案：B**

2. **（单选）定义一维数组 int a[5];，下列输入语句正确的是（ ）**
   A. `scanf("%d", a);`  B. `scanf("%d", a[5]);`
   C. `scanf("%d", &a);`  D. `scanf("%d", &a[5]);`

   **答案：A**

   **解析：** `scanf` 函数中的输入项如果是数组名，不需要再加取地址符 `&`，因为数组名代表数组的首地址。


## 💡 核心考点

1. **字符数组定义**：`char 数组名[长度];`
2. **字符数组初始化**：`char c[5] = {'C','h','i','n','a'};`
3. **字符串结束标志**：`\0`，系统自动添加
4. **字符串初始化**：`char c[] = "China";`（长度 = 6，含 `\0`）
5. **输入输出**：`%s` 格式输入输出整个字符串
6. **数组名即地址**：`scanf("%s", c);` 不需要 `&`
7. **字符串处理函数**：需包含 `<string.h>`
