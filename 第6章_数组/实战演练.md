## 六、程序设计题（共25分）

### 题目1：求数组中的最大值和最小值（10分）

编写一个C程序，定义一个包含10个整数的一维数组，从键盘输入10个整数，求出数组中的最大值和最小值并输出。

**输入样例：**
```
请输入10个整数：5 3 8 1 9 2 7 4 6 0
```

**输出样例：**
```
最大值 = 9
最小值 = 0
```

**参考代码：**

```c
#include <stdio.h>
int main()
{
    int i, max, min;
    int a[10];
    
    printf("请输入10个整数：");
    for (i = 0; i < 10; i++)
    {
        scanf("%d", &a[i]);
    }
    
    max = min = a[0];
    for (i = 1; i < 10; i++)
    {
        if (a[i] > max) max = a[i];
        if (a[i] < min) min = a[i];
    }
    
    printf("最大值 = %d\n", max);
    printf("最小值 = %d\n", min);
    return 0;
}
```


### 题目2：统计字符串中字符个数（15分）

编写一个C程序，从键盘输入一个字符串（不包含空格），统计并输出其中大写字母、小写字母和数字的个数。

**要求：**
- 使用字符数组存储字符串
- 使用 `gets` 或 `scanf` 输入字符串
- 遍历数组统计各类字符个数
- 使用字符串处理函数 `strlen` 获取字符串长度

**输入样例：**
```
请输入字符串：Hello123
```

**输出样例：**
```
大写字母：1
小写字母：4
数字：3
```

**参考代码：**

```c
#include <stdio.h>
#include <string.h>
int main()
{
    char str[100];
    int i, len;
    int upper = 0, lower = 0, digit = 0;
    
    printf("请输入字符串：");
    scanf("%s", str);
    
    len = strlen(str);
    for (i = 0; i < len; i++)
    {
        if (str[i] >= 'A' && str[i] <= 'Z')
            upper++;
        else if (str[i] >= 'a' && str[i] <= 'z')
            lower++;
        else if (str[i] >= '0' && str[i] <= '9')
            digit++;
    }
    
    printf("大写字母：%d\n", upper);
    printf("小写字母：%d\n", lower);
    printf("数字：%d\n", digit);
    return 0;
}
```


## ✅ 参考答案

### 一、选择题

| 题号 | 答案 | 解析 |
| :---: | :---: | :--- |
| 1 | **B** | `int a[10];` 是正确的一维数组定义方式 |
| 2 | **A** | 只初始化了前3个元素，`a[3]` 未赋值，自动为0 |
| 3 | **A** | 按行优先：第0行有4个元素，所以 `a[1][0]` 前面有4个元素 |
| 4 | **C** | "Hello"有5个字符 + `\0` = 6个字节 |
| 5 | **B** | `strcat` 是字符串连接函数 |

### 五、程序改错题
