### 5.3.1 循环程序的设计方法

循环程序的设计通常遵循以下步骤：

```
┌─────────────────────────┐
│  1. 分析问题，确定循环的  │
│     条件和循环体         │
├─────────────────────────┤
│  2. 选择合适循环结构     │
│     (while/do...while/for)│
├─────────────────────────┤
│  3. 确定循环初值和终值   │
├─────────────────────────┤
│  4. 编写循环体代码       │
├─────────────────────────┤
│  5. 验证循环是否正确     │
│     (边界条件、循环次数)  │
└─────────────────────────┘
```

> 💡 **核心思路**：确定循环三要素——**初始化**、**循环条件**、**循环变量的更新**。


### 5.3.2 素数判断问题 

#### 一、素数的概念

**素数（质数）**：一个大于1的自然数，除了1和它本身外，不能被其他自然数整除。

- 例如：2、3、5、7、11、13、17、19……是素数
- 4（能被2整除）、6（能被2和3整除）……不是素数（合数）

#### 二、素数判断算法

**基本思路**：让 n 被 i 除（i 从2到 n-1），如果 n 能被其中任意一个整数整除，则 n 不是素数。

**优化思路**：只需检查到 √n（平方根）即可。

> 💡 **原因**：如果 n 有一个大于 √n 的因数，那么必定有一个小于 √n 的因数与之对应。

**【例8】输入一个大于3的整数 n，判断它是否为素数。**

**方法一：基本方法（检查到 n-1）**

```c
#include <stdio.h>
int main()
{
    int n, i;
    printf("请输入一个大于3的整数：");
    scanf("%d", &n);
    
    for (i = 2; i < n; i++)
    {
        if (n % i == 0)
            break;    // 能被整除，不是素数
    }
    
    if (i == n)
        printf("%d 是素数\n", n);
    else
        printf("%d 不是素数\n", n);
    
    return 0;
}
```

**运行结果：**

```
请输入一个大于3的整数：17
17 是素数

请输入一个大于3的整数：15
15 不是素数
```

**方法二：优化方法（检查到 √n）**

```c
#include <stdio.h>
#include <math.h>
int main()
{
    int n, i, k;
    printf("请输入一个大于3的整数：");
    scanf("%d", &n);
    
    k = (int)sqrt(n);    // 只需检查到 √n
    
    for (i = 2; i <= k; i++)
    {
        if (n % i == 0)
            break;
    }
    
    if (i > k)
        printf("%d 是素数\n", n);
    else
        printf("%d 不是素数\n", n);
    
    return 0;
}
```

**程序分析：**

| 要点 | 说明 |
| :--- | :--- |
| ① 循环条件 | 从 i=2 到 i≤√n 进行检查 |
| ② 判断条件 | `n % i == 0` 表示 n 能被 i 整除 |
| ③ 提前结束 | 找到因数时用 `break` 提前结束循环 |
| ④ 结果判断 | 循环结束后检查 i 是否大于 √n，判断是否为素数 |


### 5.3.3 穷举法应用 

#### 一、穷举法的概念

**穷举法**（又称枚举法）是一种通过逐一测试所有可能情况来解决问题的方法。对于可能的情况数量有限且计算机处理速度足够快时，穷举法是一种有效的解决方案。

**适用场景：**
- 问题的解空间有限
- 验证条件明确
- 需要找到所有满足条件的解


### 5.3.4 循环与选择结构的嵌套

在实际程序中，循环和选择结构经常嵌套使用，以实现更复杂的逻辑。

#### 例9：输出指定范围内的所有素数

**问题描述**：输入两个正整数 m 和 n（m < n），输出 m 到 n 之间所有的素数。

**算法分析**：
- 外层循环遍历 m 到 n 之间的每个数
- 内层循环判断当前数是否为素数

```c
#include <stdio.h>
#include <math.h>

int main()
{
    int m, n, i, j, k;
    printf("请输入范围 m n（m < n）：");
    scanf("%d%d", &m, &n);
    
    printf("%d 到 %d 之间的素数为：\n", m, n);
    for (i = m; i <= n; i++)
    {
        if (i <= 1)
            continue;    // 1不是素数，跳过
            
        k = (int)sqrt(i);
        for (j = 2; j <= k; j++)
        {
            if (i % j == 0)
                break;
        }
        
        if (j > k)
            printf("%4d", i);
    }
    printf("\n");
    
    return 0;
}
```

**运行结果：**

```
请输入范围 m n（m < n）：10 30
10 到 30 之间的素数为：
  11  13  17  19  23  29
```


#### 例10：求两个数的最大公约数

**问题描述**：输入两个正整数，求它们的最大公约数。

**算法分析**：使用**辗转相除法**（欧几里得算法）。

```c
#include <stdio.h>
int main()
{
    int a, b, r;
    printf("请输入两个正整数：");
    scanf("%d%d", &a, &b);
    
    // 使用辗转相除法
    while (b != 0)
    {
        r = a % b;
        a = b;
        b = r;
    }
    
    printf("最大公约数为：%d\n", a);
    return 0;
}
```

**运行结果：**

```
请输入两个正整数：12 18
最大公约数为：6
```


#### 例11：求两个数的最小公倍数

**问题描述**：输入两个正整数，求它们的最小公倍数。

**算法分析**：最小公倍数 = a × b / 最大公约数。

```c
#include <stdio.h>
int main()
{
    int a, b, r, lcm;
    printf("请输入两个正整数：");
    scanf("%d%d", &a, &b);
    
    // 先求最大公约数
    int x = a, y = b;
    while (y != 0)
    {
        r = x % y;
        x = y;
        y = r;
    }
    
    // 最小公倍数 = a × b / 最大公约数
    lcm = a * b / x;
    
    printf("最小公倍数为：%d\n", lcm);
    return 0;
}
```

**运行结果：**

```
请输入两个正整数：4 6
最小公倍数为：12
```


### 5.3.5 循环程序设计的常见错误

| 常见错误 | 错误示例 | 正确写法 | 说明 |
| :--- | :--- | :--- | :--- |
| **忘记赋初值** | `int i; while(i<10) {...}` | `int i=1; while(i<10) {...}` | 循环变量必须赋初值 |
| **循环条件写反** | `while (i >= 10)` | `while (i <= 10)` | 注意条件方向 |
| **缺少增值语句** | `while(i<10) { printf("%d",i); }` | `while(i<10){ printf("%d",i); i++; }` | 缺少 i++ 导致死循环 |
| **循环体未用花括号** | `while(i<10) sum=sum+i; i++;` | `while(i<10) { sum=sum+i; i++; }` | 只有第一条语句属于循环体 |
| **判断素数时忘记考虑1** | 将1判断为素数 | 排除1的情况 | 1不是素数 |


## 💡 核心考点

1. **素数判断**：从2到√n检查是否能整除，优化方法是检查到√n
2. **穷举法**：通过逐一测试所有可能情况来解决问题
3. **循环与选择嵌套**：内层循环判断素数，外层循环遍历范围
4. **辗转相除法**：求最大公约数的经典算法
5. **循环三要素**：初始化、循环条件、循环变量更新

## 🔧 上机实践

### 实践题1：完数判断

编写一个C程序，输入一个整数 n，判断它是否为**完数**。

**说明**：完数是指一个数恰好等于它的所有真因子（除了自身以外的因子）之和。例如，6 = 1 + 2 + 3，所以6是完数。

**参考代码：**

```c
#include <stdio.h>
int main()
{
    int n, i, sum = 0;
    printf("请输入一个整数：");
    scanf("%d", &n);
    
    for (i = 1; i < n; i++)
    {
        if (n % i == 0)
            sum += i;  // 累加真因子
    }
    
    if (sum == n)
        printf("%d 是完数\n", n);
    else
        printf("%d 不是完数\n", n);
    
    return 0;
}
```
