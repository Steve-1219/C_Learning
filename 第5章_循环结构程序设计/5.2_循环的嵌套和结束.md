## 📖 知识梳理

### 5.2.1 循环的嵌套

#### 一、循环嵌套的概念

一个循环体内又包含另一个完整的循环结构，称为**循环的嵌套**。内嵌的循环体中还可以再嵌套，这就是**多层循环**。

> 💡 三种循环（while、do...while、for）可以互相嵌套。

#### 二、循环嵌套的合法形式

以下是一些合法的循环嵌套形式：

**（1）while 嵌套 while**

```c
while (表达式1)
{
    // 外层循环体
    while (表达式2)
    {
        // 内层循环体
    }
}
```

**（2）do...while 嵌套 do...while**

```c
do
{
    // 外层循环体
    do
    {
        // 内层循环体
    } while (表达式2);
} while (表达式1);
```

**（3）for 嵌套 for**

```c
for (表达式1; 表达式2; 表达式3)
{
    // 外层循环体
    for (表达式4; 表达式5; 表达式6)
    {
        // 内层循环体
    }
}
```

**（4）混合嵌套**

```c
while (表达式1)
{
    // 外层循环体
    for (表达式2; 表达式3; 表达式4)
    {
        // 内层循环体
    }
}
```

```c
do
{
    // 外层循环体
    while (表达式1)
    {
        // 内层循环体
    }
} while (表达式2);
```

#### 三、循环嵌套的示例

**【例5】计算 s = 1¹ + 2² + 3³ + ... + nⁿ**

其中 n 由键盘输入（n 为整数）。

**算法分析：**
- 对于每一项 iⁱ，需要将 i 连乘 i 次
- 外层循环控制项数（从1到n）
- 内层循环控制每项的连乘次数

**程序代码：**

```c
#include <stdio.h>
int main()
{
    int i, j, n;
    long sum = 0, term;
    
    printf("请输入n的值：");
    scanf("%d", &n);
    
    for (i = 1; i <= n; i++)      // 外层循环：控制项数
    {
        term = 1;
        j = 1;
        do                         // 内层循环：计算 i 的 i 次方
        {
            term = term * i;
            j++;
        } while (j <= i);
        sum += term;               // 累加当前项
    }
    
    printf("sum = %ld\n", sum);
    return 0;
}
```

**运行结果：**

```
请输入n的值：5
sum = 3413
```

**程序分析：**

| 要点 | 说明 |
| :--- | :--- |
| ① 外循环 | 变量 i 每取一个值时，内循环变量 j 要取 i 个值 |
| ② 内循环次数 | 由外循环变量 i 决定，每次变化 |
| ③ 执行过程 | 外循环体每执行一次，内循环体要执行 i 次 |

#### 四、多重循环的使用要点

| 要点 | 说明 | 示例 |
| :--- | :--- | :--- |
| **① 变量赋初值的位置** | 只需执行一次的赋初值操作，应放在最外层循环开始执行之前 | `sum = 0` 放在 for 语句的表达式1中 |
| **② 内层变量赋初值** | 应放在外层循环体开始执行时（内循环之前） | `term = 1; j = 1;` 位于内循环之前 |
| **③ 内外循环变量不应同名** | 否则会造成循环控制的混乱，导致死循环或计算结果错误 | 外循环用 `i`，内循环用 `j` |
| **④ 循环体用花括号** | 内循环体中执行的所有语句必须用花括号括起来 | 复合语句作为循环体 |
| **⑤ 仅需一次的操作** | 应放在进入循环之前或循环结束之后 | 输入提示、读入 n、输出结果 |


### 5.2.2 提前结束循环 

在正常情况下，循环语句会根据事先指定的循环条件来执行和终止循环。但当出现某种情况导致需要提前终止循环时，可以使用 **break** 或 **continue** 语句。

#### 一、break 语句（提前终止整个循环）

`break` 语句的一般形式为：

```c
break;
```

**作用：** 使流程跳转到循环体**之外**，接着执行循环体下面的语句。即**终止整个循环**。

**图示：**

```
    ┌─────────────┐
    │  循环条件    │
    └──────┬──────┘
       真  │
           ↓
    ┌─────────────┐
    │  循环体      │
    │    ...       │
    │  break;      │ ← 遇到break，跳出整个循环
    │    ...       │
    └─────────────┘
           │
           ↓（跳出循环）
    ┌─────────────┐
    │  循环体后的   │
    │  语句        │
    └─────────────┘
```

**【例6】慈善募捐统计**

在全系1000名学生中征集慈善募捐，当总数达到10万元时就结束，统计此时捐款人数和人均捐款数。

**程序代码：**

```c
#include <stdio.h>
#define SUM 100000    // 指定符号常量SUM代表100000

int main()
{
    float amount, aver, total = 0;
    int i;
    
    for (i = 1; total < SUM && i <= 1000; i++)
    {
        printf("请输入捐款数：");
        scanf("%f", &amount);
        total = total + amount;
        if (total >= SUM)
            break;    // 达到目标，提前结束循环
    }
    
    aver = total / (i - 1);
    printf("捐款人数为：%d\n平均每人捐款：%.2f\n", i - 1, aver);
    return 0;
}
```

**运行结果：**

```
请输入捐款数：30000
请输入捐款数：25000
请输入捐款数：18000
请输入捐款数：35000
捐款人数为：4
平均每人捐款：27000.00
```

**程序分析：**

| 要点 | 说明 |
| :--- | :--- |
| ① 循环条件 | 原计划执行1000次 |
| ② 检查条件 | 每次累加捐款后检查 `total >= SUM` |
| ③ 提前结束 | 当达到目标时执行 `break`，提前结束循环 |
| ④ 循环变量 | 此时 `i` 的值为达到目标时的循环次数 |


#### 二、continue 语句（提前结束本次循环）

`continue` 语句的一般形式为：

```c
continue;
```

**作用：** 结束**本次**循环，即跳过循环体中下面尚未执行的语句，然后进行下一次是否执行循环的判断。即**提前结束本次循环，进入下一次循环**。

**图示：**

```
    ┌─────────────┐
    │  循环条件    │
    └──────┬──────┘
       真  │
           ↓
    ┌─────────────┐
    │  循环体      │
    │    ...       │
    │ continue;    │ ← 遇到continue，跳过本轮剩余语句
    │  （跳过）     │
    └─────────────┘
           │
           ↓（进入下一次循环）
    ┌─────────────┐
    │  重新判断    │
    │  循环条件    │
    └─────────────┘
```

**【例7】输出100~200之间不能被3整除的数**

**程序代码：**

```c
#include <stdio.h>
int main()
{
    int n;
    for (n = 100; n <= 200; n++)
    {
        if (n % 3 == 0)
            continue;        // 能被3整除，结束本次循环，不输出
        printf("%5d", n);    // 不能被3整除，输出
    }
    printf("\n");
    return 0;
}
```

**运行结果（部分）：**

```
100  101  103  104  106  107  109  110  112  113  115  116  118  119  121  122  124
125  127  128  130  131  133  134  136  137  139  140  142  143  145  146  148  149
151  152  154  155  157  158  160  161  163  164  166  167  169  170  172  173  175
176  178  179  181  182  184  185  187  188  190  191  193  194  196  197  199  200
```

**程序分析：**

| 要点 | 说明 |
| :--- | :--- |
| ① 判断条件 | `n % 3 == 0` 时，n 能被3整除 |
| ② continue作用 | 跳过 `printf` 语句，不输出该数 |
| ③ 循环继续 | 执行 `n++`，进入下一次循环 |
| ④ 整体效果 | 只输出不能被3整除的数 |

> 💡 **即学即练：** while循环、do...while循环和for循环，都可以用 break 语句跳出循环，用 continue 语句结束本次循环。


#### 三、break 与 continue 的对比 

| 对比项 | break | continue |
| :--- | :--- | :--- |
| **作用** | 终止整个循环 | 结束本次循环 |
| **执行后的走向** | 跳转到循环体**之外** | 跳转到下一次循环判断 |
| **是否继续循环** | **不再继续** | **继续下一次循环** |
| **适用语句** | switch、while、do...while、for | while、do...while、for（不能用于switch） |
| **循环剩余语句** | 全部跳过，不再执行 | 本次循环的剩余语句跳过，但下次继续 |
| **图示** | 跳出循环体 → 执行循环后语句 | 跳到循环条件判断 → 继续循环 |

**示例对比：**

```c
// 使用 break
for (i = 1; i <= 10; i++)
{
    if (i == 5)
        break;      // i=5时终止循环，输出：1 2 3 4
    printf("%d ", i);
}
// 输出：1 2 3 4

// 使用 continue
for (i = 1; i <= 10; i++)
{
    if (i == 5)
        continue;   // i=5时跳过，继续下一次循环，输出：1 2 3 4 6 7 8 9 10
    printf("%d ", i);
}
// 输出：1 2 3 4 6 7 8 9 10
```


## 💡 核心考点

1. **循环嵌套**：循环体内包含另一个完整的循环结构
2. **嵌套规则**：三种循环可以互相嵌套
3. **变量赋初值位置**：外层变量在循环前赋初值，内层变量在外层循环体内赋初值
4. **内外循环变量不应同名**
5. **break语句**：终止整个循环，跳出循环体
6. **continue语句**：结束本次循环，进入下一次循环
7. **break与continue的区别**：break终止整个循环，continue结束本次循环
8. **提前结束循环的两种方式**：break（强制退出）、修改循环条件（自然退出）
