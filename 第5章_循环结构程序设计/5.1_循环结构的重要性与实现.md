# 5.1 循环结构的重要性与实现

## 🎯 学习目标

- 理解循环结构在程序设计中的重要性
- 掌握 while 循环的使用方法
- 掌握 do...while 循环的使用方法
- 掌握 for 循环的使用方法
- 能够区分三种循环语句的异同点


## 📖 知识梳理

### 5.1.1 循环结构的重要性

在程序设计中，许多问题需要进行**重复处理**：

- 分别统计全班50个学生各自的平均成绩
- 求30个整数之和
- 批量处理数据

处理以上问题，最原始的方法是分别编写若干个相同或相似的语句进行处理，这种方法**工作量大、程序冗长、难以阅读和维护**。

**循环结构**（又称重复结构）可以方便地解决需要重复处理的问题。大多数的应用程序都会包含循环结构。

> 💡 循环结构的特点：**重复执行某段代码，直到满足特定条件才结束**。


### 5.1.2 while 语句

#### 一、while 语句的一般形式

```c
while (表达式)
    语句;
```

- **表达式**：循环条件表达式，用于控制循环体的执行次数
- **语句**：循环体，可以是一条语句，也可以是复合语句（需用 `{}` 括起来）

#### 二、while 语句的执行过程

```
        ┌─────────────┐
        │  表达式     │
        └──────┬──────┘
          真 /    \ 假
             /      \
          ┌─┘        └─┐
          ↓             ↓
    ┌───────────┐  ┌───────────┐
    │  循环体   │  │  退出循环  │
    └───────────┘  └───────────┘
          │
          └──────→ 返回重新判断表达式
```

> 💡 **拿分笔记**：while 循环的特点是 **先判断条件表达式，后执行循环体语句**。如果条件第一次就不满足，循环体一次也不执行。

**要构成有效循环，应当指定两个条件：**
1. **循环体**：需要重复执行的操作
2. **循环结束的条件**：在什么情况下停止重复

#### 三、while 循环程序举例

**【例1】求 1+2+3+...+100 的和**

**算法分析：**
- 这是一个累加问题，需要从1加到100，重复进行100次加法运算
- 每次加一个数，后一个数是前一个数加1

**流程图：**

```
    ┌─────────────┐
    │  sum = 0    │
    │  i = 1      │
    └──────┬──────┘
           ↓
    ┌─────────────┐
    │  i ≤ 100 ?  │ ← 循环条件
    └──────┬──────┘
      真 /    \ 假
       /      \
    ┌─┘        └─┐
    ↓             ↓
┌───────────┐  ┌───────────┐
│ sum=sum+i │  │  输出sum  │
│ i=i+1     │  └───────────┘
└───────────┘
    │
    └──────→ 返回
```

**程序代码：**

```c
#include <stdio.h>
int main()
{
    int i = 1, sum = 0;
    
    while (i <= 100)
    {
        sum = sum + i;
        i++;
    }
    
    printf("sum = %d\n", sum);
    return 0;
}
```

**运行结果：**

```
sum = 5050
```

**程序分析：**

| 要点 | 说明 |
| :--- | :--- |
| ① 赋初值 | 在进行累加前应先给 `i` 和 `sum` 赋初值，否则它们的值是不可预测的 |
| ② 复合语句 | while语句中的循环体是复合语句，用花括号括起来。若无花括号，则while语句的范围只到第一个分号处 |
| ③ 循环结束 | 循环结束的条件是 `i > 100`，因此循环体中应设置使 `i` 增值的语句（`i++`），否则循环永不结束 |


### 5.1.3 do...while 语句

#### 一、do...while 语句的一般形式

```c
do
{
    循环体语句;
} while (表达式);
```

> 💡 do...while 循环的特点是：**先无条件地执行一次循环体，然后判断循环条件是否成立**。即先执行循环体，后判断循环条件。

#### 二、do...while 语句的执行过程

```
    ┌─────────────┐
    │  循环体     │ ← 先执行一次
    └──────┬──────┘
           ↓
    ┌─────────────┐
    │  表达式     │
    └──────┬──────┘
      真 /    \ 假
       /      \
    ┌─┘        └─┐
    ↓             ↓
  (返回)    ┌───────────┐
            │  退出循环  │
            └───────────┘
```

#### 三、do...while 循环程序举例

**【例2】用 do...while 语句求 1+2+3+...+100 的和**

```c
#include <stdio.h>
int main()
{
    int i = 1, sum = 0;
    do
    {
        sum = sum + i;
        i++;
    } while (i <= 100);
    
    printf("sum = %d\n", sum);
    return 0;
}
```

#### 四、while 与 do...while 的对比

对同一个问题，可以用 while 语句处理，也可以用 do...while 语句处理，二者可以相互转换。但两者在**首次判断时存在差异**。

**示例对比：** 求 1+2+...+10 的和，但允许用户指定起始值。

**① 用 while 循环**

```c
#include <stdio.h>
int main()
{
    int i, sum = 0;
    printf("请输入 i 的值：");
    scanf("%d", &i);
    
    while (i <= 10)
    {
        sum = sum + i;
        i++;
    }
    printf("sum = %d\n", sum);
    return 0;
}
```

**② 用 do...while 循环**

```c
#include <stdio.h>
int main()
{
    int i, sum = 0;
    printf("请输入 i 的值：");
    scanf("%d", &i);
    
    do
    {
        sum = sum + i;
        i++;
    } while (i <= 10);
    
    printf("sum = %d\n", sum);
    return 0;
}
```

**运行结果对比：**

| 输入值 | while 结果 | do...while 结果 | 说明 |
| :---: | :---: | :---: | :--- |
| i=1 | sum=55 | sum=55 | 两者相同 |
| i=11 | sum=0 | sum=11 | **不同！** while先判断，i≤10为假，不执行循环体；do...while先执行循环体一次，再判断 |

> ⚠️ **重要区别**：当循环条件第一次就不满足时，while 循环一次也不执行，do...while 循环执行一次。


### 5.1.4 for 语句

#### 一、for 语句的一般形式

```c
for (表达式1; 表达式2; 表达式3)
    语句;
```

**三个表达式的作用：**

| 表达式 | 作用 | 执行时机 |
| :--- | :--- | :--- |
| **表达式1** | 设置初始条件（循环变量赋初值） | 只执行一次 |
| **表达式2** | 循环条件表达式，判断是否继续循环 | 每次循环前执行 |
| **表达式3** | 循环调整（循环变量增值） | 每次循环体执行完后执行 |

**理解方式：**

```c
for (循环变量赋初值; 循环条件; 循环变量增值)
    语句;
```

#### 二、for 语句的执行过程

```
    ┌─────────────┐
    │  求解表达式1 │ ← 只执行一次
    └──────┬──────┘
           ↓
    ┌─────────────┐
    │  求解表达式2 │
    └──────┬──────┘
      真 /    \ 假
       /      \
    ┌─┘        └─┐
    ↓             ↓
┌───────────┐  ┌───────────┐
│  执行循环体 │  │  退出循环  │
└──────┬──────┘  └───────────┘
       ↓
┌─────────────┐
│  求解表达式3 │
└──────┬──────┘
       │
       └──────→ 返回求解表达式2
```

**for 语句的等价 while 形式：**

```c
表达式1;
while (表达式2)
{
    语句;
    表达式3;
}
```

#### 三、for 循环程序举例

**【例3】求 1+2+3+...+n 的和**

```c
#include <stdio.h>
int main()
{
    int sum = 0, i, n;
    printf("请输入n的值：");
    scanf("%d", &n);
    
    for (i = 1; i <= n; i++)
        sum += i;
    
    printf("sum = %d\n", sum);
    return 0;
}
```

**运行结果（输入100）：**

```
请输入n的值：100
sum = 5050
```

#### 四、for 语句的省略形式

| 省略形式 | 写法 | 说明 |
| :--- | :--- | :--- |
| **省略表达式1** | `for(; i<=100; i++)` | 需要在for之前给循环变量赋初值 |
| **省略表达式2** | `for(i=1; ; i++)` | 认为表达式2始终为真，可能造成无限循环 |
| **省略表达式3** | `for(i=1; i<=100; )` | 需要在循环体中使循环变量增值 |
| **省略表达式1和3** | `for(; i<=100; )` | 需在for前赋初值，在循环体中增值 |
| **表达式1和3可为逗号表达式** | `for(i=0,j=0; i<10; i++,j++)` | 多个表达式用逗号分隔 |

**示例：** 完全省略表达式2（相当于死循环）

```c
for (i=1; ; i++)
    sum = sum + i;
// 等价于：
i = 1;
while (1)
{
    sum = sum + i;
    i++;
}
```

> ⚠️ **注意**：for语句中的3个表达式都可以省略，但**分号（;）不能省略**。省略表达式2时，循环条件始终为真（无限循环）。


### 5.1.5 三种循环语句的比较

| 对比项 | while | do...while | for |
| :--- | :--- | :--- | :--- |
| **判断时机** | 先判断后执行 | 先执行后判断 | 先判断后执行 |
| **最少执行次数** | 0次 | 1次 | 0次 |
| **适用场景** | 循环次数不确定 | 至少需要执行一次 | 循环次数已知 |
| **循环变量初始化** | 在while之前完成 | 在do...while之前完成 | 在表达式1中完成 |
| **循环变量增值** | 在循环体中完成 | 在循环体中完成 | 在表达式3中完成 |
| **功能** | 较强 | 较强 | **最强**（最灵活） |

**选择建议：**

| 场景 | 推荐选择 |
| :--- | :--- |
| 已知循环次数 | **for** 更方便、更清晰 |
| 只知道循环结束条件，不知道次数 | **while** 更简洁 |
| 至少需要执行一次循环体 | **do...while** |


## 💡 核心考点

1. **循环结构的重要性**：用于解决需要重复处理的问题
2. **while循环**：先判断后执行，最少执行0次
3. **do...while循环**：先执行后判断，最少执行1次
4. **for循环**：三个表达式（初始化、条件、增值），最灵活
5. **循环结束条件**：必须在循环体内设置使循环趋于结束的语句
6. **无限循环**：循环条件始终为真时出现
7. **三种循环的转换**：一般情况下可以相互代替
8. **for的省略形式**：表达式可省略，但分号不能省略
