## 📖 知识梳理

### 2.1.1 数据的存储单位和地址

#### 一、二进制存储原理

计算机的工作基于**二进制原理**进行。在计算机内，不管是什么样的数据，都采用**二进制编码**形式表示和处理。

计算机的存储器由半导体集成电路构成，包括多个脉冲电路单元（二极管元件），每一个二极管元件如同一个开关，通常只有两种状态："导通"与"截止"，即电脉冲的"有"和"无"，可以用二进制码的两个数码 **1** 和 **0** 来表示。

当用户向计算机输入数据时，计算机先把数据转换为二进制形式，根据其每一位是0或1，使相应的电子元件设置为"导通"或"截止"状态。

> 💡 **拿分笔记**：二进制数每一位的值只有0和1两种可能，它的特点是：**逢二进一，借一当二**。

#### 二、存储单位

| 单位                | 名称 | 说明                                                         |
| :------------------ | :--- | :----------------------------------------------------------- |
| **位（bit，b）**    | 比特 | 每一个二极管元件称为一个"二进制位"，是计算机存储信息的**最小单位**，用**小写字母 b** 表示。其值为0或1，表示两种状态 |
| **字节（Byte，B）** | 字节 | 1 Byte = 8 bit，是计算机存储信息的**基本单位**               |
| **千字节（KB）**    |      | 1 KB = 1024 B                                                |
| **兆字节（MB）**    |      | 1 MB = 1024 KB                                               |
| **吉字节（GB）**    |      | 1 GB = 1024 MB                                               |
| **太字节（TB）**    |      | 1 TB = 1024 GB                                               |

#### 三、存储地址

在计算机中，每个存储单元都有一个编号，这个编号称为该存储单元的**地址**。通过地址可以快速访问对应的存储单元。

- **变量名**：是存储单元的符号地址，便于程序员记忆和使用
- **变量地址**：是存储单元的实际物理地址编号


### 2.1.2 不同类型数据的存储方式

#### 一、整数的存储

在计算机的存储器中，整数（正数和负数）都是以**补码**的形式存放的。正数的补码与原码、反码相同。

**1. 原码表示法**

把最高位（左边第一位）定义为**符号位**：
- 用 **"0"** 表示正数
- 用 **"1"** 表示负数
- 其他位用来存放数值

|        存储字节数        | 最大值           | 最大值计算       |
| :----------------------: | :--------------- | :--------------- |
|         1个字节          | 127              | \( 2^7 - 1 \)    |
|         2个字节          | 32767            | \( 2^{15} - 1 \) |
| 4个字节（现代C编译系统） | \( 2^{31} - 1 \) | \( 2^{31} - 1 \) |

**2. 补码的求法（以 -9 为例，用1个字节存储）**

```
① 先写出该数绝对值的二进制形式：
   | -9 | = 9 = 00001001

② 对该二进制数按位取反（0变1，1变0）得到反码：
   00001001 → 11110110

③ 反码加1得到补码：
   11110110 + 1 = 11110111
```

> 💡 **记忆要点**：负数的补码 = 原码按位取反后加1（符号位不变）。正数的补码 = 原码本身。

#### 二、实数的存储

对于实数，不是所有的十进制数都能精确地转换为二进制数（如20.15），所以实数不能采取整数的存储形式。

**1. 标准化指数形式**

实数一律采用**指数形式**存储。例如：20.15 可以写成标准化指数形式 \( 0.2015 \times 10^2 \)，它包括前后两个部分：
- **数值部分**：0.2015（小数部分）
- **指数部分**：\( 10^2 \)（指数部分）

> 💡 **注意**：数值部分是一个小数，小数点前的数字是0，小数点后的第一位数字不为0。

**2. 存储方式**

在计算机的存储器中，实数是以标准化指数形式存放的，一般用 **4个字节** 存储一个实数，其中：
- 3个字节存储**数值部分**
- 1个字节存储**指数部分**

#### 三、字符的存储

字符包括字母（如A、a）、专用字符（如$、@、%、#）等。由于存储单元只能存储二进制信息，因此要对其进行二进制编码表示。

**1. ASCII码（美国信息交换标准代码）**

ASCII码是国际上使用最广泛的一种字符编码，已被国际标准化组织（ISO）指定为国际标准。

| 特性         | 说明                            |
| :----------- | :------------------------------ |
| ASCII码范围  | 0~127的字符                     |
| 占用二进制位 | 7个二进制位                     |
| 可表示字符数 | \( 2^7 = 128 \) 个              |
| 实际存储     | 用1个字节（8位）表示，最高位为0 |

**2. 常见字符的ASCII码（必须熟记）**

| 字符              | 二进制形式 | 十进制值 | 存储单元中的信息（8位） |
| :---------------- | :--------- | :------: | :---------------------- |
| **'A'**           | 1000001    |  **65**  | 01000001                |
| **'a'**           | 1100001    |  **97**  | 01100001                |
| **'0'**（数字零） | 0110000    |  **48**  | 00110000                |
| **' '**（空格）   | 0100000    |  **32**  | 00100000                |

> 💡 **记忆技巧**：记住 'A'=65，'a'=97（大写字母和小写字母相差32），'0'=48，空格=32。


### 2.1.3 数据的表现形式 

在计算机高级语言中，数据有两种表现形式：**常量**和**变量**。

#### 一、常量

**常量**是指在程序运行过程中，其值**不能被改变**的量。数值常量就是数学中的常数。

**1. 常量的分类**

| 常量类型       | 说明                             | 示例                       |
| :------------- | :------------------------------- | :------------------------- |
| **整型常量**   | 整数                             | `16`、`22`、`-5`、`0`      |
| **实型常量**   | 带小数点的数                     | `3.14`、`-2.5`、`0.0`      |
| **字符常量**   | 用单引号括起来的字母、数字或符号 | `'A'`、`'a'`、`'0'`、`'$'` |
| **字符串常量** | 用双引号括起来的字符序列         | `"Hello"`、`"C语言"`       |
| **符号常量**   | 用 `#define` 定义的常量          | `#define PI 3.14159`       |

#### 二、变量

**1. 变量的概念**

**变量**是指在程序运行过程中其值**可以改变**的量，可以在程序运行过程中给它们赋予新的值。

变量代表一个有名字的、具有特定属性的一个**存储单元**，它用来存放数据，也就是存放变量的值。

> 💡 **名师点拨**：变量名、变量地址、存储单元、变量值的相互关系：
>
> ```
> 变量名（如 a） → 变量地址（如 1001） → 存储单元 → 存放变量值（如 3）
> ```
>
> 变量名代表一个变量地址（可以说变量名是存储单元的**符号地址**），通过变量地址可以找到相应的存储单元，可以在存储单元中存放或取出变量值。

**2. 变量的定义**

变量**必须先定义，后使用**。定义变量的一般形式是：

```c
类型名 变量名;
```

若需要同时定义多个同类型的变量，在两个变量间用**逗号**分隔：

```c
类型名 变量名1, 变量名2, 变量名3;
```

**示例：**

```c
int a;              // 定义一个整型变量 a
float b, c;         // 同时定义两个实型变量 b 和 c
char ch;            // 定义一个字符变量 ch
```

> 💡 **注意**：类型是抽象的，不占用存储空间，不能用来存储数据；**变量是实际存在的，占用存储空间**，可以用来存储数据。

**3. 变量的命名规则**

在计算机高级语言中，用来对变量、符号常量、函数、数组、类型等命名的有效字符序列统称为**标识符**。变量名是标识符的一种。

变量名的命名应遵守以下规则：

| 规则                 | 说明                                                         | 合法示例                                 | 非法示例                                                     |
| :------------------- | :----------------------------------------------------------- | :--------------------------------------- | :----------------------------------------------------------- |
| **① 组成字符**       | 只能由**字母、数字和下划线**3种字符组成，且第1个字符必须为**字母或下划线** | `book32`、`sum`、`_name`、`student_name` | `3day`（数字开头）、`student-name`（含连字符）、`#name`（含#）、`book's`（含'） |
| **② 区分大小写**     | `Name` 和 `name` 是两个不同的变量名                          | `Name`、`name`                           | —                                                            |
| **③ 不能使用关键字** | 不允许使用C语言关键字为变量命名                              | `sum`、`value`                           | `int`、`float`、`return`                                     |
| **④ 长度限制**       | 一般允许32个字符或更多                                       | `student_grade_score`                    | 过长（已不常见）                                             |
| **⑤ 见名知意**       | 应使用有含义的英文单词                                       | `age`、`score`、`sum`                    | `a1`、`x2`（不推荐）                                         |
| **⑥ 唯一性**         | 同一函数中不能给不同变量取相同的名字                         | —                                        | —                                                            |

**4. C语言关键字（不能用作变量名）**

关键字是C语言规定的具有特殊意义的单词。以下列出常用关键字：

| 类别         | 关键字                                                       |
| :----------- | :----------------------------------------------------------- |
| **数据类型** | `int`、`char`、`float`、`double`、`void`、`short`、`long`、`unsigned`、`signed`、`struct`、`union`、`enum` |
| **控制语句** | `if`、`else`、`switch`、`case`、`default`、`while`、`do`、`for`、`break`、`continue`、`return`、`goto` |
| **存储类型** | `auto`、`static`、`register`、`extern`                       |
| **其他**     | `const`、`sizeof`、`typedef`、`volatile`、`inline`           |

> ⚠️ **注意**：以上关键字都有特殊含义，**不能用作变量名**。


## 💡 核心考点

1. **二进制原理**：计算机采用二进制编码表示和处理数据
2. **存储单位**：位（bit）是最小单位，字节（Byte）是基本单位（1B=8bit）
3. **整数存储**：用补码形式存放，最高位为符号位（0正1负）
4. **负数的补码**：原码按位取反 +1
5. **实数存储**：采用标准化指数形式存储（4个字节）
6. **字符存储**：用ASCII码存储，1个字节，最高位为0
7. **常见ASCII码**：'A'=65、'a'=97、'0'=48、空格=32
8. **常量与变量**：常量值不能改变，变量值可以改变
9. **变量的定义**：类型名 变量名；必须先定义后使用
10. **命名规则**：字母/数字/下划线组成，数字不能开头，区分大小写，不能用关键字
