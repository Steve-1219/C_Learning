```
                       ┌─────────────┐
                       │  数据类型   │
                       └──────┬──────┘
```

### 2.2.2 整型数据 

#### 一、整型常量

在C语言中，整型常量可用以下 **3种形式** 表示：

| 进制             | 表示方法          | 示例             | 对应十进制值 |
| :--------------- | :---------------- | :--------------- | :----------: |
| **十进制整数**   | 直接书写          | `12`、`-23`、`0` |  12、-23、0  |
| **八进制整数**   | 以数字 **0** 开头 | `012`            |      10      |
| **十六进制整数** | 以 **0x** 开头    | `0x7E`           |     126      |

**① 十进制整数**

与日常生活中使用相同，如 `1`、`-23`、`100` 等。

**② 八进制整数**

八进制数的特点是：**逢八进一，借一当八**。在程序中凡是以 `0` 开头的数都认为是八进制数。

例如：`012` 表示八进制数12，其值为：
$$ 1 \times 8^1 + 2 \times 8^0 = 8 + 2 = 10 $$

> ⚠️ **注意**：`-012` 表示八进制数 `-12`，等于十进制数 `-10`。

**③ 十六进制整数**

十六进制数的特点是：**逢十六进一，借一当十六**。在十六进制数中可以用数字 `0~9` 和字母 `A~F` 来表示，规定用字母 `A、B、C、D、E、F` 分别代表 `10、11、12、13、14、15`。在程序中凡是以 `0x` 开头的数都认为是十六进制数。

例如：`0x12` 表示十六进制数12，其值为：
$$ 1 \times 16^1 + 2 \times 16^0 = 16 + 2 = 18 $$

> ⚠️ **注意**：`-0x12` 表示十六进制数 `-12`，等于十进制数 `-18`。


#### 二、整型变量的类型

整型变量用来存放整型常量。常用的整型变量如下表所示：

| 整型类型     | 类型名                         | 描述                                                         |
| :----------- | :----------------------------- | :----------------------------------------------------------- |
| **基本整型** | `int`                          | 用得最多的类型，Visual C++为每个int型数据分配 **4个字节**（32位） |
| **短整型**   | `short int` 或 `short`         | Visual C++ 6.0分配 **2个字节**                               |
| **长整型**   | `long int` 或 `long`           | Visual C++ 6.0分配 **4个字节**                               |
| **双长整型** | `long long int` 或 `long long` | C99新增，分配 **8个字节**                                    |

> ⚠️ **重要**：C标准并未规定各种整型数据在内存中所占的字节数，只要求：
> ```
> sizeof(short) ≤ sizeof(int) ≤ sizeof(long) ≤ sizeof(long long)
> ```

#### 三、有符号与无符号整型

在实际应用中，有的数据的范围常常只有正值（如学号、年龄、库存量等）。为了充分利用变量值范围，可以将变量定义为 **无符号类型**。

| 修饰符     | 含义   | 说明                                       |
| :--------- | :----- | :----------------------------------------- |
| `signed`   | 有符号 | 存储单元中最高位表示符号（0正1负），可省   |
| `unsigned` | 无符号 | 全部二进制位都用来存放数值，**不能赋负值** |

**整型数据扩展为8种：**

| 类型           | 类型名                     |
| :------------- | :------------------------- |
| 有符号基本整型 | `(signed) int`             |
| 无符号基本整型 | `unsigned int`             |
| 有符号短整型   | `(signed) short (int)`     |
| 无符号短整型   | `unsigned short (int)`     |
| 有符号长整型   | `(signed) long (int)`      |
| 无符号长整型   | `unsigned long (int)`      |
| 有符号双长整型 | `(signed) long long (int)` |
| 无符号双长整型 | `unsigned long long (int)` |

> 💡 **名师圈点**：只有整型（包括字符型）数据可以加 `signed` 或 `unsigned` 修饰符，**实型数据则不能**。在将一个变量定义为无符号整型后，**不应对其赋予一个负值**，否则会得到错误结果。

#### 四、整型数据占用的存储空间和取值范围

| 类型                                   | 字节数 | 取值范围                                  |
| :------------------------------------- | :----: | :---------------------------------------- |
| `int`（基本整型）                      |   4    | $-2^{31}$ ~ $(2^{31}-1)$                  |
| `unsigned int`（无符号基本整型）       |   4    | $0$ ~ $(2^{32}-1)$                        |
| `short`（短整型）                      |   2    | $-2^{15}$ ~ $(2^{15}-1)$，即 -32768~32767 |
| `unsigned short`（无符号短整型）       |   2    | $0$ ~ $(2^{16}-1)$，即 0~65535            |
| `long`（长整型）                       |   4    | $-2^{31}$ ~ $(2^{31}-1)$                  |
| `unsigned long`（无符号长整型）        |   4    | $0$ ~ $(2^{32}-1)$                        |
| `long long`（双长整型）                |   8    | $-2^{63}$ ~ $(2^{63}-1)$                  |
| `unsigned long long`（无符号双长整型） |   8    | $0$ ~ $(2^{64}-1)$                        |

#### 五、求长度的关键字 sizeof

`sizeof` 是C语言中的**关键字**，不是函数名。`sizeof()` 是C语言提供的专门用于测量类型或变量长度的运算符，用法是：

```c
sizeof(类型名)    // 或
sizeof(变量名)
```

**示例：**

```c
printf("%d, %d, %d\n", sizeof(int), sizeof(short), sizeof(long));
```

#### 六、整型数据运算程序举例

**【例3】设长方体的长 a=5，宽 b=3，高 c=2，求长方体的体积。**

**程序代码：**

```c
#include <stdio.h>
int main()
{
    int a = 5, b = 3, c = 2, v;  // 定义整型变量并赋初值
    v = a * b * c;               // 计算体积
    printf("长方体的体积为：%d\n", v);
    return 0;
}
```

**运行结果：**

```
长方体的体积为：30
```

**程序分析：**

| 行号  | 代码                                 | 说明                                            |
| :---: | :----------------------------------- | :---------------------------------------------- |
| 第4行 | `int a=5, b=3, c=2, v;`              | 定义4个整型变量并同时赋初值，多个变量用逗号分隔 |
| 第5行 | `v = a * b * c;`                     | 根据长方体体积公式进行计算                      |
| 第6行 | `printf("长方体的体积为：%d\n", v);` | `%d` 用于输出十进制整数                         |


### 2.2.3 实型数据 

#### 一、实型常量

实数在计算机中常称为浮点数，有以下两种表示形式：

| 表示形式           | 说明                                 | 示例                                                         |
| :----------------- | :----------------------------------- | :----------------------------------------------------------- |
| **十进制小数形式** | 由数字和小数点组成                   | `123.456`、`0.345`、`0.0`、`-56.79`、`12.0`                  |
| **指数形式**       | 以字母 `e` 或 `E` 代表以10为底的指数 | `12.34e3`（$12.34 \times 10^3$）、`-12.34e-3`（$-12.34 \times 10^{-3}$） |

> ⚠️ **名师圈点**：
> - `e` 或 `E` 之前**必须有数字**
> - `e` 或 `E` 后面**必须为整数**
> - 错误写法：`e4`、`12e2.5`

#### 二、实型变量

| 类型           | 类型名        |   Visual C++ 6.0分配字节   |
| :------------- | :------------ | :------------------------: |
| 单精度浮点型   | `float`       |          4个字节           |
| 双精度浮点型   | `double`      |          8个字节           |
| 长双精度浮点型 | `long double` | 8个字节（Turbo C为16字节） |

> 💡 **建议**：在对精度要求较高时，应将实型变量定义为 `double` 型，以避免精度损失。

#### 三、实型数据的舍入误差

由于实型变量在内存中的存储单元是由有限个字节组成的，能提供的有效数字是有限的，超出有效位数的数字将被舍去，因此可能会产生误差。

**示例：**

```c
float a = 1212.1415926;
printf("a = %f\n", a);
```

**输出结果：**

```
a = 1212.141602
```

> 💡 **解决办法**：在常量末尾加 `f` 或 `F`，强制指定为 `float` 型常量：
> ```c
> float a = 1212.1415926f;  // 编译时不会出现"警告"
> ```

#### 四、实型数据运算程序举例

**【例3】设圆柱底面半径 r=1.5，圆柱高 h=3，求圆柱体积。**

**程序代码：**

```c
#include <stdio.h>
int main()
{
    double h = 3, r = 1.5, pi = 3.1415926, v;  // 定义实型变量并赋初值
    v = pi * r * r * h;                         // 计算圆柱体积
    printf("圆柱体积为：%f\n", v);
    return 0;
}
```

**运行结果：**

```
圆柱体积为：21.205749
```

**程序分析：**

| 行号  | 代码                                  | 说明                                             |
| :---: | :------------------------------------ | :----------------------------------------------- |
| 第4行 | `double h=3, r=1.5, pi=3.1415926, v;` | 定义4个实型变量并赋初值，`double` 型避免精度损失 |
| 第5行 | `v = pi * r * r * h;`                 | 根据圆柱体积公式计算                             |
| 第6行 | `printf("圆柱体积为：%f\n", v);`      | `%f` 用于输出实数                                |


### 2.2.4 字符型数据 

#### 一、字符常量

字符常量必须用**单撇号**括起来，如 `'a'`。单撇号只是分界符，并不是字符常量的一部分，字符常量**只能是一个字符**。`'a'` 和 `'A'` 是不同的字符常量。

字符常量在计算机存储单元中存放的是其 **ASCII代码**（以二进制形式存放）。

#### 二、转义字符

转义字符是以反斜杠 `\` 开头的特殊字符，用来表示不能直接显示的控制信号。

| 转义字符 | 字符值         | 输出结果                             |
| :------- | :------------- | :----------------------------------- |
| `\'`     | 一个单撇号 `'` | 输出 `'`                             |
| `\"`     | 一个双撇号 `"` | 输出 `"`                             |
| `\?`     | 一个问号 `?`   | 输出 `?`                             |
| `\\`     | 一个反斜杠 `\` | 输出 `\`                             |
| `\a`     | 警告           | 产生声音或视觉信号                   |
| `\b`     | 退格           | 将当前位置后退一个字符               |
| `\n`     | 换行           | 将当前位置移到下一行的开头           |
| `\t`     | 水平制表符     | 将当前位置移到下一个输出区           |
| `\r`     | 回车           | 将当前位置移到本行的开头             |
| `\v`     | 垂直制表符     | 将当前位置移到下一个垂直制表符的位置 |
| `\f`     | 换页           | 将当前位置移到下一页的开头           |

> ⚠️ **注意**：转义字符必须以反斜杠 `\` 作为开头的标志，其后只能有一个字符或代表字符的八进制/十六进制代码，如 `'\nm'` 是不合法的。

**特殊转义字符：**

| 转义字符 | 说明                         | 示例              |
| :------- | :--------------------------- | :---------------- |
| `\ddd`   | 1~3位八进制数对应的ASCII字符 | `\101` 代表 `'A'` |
| `\xhh`   | 十六进制数对应的ASCII字符    | `\x41` 代表 `'A'` |

#### 三、字符变量

字符变量用来存放字符常量，**只能存放一个字符**。用类型符 `char` 定义：

```c
char 字符变量列表;
```

**示例：**

```c
char c1, c2;
c1 = '?';
c2 = 'a';
```

> 💡 **核心要点**：
> - 字符型数据属于整型，0~127之间的整数可赋给字符变量
> - 以下两条语句等价：
>   ```c
>   char c = '?';
>   char c = 63;   // 63是'?'的ASCII码
>   ```
> - 输出时可选择 `%d`（输出ASCII码）或 `%c`（输出字符本身）
> - 字符类型也可使用 `signed` 和 `unsigned` 修饰符

**【例4】利用字符的ASCII码将字符 `'d'` 转换为字符 `'D'`。**

小写字母的ASCII码比对应的大写字母的ASCII码大 **32**。

**程序代码：**

```c
#include <stdio.h>
int main()
{
    char c1 = 'd';
    c1 = c1 - 32;
    printf("%c\n", c1);
    return 0;
}
```

**运行结果：**

```
D
```

**程序分析：**

| 行号  | 代码                  | 说明                                   |
| :---: | :-------------------- | :------------------------------------- |
| 第4行 | `char c1 = 'd';`      | 将 `'d'` 的ASCII码（100）存入变量 `c1` |
| 第5行 | `c1 = c1 - 32;`       | 执行 `100 - 32 = 68`，存入 `c1`        |
| 第6行 | `printf("%c\n", c1);` | 以 `%c` 格式输出，将68转换为字符 `'D'` |

#### 四、字符串常量

字符串常量是用一对**双引号**括起来的字符序列，如 `"boy"`、`"123"`、`"a"`。

**核心区别：**

| 对比项         | 字符常量              | 字符串常量                       |
| :------------- | :-------------------- | :------------------------------- |
| 定界符         | 单撇号 `'`            | 双引号 `"`                       |
| 长度           | 只能一个字符          | 可以多个字符                     |
| 存储           | 存储ASCII码           | 末尾自动添加 `\0`                |
| 赋值给char变量 | 可以：`char c = 'a';` | **不可以**：`char c = "a";` 错误 |

> 💡 **索引笔记**：`\0` 是一个ASCII码为0的字符，系统在字符串常量结尾**自动添加**。字符串 `"GREEN"` 在内存中的存储形式：
>
> ```
> ┌───┬───┬───┬───┬───┬────┐
> │ G │ R │ E │ E │ N │ \0 │
> └───┴───┴───┴───┴───┴────┘
> ```
>
> 所以字符串 `"student"` 在内存中占 **8个字节**（7个字符 + 1个 `\0`）。


### 2.2.5 符号常量和const常量

#### 一、符号常量（#define）

**1. 符号常量的定义**

常量可以用一个标识符来表示，称为**符号常量**（或宏常量）。用 `#define` 进行定义：

```c
#define PI 3.14    // 注意行末没有分号
```

经过以上指定后，本文中从此行开始所有的 `PI` 都代表 `3.14`。在编译前，预处理器会把所有 `PI` 全部替换为 `3.14`。

> ⚠️ **注意**：`#define` 是一个**编译预处理命令**，不是C语句，该行的末尾**没有分号**。替换时不做任何语法检查，若加分号，则替换时会连同分号一起替换。

**2. 使用符号常量的优点**

| 优点         | 说明                                          |
| :----------- | :-------------------------------------------- |
| **含义清楚** | 从 `PI` 可知代表圆周率，做到"见名知意"        |
| **一改全改** | 需要改变常量值时，只需修改 `#define` 一处即可 |

**【例5】使用符号常量定义π，编程从键盘输入圆的半径 r，计算并输出圆的周长和面积。**

```c
#include <stdio.h>
#define PI 3.14

int main()
{
    float r;
    printf("请输入圆的半径：");
    scanf("%f", &r);
    printf("圆的周长为：%f\n", 2 * PI * r);
    printf("圆的面积为：%f\n", PI * r * r);
    return 0;
}
```

**运行结果：**

```
请输入圆的半径：3
圆的周长为：18.840000
圆的面积为：28.260000
```

> 💡 **提示**：符号常量只是一个**符号**，不是变量，**不占存储单元**。不能对符号常量指定类型、赋值。习惯上符号常量用**大写**，变量名用**小写**。

#### 二、const常量

**1. const常量的定义**

`const` 是C语言的一个关键字，用于定义**只读变量**（常变量）。定义形式为：

```c
const 类型名 变量名 = 常量值;
```

**示例：**

```c
const double PI = 3.14159;
const int MAX_SIZE = 100;
const char NEWLINE = '\n';
```

**2. const常量的特点**

| 特点         | 说明                                 |
| :----------- | :----------------------------------- |
| **类型安全** | 有明确的数据类型，编译时进行类型检查 |
| **占用存储** | 是变量，占用存储单元                 |
| **只读属性** | 定义时必须初始化，之后不能再修改     |
| **作用域**   | 遵循变量的作用域规则                 |

**3. 错误用法示例**

```c
const double PI;     // 错误：const常量必须初始化
PI = 3.14159;        // 错误：const常量不能赋值
```

#### 三、符号常量与const常量的区别 

| 对比项         | 符号常量（#define）      | const常量            |
| :------------- | :----------------------- | :------------------- |
| **本质**       | 编译预处理指令，文本替换 | 只读变量             |
| **类型检查**   | 无类型检查               | 有类型检查           |
| **是否占存储** | 不占存储单元             | 占存储单元           |
| **作用域**     | 从定义处到文件末尾       | 遵循变量的作用域规则 |
| **可修改性**   | 不能修改                 | 不能修改             |
| **调试**       | 无法调试查看             | 可以调试查看         |
| **内存分配**   | 预编译时展开             | 编译时分配内存       |


### 2.2.6 进制转换 

#### 一、进位计数制的基本概念

**1. 进制的基本要素**

| 概念           | 说明                                                      |
| :------------- | :-------------------------------------------------------- |
| **基数（r）**  | 该进制所用的数码个数。如二进制的基数为2，十进制的基数为10 |
| **位权（权）** | 不同位置上的数码所代表的值。位权 = 基数 的 位数次方       |
| **数码**       | 该进制所用的数字符号。如十进制的数码为0~9                 |

> 💡 **通用公式**：任意 r 进制数的值 = 各位数码 × 对应位权 之和

**2. 四种进制的对应关系**

| 十进制 | 二进制 | 八进制 | 十六进制 |
| :----: | :----: | :----: | :------: |
|   0    |   0    |   0    |    0     |
|   1    |   1    |   1    |    1     |

#### 二、r进制数转换为十进制数

**方法**：**位权展开求和**。即将各位数码乘以对应的位权，再求和。

**公式：**
$$ (N)_r = \sum_{i=-m}^{n-1} a_i \times r^i $$

**【例1】将二进制数 (1011.01)₂ 转换为十进制数**

$$ (1011.01)_2 = 1 \times 2^3 + 0 \times 2^2 + 1 \times 2^1 + 1 \times 2^0 + 0 \times 2^{-1} + 1 \times 2^{-2} $$
$$ = 8 + 0 + 2 + 1 + 0 + 0.25 = 11.25 $$

**【例2】将八进制数 (543.2)₈ 转换为十进制数**

$$ (543.2)_8 = 5 \times 8^2 + 4 \times 8^1 + 3 \times 8^0 + 2 \times 8^{-1} $$
$$ = 5 \times 64 + 4 \times 8 + 3 + 2 \times 0.125 = 320 + 32 + 3 + 0.25 = 355.25 $$

**【例3】将十六进制数 (10B.8)₁₆ 转换为十进制数**

$$ (10B.8)_{16} = 1 \times 16^2 + 0 \times 16^1 + 11 \times 16^0 + 8 \times 16^{-1} $$
$$ = 256 + 0 + 11 + 0.5 = 267.5 $$

#### 三、十进制数转换为r进制数

十进制数的整数部分和小数部分需分别转换，再组合。

**整数部分**：采用 **除 r 取余法**。即逐次除以 r，直至商为0，所得的余数**倒排**。

**小数部分**：采用 **乘 r 取整法**。即逐次乘以 r，从每次乘积的整数部分得到数码，直至小数部分为0或满足精度要求。

**【例4】将十进制数 117 转换为二进制数**

```
         ┌───────┐
    2 │ 117 │ 余数
      ├───────┤
    2 │  58  │  1  ← 低位
      ├───────┤
    2 │  29  │  0
      ├───────┤
    2 │  14  │  1
      ├───────┤
    2 │   7  │  0
      ├───────┤
    2 │   3  │  1
      ├───────┤
    2 │   1  │  1
      ├───────┤
    2 │   0  │  1  ← 高位
      └───────┘
        倒排得：1110101
```

所以：**117 = (1110101)₂**

**【例5】将十进制数 56.6 转换为二进制数**

**步骤一：整数部分 56 转换为二进制**

```
    2 │ 56 │ 余数
      ├──────┤
    2 │ 28 │  0  ← 低位
      ├──────┤
    2 │ 14 │  0
      ├──────┤
    2 │  7 │  0
      ├──────┤
    2 │  3 │  1
      ├──────┤
    2 │  1 │  1
      ├──────┤
    2 │  0 │  1  ← 高位
      └──────┘
```

得：**56 = (111000)₂**

**步骤二：小数部分 0.6 转换为二进制**

```
0.6 × 2 = 1.2  →  取1，小数部分为0.2
0.2 × 2 = 0.4  →  取0，小数部分为0.4
0.4 × 2 = 0.8  →  取0，小数部分为0.8
0.8 × 2 = 1.6  →  取1，小数部分为0.6
0.6 × 2 = 1.2  →  取1，小数部分为0.2
  ...（无限循环）
```

得：**0.6 = (0.1001...)₂**

所以：**56.6 = (111000.1001...)₂**（精确值无法完全转换）

**【例6】将十进制数 117 转换为八进制数**

```
    8 │ 117 │ 余数
      ├─────┤
    8 │  14 │  5  ← 低位
      ├─────┤
    8 │   1 │  6
      ├─────┤
    8 │   0 │  1  ← 高位
      └─────┘
```

得：**117 = (165)₈**

**【例7】将十进制数 117 转换为十六进制数**

```
   16 │ 117 │ 余数
      ├─────┤
   16 │  7  │  5  ← 低位
      ├─────┤
   16 │  0  │  7  ← 高位
      └─────┘
```

得：**117 = (75)₁₆**

#### 四、二进制与八进制、十六进制的相互转换

> 💡 **转换依据**：由于 $2^3 = 8$，$2^4 = 16$，所以：
> - 1位八进制数 = 3位二进制数
> - 1位十六进制数 = 4位二进制数

**（1）二进制数 → 八进制数**

方法：从小数点开始，整数部分向左每3位一组，不足3位的补0；小数部分向右每3位一组，不足3位的补0。每组转换成对应的八进制数。

**【例8】将 (11111101.101)₂ 转换为八进制数**

```
  011   111   101   101
   ↓     ↓     ↓     ↓
   3     7     5     5
```

所以：**(11111101.101)₂ = (375.5)₈**

**（2）八进制数 → 二进制数**

方法：将每一位八进制数转换成对应的3位二进制数，依次连接即可。

**【例9】将 (52.3)₈ 转换为二进制数**

```
   5      2      3
   ↓      ↓      ↓
  101    010    011
```

所以：**(52.3)₈ = (101010.011)₂**

**（3）二进制数 → 十六进制数**

方法：从小数点开始，整数部分向左每4位一组，不足4位的补0；小数部分向右每4位一组，不足4位的补0。每组转换成对应的十六进制数。

**【例10】将 (10110001.101)₂ 转换为十六进制数**

```
  1011   0001   1010
   ↓      ↓      ↓
   B      1      A
```

所以：**(10110001.101)₂ = (B1.A)₁₆**

**（4）十六进制数 → 二进制数**

方法：将每一位十六进制数转换成对应的4位二进制数，依次连接即可。

**【例11】将 (A2C.8)₁₆ 转换为二进制数**

```
   A      2      C      8
   ↓      ↓      ↓      ↓
  1010   0010   1100   1000
```

所以：**(A2C.8)₁₆ = (101000101100.1)₂**


## 💡 核心考点

1. **整型常量三种形式**：十进制、八进制（`0`开头）、十六进制（`0x`开头）
2. **整型变量分类**：`int`、`short`、`long`、`long long`（C99新增）
3. **有符号与无符号**：只有整型（包括字符型）可加 `signed`/`unsigned`
4. **sizeof关键字**：测量类型或变量长度，如 `sizeof(int)`
5. **实型常量两种形式**：十进制小数形式、指数形式（e/E）
6. **实型变量**：`float`（4字节）、`double`（8字节），建议用 `double`
7. **字符常量与字符串常量**：单撇号 vs 双撇号；字符串末尾有 `\0`
8. **转义字符**：`\n`（换行）、`\t`（水平制表）、`\\`（反斜杠）等
9. **大小写转换**：小写ASCII码比大写大32
10. **符号常量**：`#define PI 3.14`，末尾无分号，不占存储单元
11. **const常量**：只读变量，有类型检查，占存储单元
12. **进制转换**：位权展开求和（r→10）、除r取余法（10→r整数）、乘r取整法（10→r小数）
