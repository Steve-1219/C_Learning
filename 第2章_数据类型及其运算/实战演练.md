
**参考代码：**

```c

    int x = 012;      // 八进制数，十进制值为10

    printf("x 的十进制值为：%d\n", x);
    printf("y 的十进制值为：%d\n", y);
    printf("x + y = %d\n", sum);
    return 0;
}
```

### 四、简答题

**1. 简述 `++i` 和 `i++` 的区别，并各举一个示例。**

| 运算符 | 含义                      | 示例                            |
| :----- | :------------------------ | :------------------------------ |
| `++i`  | 先使i的值加1，再使用i的值 | `int i=5, j; j=++i;` → j=6，i=6 |
| `i++`  | 先使用i的值，再使i的值加1 | `int i=5, j; j=i++;` → j=5，i=6 |

**2. 写出C语言中算术运算符的优先级顺序（从高到低），并说明除法 `/` 和求余 `%` 的使用注意事项。**

优先级从高到低：
1. `+`（正）、`-`（负）
2. `*`、`/`、`%`
3. `+`（加）、`-`（减）

注意事项：
- **除法 `/`**：两个整数相除结果为整数（舍去小数部分）；若有一个为实数，则结果为实数
- **求余 `%`**：两个操作数必须为整数；余数符号与被除数相同

**3. 简述变量的命名规则（至少写出4条）。**

① 只能由字母、数字和下划线3种字符组成，且第1个字符必须为字母或下划线

② 区分大小写，`Name` 和 `name` 是两个不同的变量名

③ 不允许使用C语言关键字为变量命名

④ 同一函数中不能给不同的变量取相同的变量名

⑤ 变量名应简单、容易记忆，尽量做到"见名知意"
