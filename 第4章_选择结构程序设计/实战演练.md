## 六、程序设计题（共25分）

### 题目1：简单计算器（10分）

编写一个C程序，从键盘输入两个整数和一个运算符（`+`、`-`、`*`、`/`），根据运算符进行相应的算术运算并输出结果。如果输入非法运算符，输出"非法运算符"。

**输入样例：**
```
请输入表达式：8 / 4
```

**输出样例：**
```
8 / 4 = 2.00
```

**参考代码：**

```c
#include <stdio.h>
int main()
{
    float num1, num2, result;
    char op;
    printf("请输入表达式：");
    scanf("%f %c %f", &num1, &op, &num2);
    switch(op)
    {
        case '+': result = num1 + num2; printf("%.2f + %.2f = %.2f\n", num1, num2, result); break;
        case '-': result = num1 - num2; printf("%.2f - %.2f = %.2f\n", num1, num2, result); break;
        case '*': result = num1 * num2; printf("%.2f * %.2f = %.2f\n", num1, num2, result); break;
        case '/': 
            if (num2 != 0) {
                result = num1 / num2;
                printf("%.2f / %.2f = %.2f\n", num1, num2, result);
            } else {
                printf("除数不能为零！\n");
            }
            break;
        default: printf("非法运算符\n");
    }
    return 0;
}
```


### 题目2：成绩等级评定（15分）

编写一个C程序，从键盘输入一个百分制成绩（0~100），根据成绩输出对应的等级和评语。
- 90~100分：A（优秀）
- 80~89分：B（良好）
- 70~79分：C（中等）
- 60~69分：D（及格）
- 0~59分：E（不及格）

如果输入的成绩不在0~100范围内，输出"输入成绩无效！"

**输入样例1：**
```
请输入成绩：85
```

**输出样例1：**
```
等级：B（良好）
```

**输入样例2：**
```
请输入成绩：105
```

**输出样例2：**
```
输入成绩无效！
```

**参考代码：**

```c
#include <stdio.h>
int main()
{
    int score;
    printf("请输入成绩：");
    scanf("%d", &score);
    if (score < 0 || score > 100)
    {
        printf("输入成绩无效！\n");
    }
    else if (score >= 90)
        printf("等级：A（优秀）\n");
    else if (score >= 80)
        printf("等级：B（良好）\n");
    else if (score >= 70)
        printf("等级：C（中等）\n");
    else if (score >= 60)
        printf("等级：D（及格）\n");
    else
        printf("等级：E（不及格）\n");
    return 0;
}
```
