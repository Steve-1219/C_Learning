## 📖 知识梳理

### 4.3.1 选择结构程序设计的基本思路

选择结构是程序设计中三大基本结构之一，用于根据条件是否满足来决定程序的执行路径。

**设计选择结构程序的步骤：**

```
┌─────────────────────────┐
│  1. 分析问题，确定条件   │
├─────────────────────────┤
│  2. 确定条件满足和不满足 │
│     时的处理方案         │
├─────────────────────────┤
│  3. 选择合适的分支结构   │
│     (if / if-else / switch) │
├─────────────────────────┤
│  4. 编写代码并测试       │
└─────────────────────────┘
```

**选择依据：**

| 场景 | 推荐结构 |
| :--- | :--- |
| 单一条件判断 | `if` |
| 二选一分支 | `if-else` |
| 多条件连续判断 | `if-else if` |
| 多分支等值判断 | `switch-case` |


### 4.3.2 选择结构程序举例

#### 例1：判断奇偶数

**问题描述**：从键盘输入一个整数，判断它是奇数还是偶数。

**算法分析**：
- 一个数能被2整除（`num % 2 == 0`）则为偶数，否则为奇数
- 使用 `if-else` 双分支结构

**程序代码：**

```c
#include <stdio.h>
int main()
{
    int num;
    printf("请输入一个整数：");
    scanf("%d", &num);
    
    if (num % 2 == 0)
    {
        printf("%d 是偶数\n", num);
    }
    else
    {
        printf("%d 是奇数\n", num);
    }
    return 0;
}
```

**运行结果：**

```
请输入一个整数：7
7 是奇数
```


#### 例2：三个数求最大值

**问题描述**：从键盘输入三个整数，输出其中的最大值。

**算法分析**：
- 先比较前两个数，将较大值存入 `max`
- 再比较 `max` 和第三个数，更新 `max`
- 使用嵌套if语句或else-if结构

**方法一：使用else-if结构**

```c
#include <stdio.h>
int main()
{
    int a, b, c, max;
    printf("请输入三个整数：");
    scanf("%d%d%d", &a, &b, &c);
    
    if (a > b && a > c)
        max = a;
    else if (b > a && b > c)
        max = b;
    else
        max = c;
    
    printf("最大值为：%d\n", max);
    return 0;
}
```

**方法二：使用嵌套if**

```c
#include <stdio.h>
int main()
{
    int a, b, c, max;
    printf("请输入三个整数：");
    scanf("%d%d%d", &a, &b, &c);
    
    if (a > b)
    {
        if (a > c)
            max = a;
        else
            max = c;
    }
    else
    {
        if (b > c)
            max = b;
        else
            max = c;
    }
    
    printf("最大值为：%d\n", max);
    return 0;
}
```

**运行结果：**

```
请输入三个整数：15 8 12
最大值为：15
```


#### 例3：成绩等级评定

**问题描述**：从键盘输入一个百分制成绩，根据成绩输出对应的等级。
- 90~100分：A（优秀）
- 80~89分：B（良好）
- 70~79分：C（中等）
- 60~69分：D（及格）
- 0~59分：E（不及格）

**方法一：使用if-else if结构**

```c
#include <stdio.h>
int main()
{
    float score;
    printf("请输入成绩：");
    scanf("%f", &score);
    
    if (score >= 90 && score <= 100)
        printf("等级：A（优秀）\n");
    else if (score >= 80 && score < 90)
        printf("等级：B（良好）\n");
    else if (score >= 70 && score < 80)
        printf("等级：C（中等）\n");
    else if (score >= 60 && score < 70)
        printf("等级：D（及格）\n");
    else if (score >= 0 && score < 60)
        printf("等级：E（不及格）\n");
    else
        printf("输入成绩无效！\n");
    
    return 0;
}
```

**方法二：简化版（利用顺序判断）**

```c
#include <stdio.h>
int main()
{
    float score;
    printf("请输入成绩：");
    scanf("%f", &score);
    
    if (score >= 90 && score <= 100)
        printf("等级：A（优秀）\n");
    else if (score >= 80)      // 到这里时 score < 90 已经隐含
        printf("等级：B（良好）\n");
    else if (score >= 70)      // 到这里时 score < 80 已经隐含
        printf("等级：C（中等）\n");
    else if (score >= 60)      // 到这里时 score < 70 已经隐含
        printf("等级：D（及格）\n");
    else if (score >= 0)       // 到这里时 score < 60 已经隐含
        printf("等级：E（不及格）\n");
    else
        printf("输入成绩无效！\n");
    
    return 0;
}
```

**运行结果：**

```
请输入成绩：85
等级：B（良好）
```


#### 例4：计算器程序

**问题描述**：从键盘输入两个整数和一个运算符（`+`、`-`、`*`、`/`），根据运算符进行相应的算术运算。

**算法分析**：
- 这是一个多分支等值判断问题，使用 `switch` 语句最合适
- 注意除法中除数不能为0

**程序代码：**

```c
#include <stdio.h>
int main()
{
    float num1, num2, result;
    char operator;
    
    printf("请输入表达式（如 3 + 5）：");
    scanf("%f %c %f", &num1, &operator, &num2);
    
    switch (operator)
    {
        case '+':
            result = num1 + num2;
            printf("%.2f + %.2f = %.2f\n", num1, num2, result);
            break;
        case '-':
            result = num1 - num2;
            printf("%.2f - %.2f = %.2f\n", num1, num2, result);
            break;
        case '*':
            result = num1 * num2;
            printf("%.2f * %.2f = %.2f\n", num1, num2, result);
            break;
        case '/':
            if (num2 != 0)
            {
                result = num1 / num2;
                printf("%.2f / %.2f = %.2f\n", num1, num2, result);
            }
            else
            {
                printf("错误：除数不能为零！\n");
            }
            break;
        default:
            printf("错误：不支持的运算符！\n");
    }
    
    return 0;
}
```

**运行结果：**

```
请输入表达式（如 3 + 5）：3 + 5
3.00 + 5.00 = 8.00
```


#### 例5：闰年判断

**问题描述**：从键盘输入一个年份，判断该年是否为闰年。

**算法分析**：
- 闰年的条件：能被4整除但不能被100整除，或者能被400整除
- 条件表达式：`(year % 4 == 0 && year % 100 != 0) || (year % 400 == 0)`

**程序代码：**

```c
#include <stdio.h>
int main()
{
    int year;
    printf("请输入年份：");
    scanf("%d", &year);
    
    if ((year % 4 == 0 && year % 100 != 0) || (year % 400 == 0))
    {
        printf("%d 是闰年\n", year);
    }
    else
    {
        printf("%d 不是闰年\n", year);
    }
    
    return 0;
}
```

**运行结果：**

```
请输入年份：2000
2000 是闰年

请输入年份：1900
1900 不是闰年
```


#### 例6：分段函数求值

**问题描述**：计算以下分段函数的值。

```
       ┌  x² - 1    (x < 0)
f(x) = ┤  2x + 3    (0 ≤ x < 5)
       └  x³ - 10   (x ≥ 5)
```

**程序代码：**

```c
#include <stdio.h>
#include <math.h>
int main()
{
    float x, y;
    printf("请输入x的值：");
    scanf("%f", &x);
    
    if (x < 0)
    {
        y = pow(x, 2) - 1;
    }
    else if (x >= 0 && x < 5)
    {
        y = 2 * x + 3;
    }
    else    // x >= 5
    {
        y = pow(x, 3) - 10;
    }
    
    printf("f(%.2f) = %.2f\n", x, y);
    return 0;
}
```

**运行结果：**

```
请输入x的值：3
f(3.00) = 9.00
```


### 4.3.3 选择结构程序设计的常见错误

| 常见错误 | 错误示例 | 正确写法 | 说明 |
| :--- | :--- | :--- | :--- |
| **赋值当比较** | `if (a = 5)` | `if (a == 5)` | `=`是赋值，`==`是比较 |
| **缺少花括号** | `if (a > b) a = b; b = 0;` | `if (a > b) { a = b; b = 0; }` | 多条语句必须用花括号 |
| **else语法错误** | `if (a > b) { ... } else;` | `if (a > b) { ... } else { ... }` | else不能单独使用 |
| **switch缺少break** | `case 1: printf("一");` | `case 1: printf("一"); break;` | 缺少break会穿透 |
| **case后用变量** | `case num:` | `case 1:` | case后必须是常量表达式 |


## 💡 核心考点

1. **选择结构的适用场景**：根据条件选择执行不同路径
2. **if-else if结构**：适用于多条件连续判断
3. **switch结构**：适用于多分支等值判断
4. **条件表达式**：关系表达式、逻辑表达式
5. **闰年判断条件**：`(year%4==0 && year%100!=0) || (year%400==0)`
6. **常见错误**：赋值当比较、缺少花括号、缺少break
