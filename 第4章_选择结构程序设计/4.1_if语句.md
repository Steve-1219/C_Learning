# 4.1 用if语句实现选择结构

## 🎯 学习目标

- 掌握if语句的两种基本形式
- 理解if语句中的表达式与条件判断
- 掌握if语句的嵌套使用方法
- 能够运用if语句解决实际问题


## 📖 知识梳理

### 4.1.1 if语句的一般形式

在现实生活中需要进行判断和选择的情况很多。例如：
- 如果考试成绩为班级前3名，可以得到奖励
- 如果遇到绿灯，可以通行

从以上例子可以看出，条件判断的结果是一个逻辑值："是"或"否"。在计算机语言中，用"真"和"假"表示"是"和"否"。

在程序中，用选择结构来检查所指定的条件是否满足，并根据判断的结果决定执行哪种操作。

if语句有两种形式：一种是不带else子句的；另一种是带else子句的。

#### 一、if（表达式）语句（不带else）

**格式：**

```c
if (表达式)
    语句;
```

**流程图：**

```
        ┌─────────┐
        │ 表达式  │
        └────┬────┘
        真 /    \ 假
           /      \
        ┌─┘        └─┐
        ↓             ↓
    ┌───────┐    ┌───────┐
    │ 语句  │    │ 跳过  │
    └───────┘    └───────┘
        ↓             ↓
    ┌───┴─────────────┴───┐
    │      后续语句        │
    └─────────────────────┘
```

**特点：**
- 条件为"真"（非0）时，执行语句
- 条件为"假"（0）时，跳过语句
- 语句可以是一条或多条（多条需用花括号括起来）

**示例：**

```c
if (x > y)
    printf("%d", x);    // x大于y时输出x的值
```

#### 二、if（表达式）语句1 else 语句2（带else）

**格式：**

```c
if (表达式)
    语句1;
else
    语句2;
```

**流程图：**

```
        ┌─────────┐
        │ 表达式  │
        └────┬────┘
        真 /    \ 假
           /      \
        ┌─┘        └─┐
        ↓             ↓
    ┌───────┐    ┌───────┐
    │ 语句1 │    │ 语句2 │
    └───────┘    └───────┘
        ↓             ↓
    ┌───┴─────────────┴───┐
    │      后续语句        │
    └─────────────────────┘
```

**特点：**
- 条件为"真"（非0）时，执行语句1
- 条件为"假"（0）时，执行语句2
- 两个分支二选一执行

**示例：**

```c
if (x > y)
    z = x;
else
    z = y;
// 将x和y中的较大值赋给z
```

#### 三、名师点拨

**1. 关于条件表达式**

- if后面跟的表达式一般为**逻辑表达式**或**关系表达式**
- 表达式也可以是任意的数值类型：

```c
if (a > b || c > b)
    printf("%f", c);

if (10)
    printf("TRUE");    // 10非0，条件为真
```

**2. 内嵌语句必须以分号结束**

```c
if (a > 10)
    printf("a大于10");
```

**3. else子句的配对规则**

- else子句不能作为语句单独使用
- else子句总是与最近的、未配对的if语句配对

**4. 条件运算符的等价写法**

```c
// 使用if-else
if (x > y)
    z = x;
else
    z = y;

// 等价于条件运算符
z = x > y ? x : y;
```


### 4.1.2 if语句的嵌套 

在if语句中可以包含一个或多个if语句，称为**if语句的嵌套**。

#### 一、嵌套if语句的一般形式

```c
if (表达式1)
{
    if (表达式2)
    {
        语句1;
    }
    else
    {
        语句2;
    }
}
else
{
    if (表达式3)
    {
        语句3;
    }
    else
    {
        语句4;
    }
}
```

#### 二、if-else if 多分支结构

实际应用中常用的一种嵌套形式：

```c
if (表达式1)
    语句1;
else if (表达式2)
    语句2;
else if (表达式3)
    语句3;
else
    语句4;
```

#### 三、使用注意事项

**1. else的配对规则**

else子句总是与**最近的、未配对的if语句**配对。

**示例：**

```c
if (a > b)
    if (c > d)
        printf("条件成立");
    else
        printf("条件不成立");   // 这个else与最近的if(c>d)配对
```

**2. 使用花括号明确层次**

为了避免混淆，建议使用花括号明确if的层次结构：

```c
if (a > b)
{
    if (c > d)
    {
        printf("条件成立");
    }
}
else
{
    printf("a不大于b");
}
```

#### 四、例题讲解

**【例1】分段函数的实现**

有一函数：
```
       ┌ -1   (x < 0)
f(x) = ┤ 0    (x = 0)
       └ 1    (x > 0)
```

**程序代码：**

```c
#include <stdio.h>
int main()
{
    int x, y;
    scanf("%d", &x);        // 从键盘接收x的值
    
    if (x < 0)              // 判断x < 0
    {
        y = -1;
    }
    else
    {
        if (x == 0)         // 内嵌if语句，判断x == 0
            y = 0;
        else                // x > 0
            y = 1;
    }
    
    printf("x = %d, y = %d\n", x, y);
    return 0;
}
```

**运行结果：**

```
输入：3
输出：x = 3, y = 1

输入：0
输出：x = 0, y = 0

输入：-5
输出：x = -5, y = -1
```

**流程图：**

```
      ┌─────────────┐
      │   开始      │
      └──────┬──────┘
             ↓
      ┌─────────────┐
      │   输入x     │
      └──────┬──────┘
             ↓
      ┌─────────────┐
      │  x < 0 ?    │──── 否 ────→ ┌─────────────┐
      └──────┬──────┘              │  x == 0 ?   │
       是    │                     └──────┬──────┘
             ↓                         是 │    │ 否
      ┌─────────────┐                    ↓    ↓
      │  y = -1     │              ┌─────────┐ ┌─────────┐
      └──────┬──────┘              │ y = 0   │ │ y = 1   │
             ↓                     └────┬────┘ └────┬────┘
             └──────────────────────────┴───────────┘
                                       ↓
                              ┌─────────────┐
                              │  输出y      │
                              └──────┬──────┘
                                     ↓
                              ┌─────────────┐
                              │   结束      │
                              └─────────────┘
```


## 💡 核心考点

1. **if语句的两种形式**：不带else（单分支）和带else（双分支）
2. **条件表达式**：可以是关系表达式、逻辑表达式或任意数值类型（非0为真）
3. **else的配对规则**：与最近的、未配对的if语句配对
4. **嵌套if语句**：if语句中再包含if语句
5. **if-else if结构**：用于多分支选择
6. **花括号的使用**：多条语句时需要用花括号括起来
