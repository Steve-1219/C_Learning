# 4.2 用switch语句实现多分支选择结构

## 🎯 学习目标

- 理解switch语句的适用场景
- 掌握switch语句的一般形式
- 理解break语句在switch中的作用
- 能够使用switch语句解决多分支选择问题


## 📖 知识梳理

### 4.2.1 为什么需要switch语句

if语句只有两个分支可供选择，但实际生活中经常会遇到多分支的选择。例如：
- 公民的个人所得税起征点和税款计算
- 成绩评级（优、良、中、及格、不及格）
- 根据用户输入的指令执行不同操作

虽然可以使用if的嵌套结构来处理多分支，但过多的if嵌套语句会造成程序**冗长、易出错、可读性降低**。C语言提供了**switch语句**直接处理多分支选择。


### 4.2.2 switch语句的一般形式 

switch语句可以根据表达式的值，使流程跳转到不同的语句。

**一般形式：**

```c
switch (表达式)
{
    case 常量表达式1: 语句1;
    case 常量表达式2: 语句2;
    ...
    case 常量表达式n: 语句n;
    default: 语句n+1;
}
```

**各部分说明：**

| 部分 | 说明 |
| :--- | :--- |
| `switch (表达式)` | 表达式可以是**整型**或**字符型**数据，不能是实型 |
| `case 常量表达式:` | case后面跟一个常量表达式，用来标记一个位置。如果表达式的值等于该常量值，则从该case开始执行 |
| `default:` | 可选的，当表达式的值与所有case都不匹配时执行 |
| `break;` | 跳出switch语句，终止执行 |

**流程图：**

```
                    ┌─────────────────────┐
                    │  switch (表达式)    │
                    └──────────┬──────────┘
                               ↓
                    ┌─────────────────────┐

                    └──────────┬──────────┘
                               ↓
                ┌──────────────┼──────────────┐
                ↓              ↓              ↓
          ┌──────────┐  ┌──────────┐  ┌──────────┐
          │case值1   │  │case值2   │  │case值n   │
          └────┬─────┘  └────┬─────┘  └────┬─────┘
               │             │             │
          ┌────┴─────┐  ┌────┴─────┐  ┌────┴─────┐
          │ 匹配？   │  │ 匹配？   │  │ 匹配？   │
          └────┬─────┘  └────┬─────┘  └────┬─────┘
            是 │            是 │            是 │
               ↓              ↓              ↓
          ┌──────────┐  ┌──────────┐  ┌──────────┐
          │ 执行语句 │  │ 执行语句 │  │ 执行语句 │

                       └─────────────┘
```

**重要规则：**

1. **表达式类型**：switch后面括号内的"表达式"，其值可以为**数值型**或**字符型**数据
2. **case常量**：case后面跟一个**常量表达式**，用来标记一个位置
3. **default可选**：可以没有default语句，此时如果没有与switch表达式相匹配的case常量，则**不执行任何语句**
4. **break的作用**：break用于跳出switch语句；如果没有break，程序会继续执行下一个case的语句（**"穿透"现象**）
5. **case顺序**：case和default的顺序可以任意，但default通常放在最后
6. **多个case共用一个执行体**：多个case可以共用同一组语句


### 4.2.3 switch语句举例

**【例2】成绩等级转分数段**

要求按照考试成绩的等级输出百分制分数段：
- A等：85分及以上
- B等：70分~84分
- C等：60分~69分
- D等：60分以下

成绩的等级由键盘输入。

**程序代码：**

```c
#include <stdio.h>
int main()
{
    char grade;
    scanf("%c", &grade);
    printf("分数段：");
    switch(grade)
    {
        case 'A': printf("85~100\n"); break;
        case 'B': printf("70~84\n"); break;
        case 'C': printf("60~69\n"); break;
        case 'D': printf("0~59\n"); break;
        default: printf("输入数据有错\n");
    }
    return 0;
}
```

**运行结果：**

```
输入：B
输出：分数段：70~84
```

**程序分析：**

| 行号 | 代码 | 说明 |
| :---: | :--- | :--- |
| 第5行 | `scanf("%c", &grade);` | 从键盘输入一个字符，赋给grade |
| 第6行 | `printf("分数段：");` | 输出提示信息 |
| 第7行 | `switch(grade)` | 根据grade的值进行分支选择 |
| 第8行 | `case 'A': printf("85~100\n"); break;` | 如果grade为'A'，输出分数段，然后break跳出 |
| 第9行 | `case 'B': printf("70~84\n"); break;` | 如果grade为'B'，输出分数段，然后break跳出 |
| 第10行 | `case 'C': printf("60~69\n"); break;` | 如果grade为'C'，输出分数段，然后break跳出 |
| 第11行 | `case 'D': printf("0~59\n"); break;` | 如果grade为'D'，输出分数段，然后break跳出 |
| 第12行 | `default: printf("输入数据有错\n");` | 如果grade不是'A'、'B'、'C'、'D'，输出错误信息 |


### 4.2.4 break语句的重要性

**没有break的现象（"穿透"）：**

```c
switch (grade)
{
    case 'A': printf("85~100\n");
    case 'B': printf("70~84\n");
    case 'C': printf("60~69\n");
    case 'D': printf("0~59\n");
    default: printf("输入数据有错\n");
}
```

**输入：'A'**

```
输出：85~100
     70~84
     60~69
     0~59
     输入数据有错
```

> ⚠️ **注意**：如果没有break，程序会从匹配的case开始，一直执行到switch结束。这称为"**穿透**"现象。

**多个case共用一个执行体：**

```c
switch (grade)
{
    case 'A':
    case 'B':
    case 'C':
        printf("及格\n");
        break;
    case 'D':
        printf("不及格\n");
        break;
    default:
        printf("输入错误\n");
}
```

- 输入'A'、'B'、'C'都会输出"及格"
- 输入'D'输出"不及格"


### 4.2.5 switch语句与if-else if的对比

| 对比项 | switch语句 | if-else if |
| :--- | :--- | :--- |
| **适用场景** | 单一表达式与多个常量值比较 | 复杂条件判断、范围判断 |
| **表达式类型** | 整型或字符型 | 任意类型 |
| **效率** | 较高（跳转表） | 较低（逐个比较） |
| **可读性** | 多分支时清晰 | 多分支时冗长 |
| **条件判断** | 等值判断 | 可以处理范围、关系等复杂条件 |


## 💡 核心考点

1. **switch语句格式**：`switch(表达式){ case 常量表达式: 语句; break; ... default: 语句; }`
2. **表达式类型**：必须是整型或字符型，不能是实型
3. **case后跟常量表达式**：不能是变量或表达式
4. **break的作用**：跳出switch语句，防止穿透
5. **default可选**：所有case都不匹配时执行
6. **多个case共用一个执行体**：不写break实现
7. **switch与if的区别**：switch适合多分支等值判断，if适合复杂条件
