# 9.1 定义和使用结构体变量

## 🎯 学习目标

- 理解结构体的概念和作用
- 掌握结构体类型的声明方法
- 掌握结构体变量的定义方法
- 掌握结构体变量的初始化和引用


## 📖 知识梳理

### 9.1.1 建立结构体类型 

#### 一、为什么需要结构体

在前面学到的程序中，所用到的变量大多都是相互独立、无内在联系的。但在实际生活中，有些数据是有内在联系、成组出现的。

例如，学生的基本信息表中，一个学生的学号（num）、姓名（name）、性别（sex）、年龄（age）、成绩（score）、地址（addr）等信息是属于**同一个学生**的：

| num | name | sex | age | score | addr |
| :---: | :--- | :---: | :---: | :---: | :--- |
| 10001 | Mike | M | 18 | 87.5 | Shanghai |

这些数据项组成一个组合数据，可以定义一个名为 `student_1` 的变量，这个变量中包括学生的所有信息。

> 💡 **核心概念**：C语言允许用户自己建立由**不同类型数据**组成的结合型的数据结构，称为**结构体**（struct）。

#### 二、声明结构体类型

声明一个结构体类型的一般形式为：

```c
struct 结构体名
{
    成员列表;
};
```

**示例：声明一个学生结构体类型**

```c
struct Student
{
    int num;            // 学号为整型
    char name[20];      // 姓名为字符数组
    char sex;           // 性别为字符型
    int age;            // 年龄为整型
    float score;        // 成绩为浮点型
    char addr[30];      // 地址为字符数组
};
```

**各部分说明：**

| 部分 | 说明 |
| :--- | :--- |
| `struct` | 声明结构体类型时必须使用的关键字，不能省略 |
| `Student` | 结构体名（结构体标记），由用户指定 |
| `struct Student` | 结构体类型的完整名字 |
| 花括号内的内容 | 结构体的**成员列表**（也称"域表"） |

> ⚠️ **注意**：声明结构体类型时，系统**并不分配存储空间**，只有在用结构体类型定义结构体变量时才分配空间。

#### 三、声明结构体类型的注意事项

| 序号 | 注意事项 | 说明 |
| :---: | :--- | :--- |
| ① | 结构体类型可以设计多种 | 如 `struct Teacher`、`struct Worker`、`struct Date` 等 |
| ② | 成员可以属于另一个结构体类型 | 结构体可以嵌套 |
| ③ | 成员名命名规则与变量名相同 | — |

**嵌套结构体示例：**

```c
struct Date
{
    int month;
    int day;
    int year;
};

struct Teacher
{
    int num;
    char name[20];
    char sex;
    int age;
    struct Date birthday;    // 成员 birthday 属于 struct Date 类型
    char addr[30];
};
```


### 9.1.2 定义结构体类型变量 

为了能在程序中使用结构体类型的数据，应当定义结构体类型的变量，并在其中存放具体的数据。可以采取以下 **3种方法** 定义结构体类型变量。

#### 一、先声明结构体类型，再定义该类型的变量

```c
struct Student
{
    int num;
    char name[20];
    char sex;
    int age;
    float score;
    char addr[30];
};

struct Student student1, student2;    // 定义结构体变量
```

> 💡 这种形式和定义其他类型的变量形式（如 `int a, b;`）是相似的。

#### 二、在声明类型的同时定义变量

```c
struct Student
{
    int num;
    char name[20];
    char sex;
    int age;
    float score;
    char addr[30];
} student1, student2;    // 在声明类型的同时定义变量
```

#### 三、不指定类型名而直接定义结构体类型变量

```c
struct
{
    int num;
    char name[20];
    char sex;
    int age;
    float score;
    char addr[30];
} student1, student2;    // 无名结构体类型
```

> ⚠️ **注意**：这种方式指定了一个**无名的结构体类型**，它没有名字（不出现结构体名）。显然不能再以此结构体类型去定义其他变量，这种方式用得不多。

#### 四、名师点拨

| 要点 | 说明 |
| :--- | :--- |
| ① | 结构体类型与结构体变量是不同的概念。**类型不分配空间，变量分配空间** |
| ② | 结构体类型中的成员名可以与程序中的变量名相同，它们互不干扰 |
| ③ | 对结构体变量中的成员（即"域"）可以单独使用，其作用与地位相当于普通变量 |
| ④ | 结构体变量占用内存的大小等于各成员所占字节数之和 |


### 9.1.3 结构体变量的初始化和引用 

#### 一、初始化结构体变量

在定义结构体变量时，可以对它进行初始化，即赋予初始值。

**【例1】把一个学生的信息（包括学号、姓名、性别、住址）放在一个结构体变量中，然后输出这个学生的信息。**

```c
#include <stdio.h>
int main()
{
    struct Student    // 声明结构体类型 struct Student
    {
        long int num;
        char name[20];
        char sex;
        char addr[20];
    } a = {10101, "Mike", 'M', "Beijing"};    // 定义结构体变量 a 并初始化
    
    printf("学号：%ld\n姓名：%s\n性别：%c\n住址：%s\n", a.num, a.name, a.sex, a.addr);
    return 0;
}
```

**运行结果：**

```
学号：10101
姓名：Mike
性别：M
住址：Beijing
```

**程序分析：**

| 行号 | 代码 | 说明 |
| :---: | :--- | :--- |
| 第3-9行 | 声明 `struct Student` 类型 | 包含4个成员：num、name、sex、addr |
| 第10行 | `a = {10101, "Mike", 'M', "Beijing"}` | 在定义变量的同时进行初始化 |
| 第11行 | `a.num`、`a.name` 等 | 用成员运算符 `.` 引用各成员 |

#### 二、引用结构体变量

引用结构体变量应遵守以下规则：

**1. 引用方式**

```c
结构体变量名.成员名
```

**示例：**

```c
student1.num = 10101;    // 对成员赋值
printf("%d", student1.num);    // 输出成员的值
```

> 💡 **了解**：不能通过结构体变量名来得到结构体变量中所有成员的值。只能对结构体变量中的各个成员分别进行输入和输出。

**2. 成员运算符 `.`**

`.` 是**成员运算符**（也称"圆点运算符"），它在所有的运算符中优先级**最高**。因此 `student1.num` 可以作为一个整体来看待，相当于一个变量。

**3. 嵌套结构体成员的引用**

如果成员本身又属于一个结构体类型，则要用若干个成员运算符，一级一级地找到最低级的成员。

```c
student1.num                    // 普通成员
student1.birthday.month         // 嵌套成员：birthday中的month
```

> ⚠️ **注意**：不能用 `student1.birthday` 来访问整个嵌套结构体，只能访问到最底层的成员。

**4. 结构体成员的运算**

对结构体变量的成员可以像普通变量一样进行各种运算：

```c
student2.score = student1.score;    // 赋值运算
sum = student1.score + student2.score;    // 加法运算
student1.age++;    // 自加运算（. 优先级最高，对 age 自加）
```

**5. 同类型结构体变量可以互相赋值**

```c
student1 = student2;    // 假设 student1 和 student2 是同类型结构体变量
```

**6. 结构体变量的地址**

可以引用结构体变量成员的地址，也可以引用结构体变量的地址：

```c
scanf("%d", &student1.num);    // 输入 student1.num 的值
printf("%o", &student1);       // 输出结构体变量 student1 的首地址
```

> ⚠️ **错误示例**：不能整体读入结构体变量
> ```c
> scanf("%d,%s,%c,%d,%f,%s", &student1);    // 错误！
> ```


## 💡 核心考点

1. **结构体**：由不同类型数据组成的结合型数据结构
2. **声明结构体类型**：`struct 结构体名 { 成员列表 };`
3. **定义结构体变量**：三种方法（先声明后定义、声明时定义、无名结构体）
4. **结构体变量初始化**：在定义时用花括号提供各成员的值
5. **引用成员**：`结构体变量名.成员名`
6. **成员运算符 `.`**：优先级最高
7. **嵌套结构体**：成员可以是另一个结构体类型
8. **结构体变量赋值**：同类型结构体变量可以互相赋值
