# 9.7 枚举类型

## 🎯 学习目标

- 理解枚举类型的概念和作用
- 掌握枚举类型的声明方法
- 掌握枚举变量的定义方法
- 理解枚举元素的值和用法


## 📖 知识梳理

### 9.7.1 枚举类型的概念 

如果一个变量只有几种可能的值，则可以定义为**枚举类型**。所谓"枚举"就是将变量的值**一一列举**出来，变量的值只限于列举出来的值的范围内。

> 💡 **核心概念**：枚举类型是一种**用户自定义的数据类型**，用于表示一组整型常量，提高程序的可读性。

### 9.7.2 枚举类型的声明 

声明枚举类型用 `enum` 开头，一般形式为：

```c
enum [枚举名] { 枚举元素表列 };
```

**示例：**

```c
enum Weekday { sun, mon, tue, wed, thu, fri, sat };
```

以上声明了一个枚举类型 `enum Weekday`，花括号中的 `sun, mon, ..., sat` 称为**枚举元素**或**枚举常量**，它们是用户指定的名字。

**各部分说明：**

| 部分 | 说明 |
| :--- | :--- |
| `enum` | 声明枚举类型的关键字 |
| `Weekday` | 枚举类型名（可省略） |
| `{ sun, mon, ..., sat }` | 枚举元素列表，列出所有可能的取值 |

### 9.7.3 定义枚举变量 

可以用声明的枚举类型来定义变量：

```c
enum Weekday workday, week_end;
```

- `workday` 和 `week_end` 被定义为枚举变量
- 它们的值只限于花括号中指定的值之一（`sun` 到 `sat`）

**赋值示例：**

```c
workday = mon;
week_end = sun;
```

也可以不声明有名字的枚举类型，而直接定义枚举变量：

```c
enum { sun, mon, tue, wed, thu, fri, sat } workday, week_end;
```

> ⚠️ **注意**：如果赋予枚举变量一个不在枚举范围内的值，编译器会提示出错信息，便于检查。

### 9.7.4 枚举元素的值 

**重要规则：**

| 规则 | 说明 |
| :--- | :--- |
| ① 默认值 | 枚举元素的默认值从 **0** 开始，依次加1 |
| ② 可指定值 | 可以在声明时**指定**枚举元素的值 |
| ③ 未指定则递增 | 未指定值的枚举元素，其值为前一个元素的值**加1** |
| ④ 可重复 | 不同枚举元素可以具有**相同的值** |

**示例1：默认值**

```c
enum Weekday { sun, mon, tue, wed, thu, fri, sat };
```

| 枚举元素 | sun | mon | tue | wed | thu | fri | sat |
| :---: | :---: | :---: | :---: | :---: | :---: | :---: | :---: |
| 值 | 0 | 1 | 2 | 3 | 4 | 5 | 6 |

**示例2：指定部分值**

```c
enum Color { red = 1, green = 3, blue = 5 };
```

| 枚举元素 | red | green | blue |
| :---: | :---: | :---: | :---: |
| 值 | 1 | 3 | 5 |

**示例3：混合指定**

```c
enum Month { Jan = 1, Feb, Mar, Apr = 10, May, Jun };
```

| 枚举元素 | Jan | Feb | Mar | Apr | May | Jun |
| :---: | :---: | :---: | :---: | :---: | :---: | :---: |
| 值 | 1 | 2 | 3 | 10 | 11 | 12 |

### 9.7.5 枚举类型的应用 

**【例】使用枚举类型表示一周的天数**

```c
#include <stdio.h>

int main()
{
    enum Weekday { sun, mon, tue, wed, thu, fri, sat };
    enum Weekday today;
    
    today = wed;
    
    printf("today = %d\n", today);    // 输出：3
    
    // 遍历枚举
    for (int i = sun; i <= sat; i++)
    {
        printf("%d ", i);             // 输出：0 1 2 3 4 5 6
    }
    printf("\n");
    
    return 0;
}
```

**输出结果：**

```
today = 3
0 1 2 3 4 5 6
```


## 💡 核心考点

1. **枚举类型**：将变量的值一一列举出来的用户自定义类型
2. **声明格式**：`enum 枚举名 { 枚举元素表列 };`
3. **枚举变量定义**：`enum 枚举名 变量名;`
4. **枚举元素默认值**：从0开始，依次递增1
5. **可指定值**：可以在声明时为枚举元素指定整数值
6. **值范围限制**：枚举变量的值只能是列举出来的值之一
7. **提高可读性**：用有意义的名称代替数字常量
