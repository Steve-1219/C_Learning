## 📖 知识梳理

### 9.5.1 链表的定义 

#### 一、什么是链表

**链表**是一种常见的重要的数据结构。它是动态地进行存储分配的一种结构，可以根据需要开辟内存单元。

**单向链表的结构：**

```
head
  ↓
┌─────────┐    ┌─────────┐    ┌─────────┐
│ 数据域  │ →  │ 数据域  │ →  │ 数据域  │ → NULL
│ 指针域  │    │ 指针域  │    │ 指针域  │
└─────────┘    └─────────┘    └─────────┘
   结点1          结点2          结点3（表尾）
```

**链表的基本概念：**

| 术语 | 含义 |
| :--- | :--- |
| **头指针（head）** | 存放第一个结点地址的指针变量 |
| **结点** | 链表中的每一个元素 |
| **数据域** | 结点中存放用户数据的部分 |
| **指针域** | 结点中存放下一个结点地址的部分 |
| **表尾** | 最后一个结点，指针域为 `NULL` |

> 💡 **重要**：链表中的各元素在内存中的地址**可以不连续**。要找到某一元素，必须通过前一个结点的指针域。如果没有头指针（head），则整个链表都无法访问。

#### 二、链表结点的定义

链表这种数据结构，必须利用**结构体变量**和**指针变量**才能实现。一个结点中应包含一个指针变量，用它存放下一个结点的地址。

```c
struct Student
{
    int num;                 // 数据域：学号
    float score;             // 数据域：成绩
    struct Student *next;    // 指针域：指向下一个结点
};
```

> 💡 **理解**：`next` 是指针类型的成员，它指向 `struct Student` 类型的数据。一个指针类型的成员既可以指向其他类型的结构体数据，也可以指向自己所在的结构体类型的数据。


### 9.5.2 建立静态链表 

**静态链表**是指在程序编译时就确定了各个结点的链接关系，各结点的内存空间在编译时分配。

**【例7】建立一个由3个学生数据的结点组成的简单链表，要求输出各结点中的数据。**

```c
#include <stdio.h>

struct Student
{
    int num;
    float score;
    struct Student *next;
};

int main()
{
    struct Student a, b, c, *head, *p;
    
    // 给各结点数据域赋值
    a.num = 1010;
    a.score = 70.0;
    b.num = 1014;
    b.score = 95.2;
    c.num = 1017;
    c.score = 90.0;
    
    // 建立链表关系
    head = &a;        // head 指向 a 结点
    a.next = &b;      // a 指向 b
    b.next = &c;      // b 指向 c
    c.next = NULL;    // c 是表尾，指向空
    
    // 遍历输出链表
    p = head;
    do
    {
        printf("%d %5.1f\n", p->num, p->score);
        p = p->next;    // 指向下一个结点
    } while (p != NULL);
    
    return 0;
}
```

**运行结果：**

```
1010  70.0
1014  95.2
1017  90.0
```

**程序分析：**

| 步骤 | 操作 | 说明 |
| :---: | :--- | :--- |
| ① | `head = &a;` | 头指针指向第一个结点 |
| ② | `a.next = &b;` | 结点a的指针域指向结点b |
| ③ | `b.next = &c;` | 结点b的指针域指向结点c |
| ④ | `c.next = NULL;` | 结点c的指针域为空，作为表尾 |
| ⑤ | `p = head;` | 从第一个结点开始遍历 |
| ⑥ | `p = p->next;` | 依次指向下一个结点 |


### 9.5.3 建立动态链表 

**动态链表**是指在程序执行过程中从无到有地建立起一个链表，即逐个地开辟结点、输入各结点数据，并建立起前后相链的关系。

#### 一、malloc 函数

在C语言中，开辟内存单元需要用 `malloc` 函数。

```c
#include <malloc.h>

void *malloc(unsigned int size);
```

| 要点 | 说明 |
| :--- | :--- |
| **功能** | 在内存的动态存储区中分配 `size` 个字节的连续存储空间 |
| **返回值** | 分配成功时返回所分配内存空间的**首地址**，类型为 `void *` |
| **失败时** | 返回 `NULL` |
| **头文件** | `malloc.h` 或 `stdlib.h` |

> 💡 **注意**：`malloc` 函数返回的地址是 `void *` 类型，即不指定特定类型的对象。使用时需要**强制类型转换**为目标指针类型。

#### 二、动态链表的建立

**【例8】建立一个有两个学生学号和成绩数据的单向动态链表。**

```c
#include <stdio.h>
#include <malloc.h>

#define LEN sizeof(struct Student)    // LEN 代表 struct Student 类型数据的字节数

struct Student
{
    int num;
    float score;
    struct Student *next;
};

int main()
{
    struct Student *head, *p;
    
    // 开辟第1个结点
    head = p = (struct Student *)malloc(LEN);
    printf("请输入第1个学生的学号和成绩：");
    scanf("%d%f", &p->num, &p->score);
    
    // 开辟第2个结点
    p = (struct Student *)malloc(LEN);
    printf("请输入第2个学生的学号和成绩：");
    scanf("%d%f", &p->num, &p->score);
    
    // 建立链接关系
    head->next = p;    // 第1个结点的next指向第2个结点
    p->next = NULL;    // 第2个结点作为表尾
    
    // 输出链表
    printf("\n结点1：%d, %6.2f\n", head->num, head->score);
    p = head->next;
    printf("结点2：%d, %6.2f\n", p->num, p->score);
    
    return 0;
}
```

**运行结果：**

```
请输入第1个学生的学号和成绩：10101 89.5
请输入第2个学生的学号和成绩：10102 90.0

结点1：10101,  89.50
结点2：10102,  90.00
```

**程序分析：**

| 步骤 | 代码 | 说明 |
| :---: | :--- | :--- |
| ① | `#define LEN sizeof(struct Student)` | 定义 LEN 为结构体类型的字节数 |
| ② | `head = p = (struct Student *)malloc(LEN);` | 开辟新结点，head和p同时指向它 |
| ③ | `scanf("%d%f", &p->num, &p->score);` | 输入第1个结点的数据 |
| ④ | `p = (struct Student *)malloc(LEN);` | 开辟第2个结点，p指向它 |
| ⑤ | `head->next = p;` | 第1个结点的next指向第2个结点 |
| ⑥ | `p->next = NULL;` | 第2个结点的next为空，成为表尾 |
| ⑦ | 输出链表 | 从head开始依次输出 |

**示意图：**

```
head
  ↓
┌──────────────┐     ┌──────────────┐
│  num: 10101  │     │  num: 10102  │
│  score: 89.5 │  →  │  score: 90.0 │
│  next: ──────┼────→│  next: NULL  │
└──────────────┘     └──────────────┘
     结点1               结点2（表尾）
```


## 💡 核心考点

1. **链表**：动态数据结构，由结点组成，结点包含数据域和指针域
2. **头指针**：指向第一个结点，是访问链表的入口
3. **表尾**：最后一个结点，指针域为 `NULL`
4. **静态链表**：编译时确定链接关系，内存静态分配
5. **动态链表**：运行时动态开辟结点，使用 `malloc` 函数
6. **malloc 函数**：动态分配内存，返回 `void *`，需强制类型转换
7. **结点定义**：结构体包含指向自身类型的指针成员
