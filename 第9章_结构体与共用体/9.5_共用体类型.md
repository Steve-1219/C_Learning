## 📖 知识梳理

### 9.6.1 共用体类型的定义 

#### 一、为什么需要共用体

有时需要在同一段内存单元中存放不同类型的变量。例如，把一个短整型变量、一个字符型变量和一个实型变量放在同一个地址开始的内存单元中：

```
        ┌─────────────────┐
 1000   │   short int i   │  ← 短整型变量占2字节
        ├─────────────────┤
 1000   │   char ch       │  ← 字符型变量占1字节
        ├─────────────────┤
 1000   │   float f       │  ← 实型变量占4字节
        └─────────────────┘
```

> 💡 **核心概念**：这种使几个不同的变量**共享同一段内存**的结构，称为**"共用体"**（union）类型的结构。共用体使用**覆盖技术**，后一个数据会覆盖前面的数据。

#### 二、定义共用体类型

定义共用体类型变量的一般形式为：

```c
union 共用体名
{
    成员表列;
} 变量表列;
```

**示例：**

```c
union Data
{
    short int i;    // 短整型成员
    char ch;        // 字符型成员
    float f;        // 实型成员
} a, b, c;
```

**与结构体的对比：**

| 对比项 | 结构体（struct） | 共用体（union） |
| :--- | :--- | :--- |
| **内存分配** | 各成员**分别占用**自己的内存单元 | 各成员**共享同一段**内存单元 |
| **内存大小** | 各成员内存长度**之和** | 等于**最长的成员**的长度 |
| **存储内容** | 同时存放所有成员的值 | **同一时刻只能存放一个**成员的值 |

> 💡 **拿分笔记**：上面定义的共用体变量 `a`、`b`、`c` 均占 **4个字节**（因为 `float` 型变量占4个字节，是最长的成员）。


### 9.6.2 引用共用体变量的方式 

只有先定义了共用体变量才能引用它，但应注意，**不能引用共用体变量本身**，而只能引用共用体变量中的**成员**。

**正确的引用方式：**

```c
a.i      // 引用共用体变量中的整型成员 i
a.ch     // 引用共用体变量中的字符成员 ch
a.f      // 引用共用体变量中的实型成员 f
```

**错误的引用方式：**

```c
printf("%d", a);    // 错误！不能直接引用共用体变量名
```

> ⚠️ **原因**：`a` 的存储区可以按不同的类型存放数据，有不同的长度，仅写共用体变量名 `a`，系统无法知道究竟应输出哪一个成员的值。

**正确写法：**

```c
printf("%d", a.i);    // 输出整型成员的值
printf("%c", a.ch);   // 输出字符成员的值
```


### 9.6.3 共用体类型数据的特点 

在使用共用体类型数据时，需要注意以下特点：

| 序号 | 特点 | 说明 |
| :---: | :--- | :--- |
| ① | **同一时刻只能存放一个成员** | 共用体变量中**只能存放一个值**，最后一个赋值的成员有效 |
| ② | **初始化只能有一个值** | 对共用体变量初始化时，初始化表中**只能有一个值** |
| ③ | **最后赋值者生效** | 共用体变量中起作用的成员是**最后一次被赋值**的成员，会覆盖之前的值 |
| ④ | **地址相同** | 共用体变量的地址和它的各成员的地址**都是同一个地址** |
| ⑤ | **不能对变量名赋值** | 不能对共用体变量名赋值，也不能引用变量名来得到一个值 |
| ⑥ | **函数参数** | 以前C规定不能用共用体变量作为函数参数，但**C99允许**；可以使用指向共用体的指针作为参数 |
| ⑦ | **可以嵌套** | 共用体类型可以出现在结构体类型定义中，结构体也可以出现在共用体类型定义中 |

**重点示例：最后赋值者生效**

```c
union Data
{
    short int i;
    char ch;
    float f;
} a;

a.i = 100;      // 存储单元中存放 100（整型）
a.ch = 'A';     // 存储单元中被覆盖，现在存放 'A'（字符型）
a.f = 3.14;     // 存储单元中被覆盖，现在存放 3.14（实型）

printf("%d\n", a.i);   // 输出不确定的值（已被覆盖）
printf("%c\n", a.ch);  // 输出不确定的值（已被覆盖）
printf("%f\n", a.f);   // 输出 3.14（最后一个赋值的成员有效）
```

> ⚠️ **重要**：在引用共用体变量时，应十分注意当前存放在共用体变量中的究竟是**哪个成员的值**。


## 💡 核心考点

1. **共用体**：使几个不同的变量共享同一段内存的结构
2. **内存大小**：共用体变量所占内存长度等于**最长的成员**的长度
3. **与结构体的区别**：结构体各成员分别占内存，共用体各成员共享内存
4. **引用方式**：只能引用成员（`变量名.成员名`），不能引用变量名本身
5. **最后赋值者生效**：共用体变量中起作用的成员是最后一次被赋值的成员
6. **地址相同**：共用体变量和所有成员的地址相同
7. **初始化**：只能有一个初值
