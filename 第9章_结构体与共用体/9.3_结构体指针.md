# 9.3 结构体指针

## 🎯 学习目标

- 掌握指向结构体变量的指针的定义和使用
- 掌握通过指针访问结构体成员的三种方式
- 掌握指向结构体数组的指针的使用
- 理解结构体指针的应用场景


## 📖 知识梳理

### 9.3.1 指向结构体变量的指针

所谓**结构体指针**就是指向结构体变量的指针，一个结构体变量的**起始地址**就是这个结构体变量的指针。

> 💡 **核心概念**：将结构体变量的起始地址存放在一个指针变量中，那么这个指针变量就指向该结构体变量。

#### 一、定义指向结构体变量的指针

指向结构体对象的指针变量既可指向**结构体变量**，也可指向**结构体数组中的元素**。指针变量的基类型必须与结构体变量的类型相同。

```c
struct Student *pt;    // pt 可以指向 struct Student 类型的变量或数组元素
```

#### 二、通过指针访问结构体成员

访问结构体成员有三种等价方式：

| 方式 | 语法 | 说明 |
| :--- | :--- | :--- |
| ① 直接访问 | `stu_1.num` | 通过结构体变量名访问 |
| ② 指针解引用 | `(*p).num` | 通过指针解引用后访问 |
| ③ 指向运算符 | `p->num` | 使用 `->` 运算符（最常用） |

> 💡 **名师点拨**：
> - `(*p).num` 中的括号不可省略，因为成员运算符 `.` 优先于 `*` 运算符
> - `*p.num` 等价于 `*(p.num)`，是错误的写法
> - `p->num` 最直观、最常用

> 💡 **注意**：`->` 称为**指向运算符**（箭头运算符），`p->num` 表示 p 所指向的结构体变量中的 num 成员。

**【例4】通过指向结构体变量的指针变量输出结构体变量中成员的信息。**

```c
#include <stdio.h>
#include <string.h>

int main()
{
    struct Student
    {
        long num;
        char name[20];
        char sex;
        float score;
    };
    
    struct Student stu_1;          // 定义 struct Student 类型的变量 stu_1
    struct Student *p;             // 定义指向 struct Student 类型数据的指针变量 p
    p = &stu_1;                    // p 指向 stu_1
    
    stu_1.num = 10101;
    strcpy(stu_1.name, "Mike");    // 用字符串赋值函数给 stu_1.name 赋值
    stu_1.sex = 'M';
    stu_1.score = 89.5;
    
    // 方式①：通过结构体变量名访问
    printf("学号：%ld\n姓名：%s\n性别：%c\n成绩：%3.1f\n", 
           stu_1.num, stu_1.name, stu_1.sex, stu_1.score);
    
    // 方式②：通过指针解引用访问（等价写法）
    printf("\n学号：%ld\n姓名：%s\n性别：%c\n成绩：%3.1f\n", 
           (*p).num, (*p).name, (*p).sex, (*p).score);
    
    // 方式③：通过指向运算符访问（等价写法）
    printf("\n学号：%ld\n姓名：%s\n性别：%c\n成绩：%3.1f\n", 
           p->num, p->name, p->sex, p->score);
    
    return 0;
}
```

**运行结果：**

```
学号：10101
姓名：Mike
性别：M
成绩：89.5

学号：10101
姓名：Mike
性别：M
成绩：89.5

学号：10101
姓名：Mike
性别：M
成绩：89.5
```

**三种方式对比：**

| 方式 | 写法 | 说明 |
| :---: | :--- | :--- |
| ① | `stu_1.num` | 通过变量名直接访问 |
| ② | `(*p).num` | 通过指针解引用访问（括号不可省） |
| ③ | `p->num` | 通过指向运算符访问（最常用） |

> 💡 **记忆**：如果 p 指向结构体变量 stu，以下三种用法**完全等价**：
> 1. `stu.成员名`，如 `stu.num`
> 2. `(*p).成员名`，如 `(*p).num`
> 3. `p->成员名`，如 `p->num`


### 9.3.2 指向结构体数组的指针

可以用指针变量指向结构体数组的元素。

**【例5】有3个学生的信息，存放在结构体数组中，要求输出全部学生的信息。**

```c
#include <stdio.h>

struct Student
{
    int num;
    char name[20];
    float score;
};

int main()
{
    struct Student stu[3] = {
        {10101, "Mike", 89.5},
        {10102, "Mary", 92.0},
        {10103, "Sunny", 78.5}
    };
    struct Student *p;
    
    printf("学生信息：\n");
    for (p = stu; p < stu + 3; p++)
    {
        printf("%d  %s  %.2f\n", p->num, p->name, p->score);
    }
    return 0;
}
```

**运行结果：**

```
学生信息：
10101  Mike  89.50
10102  Mary  92.00
10103  Sunny 78.50
```

**程序分析：**

| 行号 | 代码 | 说明 |
| :---: | :--- | :--- |
| 第11行 | `struct Student *p;` | 定义指向 struct Student 类型的指针变量 |
| 第14行 | `for (p = stu; p < stu + 3; p++)` | p 依次指向数组的每个元素 |
| 第15行 | `p->num, p->name, p->score` | 通过指向运算符访问成员 |

> ⚠️ **注意**：p 是指向结构体类型对象的指针变量，不应指向数组元素中的某一成员。
>
> 错误示例：
> ```c
> p = stu[1].name;    // 错误：类型不匹配
> ```
>
> 如果要将成员的地址赋给 p，需要强制类型转换：
> ```c
> p = (struct Student *)stu[0].name;    // 强制转换类型
> ```


### 9.3.3 结构体指针的应用场景

| 场景 | 说明 |
| :--- | :--- |
| ① 函数传参 | 将结构体变量的地址传递给函数，避免复制整个结构体 |
| ② 动态链表 | 使用指针链接多个结构体变量 |
| ③ 遍历数组 | 用指针遍历结构体数组 |
| ④ 动态内存分配 | 配合 `malloc()` 动态创建结构体变量 |


## 💡 核心考点

1. **结构体指针**：指向结构体变量的指针，存放结构体变量的起始地址
2. **定义方式**：`struct 结构体名 *指针变量名;`
3. **访问成员的三种方式**：
   - `结构体变量.成员名`
   - `(*指针变量).成员名`
   - `指针变量->成员名`
4. **指向运算符 `->`**：最常用的访问方式
5. **括号不可省**：`(*p).num` 的括号不能省略
6. **遍历结构体数组**：用指针依次指向数组元素
