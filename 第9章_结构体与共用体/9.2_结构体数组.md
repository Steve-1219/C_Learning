# 9.2 结构体数组

## 🎯 学习目标

- 理解结构体数组的概念
- 掌握结构体数组的定义方法
- 掌握结构体数组的初始化方法
- 能够运用结构体数组解决实际问题


## 📖 知识梳理

### 9.2.1 定义结构体数组 

一个结构体变量中可以存放一组有关联的数据（如一个学生的学号、姓名、成绩等数据）。如果有一组学生，需要存储多个学生的数据，显然应该用**数组**，这就是**结构体数组**。

> 💡 **核心区别**：
> - **普通数组**：只能存储同类型的数据
> - **结构体数组**：可以存储不同类型的数据（每个元素是一个结构体变量）

#### 一、定义结构体数组的一般形式

**方法一：先声明结构体类型，再定义数组**

```c
struct Student
{
    int num;
    char name[20];
    float score;
};

struct Student stu[30];    // 定义结构体数组，有30个元素
```

**方法二：在声明类型的同时定义数组**

```c
struct Student
{
    int num;
    char name[20];
    float score;
} stu[30];    // 在声明类型的同时定义数组
```

**方法三：不指定类型名直接定义数组（无名结构体）**

```c
struct
{
    int num;
    char name[20];
    float score;
} stu[30];
```

#### 二、结构体数组的初始化

对结构体数组初始化的形式是在定义数组的后面加上：

```c
= { 初值表列 };
```

**示例：**

```c
struct Person leader[3] = { "Mike", 0, "Mary", 0, "Sunny", 0 };
```

> 💡 **推荐写法**：为清晰起见，可将每个元素的信息用一对花括号括起来。

```c
struct Person leader[3] = {
    {"Mike", 0},
    {"Mary", 0},
    {"Sunny", 0}
};
```

**初始化规则：**
- 花括号中的数据按顺序依次赋给数组元素的各成员
- 可以用嵌套的花括号区分不同元素
- 未初始化的成员自动为 0（数值型）或空字符（字符型）


### 9.2.2 结构体数组应用举例 

**【例9】有3个候选人，每个选民只能投票选一人，要求编写一个统计选票的程序，先后输入被选人的名字，最后输出各人得票结果。**

**问题分析：**
- 需要定义一个结构体数组，数组中包含3个元素
- 每个元素中的信息应包括：候选人的**姓名**（字符型）和**得票数**（整型）
- 输入被选人的姓名，然后与数组元素中的"姓名"成员比较
- 如果相同，则对应候选人的得票数加1

**程序代码：**

```c
#include <stdio.h>
#include <string.h>

struct Person
{
    char name[20];
    int count;
} leader[3] = {
    {"Mike", 0},
    {"Mary", 0},
    {"Sunny", 0}
};

int main()
{
    int i, j;
    char leader_name[20];
    
    printf("请输入10张选票上的名字：\n");
    for (i = 1; i <= 10; i++)
    {
        scanf("%s", leader_name);    // 输入所选的候选人姓名
        
        for (j = 0; j < 3; j++)
        {
            if (strcmp(leader_name, leader[j].name) == 0)
                leader[j].count++;
        }
    }
    
    printf("结果：\n");
    for (i = 0; i < 3; i++)
    {
        printf("%5s: %d\n", leader[i].name, leader[i].count);
    }
    return 0;
}
```

**运行结果：**

```
请输入10张选票上的名字：
Mike Mike Mary Sunny Sunny Mike Mary Mary Sunny Sunny
结果：
 Mike: 3
 Mary: 3
Sunny: 4
```

**程序分析：**

| 行号 | 代码 | 说明 |
| :---: | :--- | :--- |
| 第4-8行 | 声明 `struct Person` 类型并定义数组 `leader` | 包含 name（姓名）和 count（票数）两个成员 |
| 第9-12行 | 初始化结构体数组 | 将3位候选人的姓名和票数（均为0）赋初值 |
| 第19行 | `scanf("%s", leader_name)` | 输入被选人的姓名 |
| 第22行 | `strcmp(leader_name, leader[j].name) == 0` | 比较输入的姓名与候选人姓名是否相同 |
| 第23行 | `leader[j].count++` | 相同则票数加1（`.` 优先级高于 `++`） |
| 第27-30行 | 输出结果 | 输出3位候选人的姓名和得票数 |


### 9.2.3 结构体数组的使用要点

| 要点 | 说明 |
| :--- | :--- |
| ① 数组元素 | 每个元素都是一个结构体变量 |
| ② 成员访问 | 使用 `数组名[下标].成员名` 格式 |
| ③ 字符串比较 | 使用 `strcmp()` 函数比较字符串成员 |
| ④ 数组遍历 | 通常使用循环遍历结构体数组 |
| ⑤ 初始化 | 可以为每个元素分别用花括号括起来 |


## 💡 核心考点

1. **结构体数组**：每个元素都是结构体变量的数组
2. **定义方式**：三种方式（先声明后定义、声明时定义、无名结构体）
3. **初始化**：`struct 类型 数组名[长度] = { {成员1, 成员2}, ... };`
4. **成员访问**：`数组名[下标].成员名`
5. **字符串比较**：使用 `strcmp()` 函数
6. **典型应用**：统计投票、学生信息管理
