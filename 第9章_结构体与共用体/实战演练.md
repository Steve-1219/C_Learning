## 六、程序设计题（共25分）

### 题目1：学生信息管理（10分）

编写一个C程序，定义一个结构体类型 `Student`，包含学号（`num`）、姓名（`name`）、三科成绩（`score[3]`）和平均分（`avg`）。输入3个学生的信息，计算每个学生的平均分，并输出平均分最高的学生信息。

**输入样例：**
```
10101 Mike 78 89 98
10103 Mary 98.5 87 69
10106 Sunny 88 76 89
```

**输出样例：**
```
成绩最高的学生是：
学号：10101
姓名：Mike
成绩：78.0 89.0 98.0
平均分：88.33
```

**参考代码：**

```c
#include <stdio.h>
#define N 3

struct Student
{
    int num;
    char name[20];
    float score[3];
    float avg;
};

int main()
{
    struct Student stu[N];
    int i, j, max_idx = 0;
    
    printf("请输入%d个学生的信息（学号 姓名 三科成绩）：\n", N);
    for (i = 0; i < N; i++)
    {
        scanf("%d %s %f %f %f", &stu[i].num, stu[i].name,
              &stu[i].score[0], &stu[i].score[1], &stu[i].score[2]);
        stu[i].avg = (stu[i].score[0] + stu[i].score[1] + stu[i].score[2]) / 3.0;
    }
    
    for (i = 1; i < N; i++)
    {
        if (stu[i].avg > stu[max_idx].avg)
            max_idx = i;
    }
    
    printf("\n成绩最高的学生是：\n");
    printf("学号：%d\n", stu[max_idx].num);
    printf("姓名：%s\n", stu[max_idx].name);
    printf("成绩：%.1f %.1f %.1f\n", stu[max_idx].score[0],
           stu[max_idx].score[1], stu[max_idx].score[2]);
    printf("平均分：%.2f\n", stu[max_idx].avg);
    
    return 0;
}
```


### 题目2：投票统计（15分）

编写一个C程序，有3个候选人，每个选民只能投票选一人，要求统计选票并输出各人得票结果。

**要求：**
- 使用结构体数组存储候选人信息（姓名和票数）
- 输入10张选票上的名字
- 统计并输出各候选人得票数

**输入样例：**
```
Mike Mike Mary Sunny Sunny Mike Mary Mary Sunny Sunny
```

**输出样例：**
```
投票结果：
Mike: 3
Mary: 3
Sunny: 4
```

**参考代码：**

```c
#include <stdio.h>
#include <string.h>
#define N 3

struct Person
{
    char name[20];
    int count;
};

int main()
{
    struct Person leader[N] = {
        {"Mike", 0},
        {"Mary", 0},
        {"Sunny", 0}
    };
    char leader_name[20];
    int i, j;
    
    printf("请输入10张选票上的名字：\n");
    for (i = 0; i < 10; i++)
    {
        scanf("%s", leader_name);
        for (j = 0; j < N; j++)
        {
            if (strcmp(leader_name, leader[j].name) == 0)
                leader[j].count++;
        }
    }
    
    printf("\n投票结果：\n");
    for (i = 0; i < N; i++)
    {
        printf("%s: %d\n", leader[i].name, leader[i].count);
    }
    
    return 0;
}
```

### 五、程序改错题

| :---: | :--- | :--- | :--- |
