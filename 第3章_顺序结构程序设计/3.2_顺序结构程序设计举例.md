## 📖 知识梳理

### 3.2.1 顺序结构程序设计的基本思路

顺序结构是程序设计的三种基本结构中最简单的一种，它的执行方式是**从上到下逐行执行**，每一行代码执行完后自动进入下一行，没有分支和循环。

**顺序结构程序的基本框架：**

```
┌─────────────────────────┐
│  1. 定义变量             │  ← 声明部分
├─────────────────────────┤
│  2. 输入数据             │  ← 使用 scanf 或 getchar
├─────────────────────────┤
│  3. 处理/计算数据         │  ← 使用赋值语句和表达式
├─────────────────────────┤
│  4. 输出结果             │  ← 使用 printf 或 putchar
└─────────────────────────┘
```

> 💡 **设计要点**：顺序结构程序的核心是按照**解决问题的步骤**，依次用C语句描述每个操作，确保数据流的方向正确。


### 3.2.2 顺序结构程序举例

#### 例1：计算长方形的周长和面积

**问题描述**：从键盘输入长方形的长和宽，计算并输出其周长和面积。

**算法分析**：
- 周长 = 2 × (长 + 宽)
- 面积 = 长 × 宽

**程序代码：**

```c
#include <stdio.h>
int main()
{
    float length, width, perimeter, area;
    
    // 输入数据
    printf("请输入长方形的长和宽：");
    scanf("%f%f", &length, &width);
    
    // 计算
    perimeter = 2 * (length + width);
    area = length * width;
    
    // 输出结果
    printf("周长 = %.2f\n", perimeter);
    printf("面积 = %.2f\n", area);
    
    return 0;
}
```

**运行结果：**

```
请输入长方形的长和宽：5.5 3.2
周长 = 17.40
面积 = 17.60
```

**程序分析：**

| 步骤 | 代码                                    | 说明                  |
| :--: | :-------------------------------------- | :-------------------- |
|  ①   | `float length, width, perimeter, area;` | 定义4个实型变量       |
|  ②   | `scanf("%f%f", &length, &width);`       | 从键盘输入长和宽      |
|  ③   | `perimeter = 2 * (length + width);`     | 计算周长              |
|  ④   | `area = length * width;`                | 计算面积              |
|  ⑤   | `printf("周长 = %.2f\n", perimeter);`   | 输出周长，保留2位小数 |


#### 例2：输入一个三位数，逆序输出

**问题描述**：从键盘输入一个三位整数，将其逆序输出。例如输入 123，输出 321。

**算法分析**：
- 个位：`n % 10`
- 十位：`(n / 10) % 10`
- 百位：`n / 100`
- 逆序数 = 个位 × 100 + 十位 × 10 + 百位

**程序代码：**

```c
#include <stdio.h>
int main()
{
    int n, ge, shi, bai, result;
    
    printf("请输入一个三位整数：");
    scanf("%d", &n);
    
    ge = n % 10;              // 个位
    shi = (n / 10) % 10;      // 十位
    bai = n / 100;            // 百位
    result = ge * 100 + shi * 10 + bai;
    
    printf("逆序数为：%d\n", result);
    return 0;
}
```

**运行结果：**

```
请输入一个三位整数：123
逆序数为：321
```


#### 例3：交换两个变量的值

**问题描述**：输入两个整数 a 和 b，交换它们的值后输出。

**算法分析**：通过一个**临时变量** temp 实现交换。

```
交换前：a = 5, b = 10
temp = a;    → temp = 5
a = b;       → a = 10
b = temp;    → b = 5
交换后：a = 10, b = 5
```

**程序代码：**

```c
#include <stdio.h>
int main()
{
    int a, b, temp;
    
    printf("请输入两个整数：");
    scanf("%d%d", &a, &b);
    
    printf("交换前：a = %d, b = %d\n", a, b);
    
    temp = a;
    a = b;
    b = temp;
    
    printf("交换后：a = %d, b = %d\n", a, b);
    return 0;
}
```

**运行结果：**

```
请输入两个整数：5 10
交换前：a = 5, b = 10
交换后：a = 10, b = 5
```


### 3.2.3 常用数学函数 

在C语言中，除了基本的算术运算外，还提供了丰富的**数学函数**，用于完成更复杂的数学计算。使用这些函数需要包含数学头文件 `math.h`。

**使用数学函数的步骤：**

```c
#include <stdio.h>
#include <math.h>    // 包含数学头文件

int main()
{
    // 使用数学函数
    double result = sqrt(16.0);    // 求平方根
    return 0;
}
```

> ⚠️ **编译注意事项**：在某些编译器中，使用 `math.h` 中的函数需要在编译时链接数学库，添加 `-lm` 参数：
> ```bash
> gcc program.c -o program -lm
> ```

### 3.2.4 结合数学函数的顺序结构程序举例

#### 例4：计算一元二次方程的根

**问题描述**：求一元二次方程 `ax² + bx + c = 0` 的根，其中 a、b、c 由键盘输入。假设 `b² - 4ac ≥ 0`。

**算法分析**：
- 判别式：`delta = b² - 4ac`
- 两个根：
  - `x1 = (-b + √delta) / (2a)`
  - `x2 = (-b - √delta) / (2a)`

**程序代码：**

```c
#include <stdio.h>
#include <math.h>
int main()
{
    double a, b, c, delta, x1, x2;
    
    printf("请输入方程的三个系数 a, b, c：");
    scanf("%lf%lf%lf", &a, &b, &c);
    
    delta = b * b - 4 * a * c;
    x1 = (-b + sqrt(delta)) / (2 * a);
    x2 = (-b - sqrt(delta)) / (2 * a);
    
    printf("x1 = %.2f\n", x1);
    printf("x2 = %.2f\n", x2);
    
    return 0;
}
```

**运行结果：**

```
请输入方程的三个系数 a, b, c：1 -3 2
x1 = 2.00
x2 = 1.00
```


#### 例5：计算圆的周长、面积和球体体积

**问题描述**：输入圆的半径 r，计算圆的周长、面积以及球体体积。π 取 3.14159。

**算法分析**：
- 圆的周长：`C = 2πr`
- 圆的面积：`S = πr²`
- 球体体积：`V = (4/3)πr³`

**程序代码：**

```c
#include <stdio.h>
#include <math.h>
#define PI 3.14159

int main()
{
    double r, circumference, area, volume;
    
    printf("请输入圆的半径：");
    scanf("%lf", &r);
    
    circumference = 2 * PI * r;
    area = PI * pow(r, 2);
    volume = (4.0 / 3.0) * PI * pow(r, 3);
    
    printf("圆的周长 = %.2f\n", circumference);
    printf("圆的面积 = %.2f\n", area);
    printf("球体体积 = %.2f\n", volume);
    
    return 0;
}
```

**运行结果：**

```
请输入圆的半径：2
圆的周长 = 12.57
圆的面积 = 12.57
球体体积 = 33.51
```


## 💡 核心考点

1. **顺序结构**：程序从上到下逐行执行，无分支和循环
2. **顺序结构程序框架**：定义变量 → 输入数据 → 处理数据 → 输出结果
3. **数学库函数**：使用前需包含 `#include <math.h>`
4. **编译选项**：使用数学库时可能需加 `-lm` 参数
5. **常用函数**：`sqrt`、`pow`、`fabs`、`ceil`、`floor`、三角函数等
6. **三角函数的参数**：以**弧度**为单位
7. **求根公式**：一元二次方程 `ax² + bx + c = 0` 的求根公式
8. **变量交换**：使用临时变量实现两数交换

## 🔧 上机实践

### 实践题1：计算三角形面积

编写一个C程序，从键盘输入三角形的三条边长，使用海伦公式计算三角形面积。

**要求：**
- 边长由键盘输入
- 使用海伦公式：面积 = √(s × (s-a) × (s-b) × (s-c))，其中 s = (a+b+c)/2
- 使用 `sqrt` 函数
- 输出结果保留2位小数

**参考代码：**

```c
#include <stdio.h>
#include <math.h>

int main()
{
    double a, b, c, s, area;
    
    printf("请输入三角形的三条边长：");
    scanf("%lf%lf%lf", &a, &b, &c);
    
    s = (a + b + c) / 2;
    area = sqrt(s * (s - a) * (s - b) * (s - c));
    
    printf("三角形的面积 = %.2f\n", area);
    return 0;
}
```


### 实践题2：计算两点之间的距离

编写一个C程序，输入平面上两点的坐标 `(x1, y1)` 和 `(x2, y2)`，计算并输出两点之间的距离。

**要求：**
- 使用距离公式：`d = √[(x2-x1)² + (y2-y1)²]`
- 使用 `pow` 和 `sqrt` 函数
- 输出结果保留2位小数

**参考代码：**

```c
#include <stdio.h>
#include <math.h>

int main()
{
    double x1, y1, x2, y2, distance;
    
    printf("请输入第一个点的坐标 (x1 y1)：");
    scanf("%lf%lf", &x1, &y1);
    printf("请输入第二个点的坐标 (x2 y2)：");
    scanf("%lf%lf", &x2, &y2);
    
    distance = sqrt(pow(x2 - x1, 2) + pow(y2 - y1, 2));
    
    printf("两点之间的距离 = %.2f\n", distance);
    return 0;
}
```
