## 六、程序设计题（每题10分，共20分）

### 题目1：计算圆的周长和面积

从键盘输入圆的半径 `r`，计算并输出圆的周长和面积。π 取 3.14159。

**要求：**
- 半径由键盘输入
- 圆的周长公式：`C = 2πr`
- 圆的面积公式：`S = πr²`
- 使用 `pow` 函数计算平方
- 输出结果保留2位小数

**输入样例：**
```
请输入圆的半径：3
```

**输出样例：**
```
圆的周长 = 18.85
圆的面积 = 28.27
```

**参考代码：**

```c
#include <stdio.h>
#include <math.h>
#define PI 3.14159

int main()
{
    double r, circumference, area;
    printf("请输入圆的半径：");
    scanf("%lf", &r);
    
    circumference = 2 * PI * r;
    area = PI * pow(r, 2);
    
    printf("圆的周长 = %.2f\n", circumference);
    printf("圆的面积 = %.2f\n", area);
    return 0;
}
```


### 题目2：华氏温度转摄氏温度

编写一个C程序，将输入的华氏温度转换为摄氏温度。

**要求：**
- 从键盘输入华氏温度
- 使用公式：摄氏温度 = (华氏温度 - 32) × 5 / 9
- 输出结果保留2位小数

**输入样例：**
```
请输入华氏温度：100
```

**输出样例：**
```
摄氏温度为：37.78
```

**参考代码：**

```c
#include <stdio.h>
int main()
{
    double f, c;
    printf("请输入华氏温度：");
    scanf("%lf", &f);
    
    c = (f - 32) * 5 / 9;
    
    printf("摄氏温度为：%.2f\n", c);
    return 0;
}
```


## ✅ 参考答案

### 一、选择题

| 题号 | 答案  | 解析                                           |
| :--: | :---: | :--------------------------------------------- |
|  1   | **B** | C语言本身不提供输入输出语句，由库函数实现      |
|  2   | **C** | `getchar` 函数无参数，只能接收一个字符         |
|  3   | **C** | `sqrt(16.0)` 求平方根，返回 4.0（double 类型） |
|  4   | **B** | `math.h` 是数学函数库的头文件                  |
|  5   | **B** | `scanf` 的地址表列应为变量地址，即 `&变量名`   |

### 二、填空题

| 题号 | 答案               |
| :--: | :----------------- |
|  1   | `stdio.h`          |
|  2   | 一                 |
|  3   | `B`                |
|  4   | `%`                |
|  5   | `&`                |
|  6   | 4.0、3.0           |
|  7   | 1.0                |
|  8   | 输入数据、输出结果 |

**解析：**
- 第3题：`'A' + 1 = 66`，ASCII码66对应的字符是 `'B'`
- 第4题：格式声明以 `%` 开头，如 `%d`、`%f` 等
- 第5题：`scanf` 中变量名前加 `&` 表示取变量地址
- 第6题：`ceil` 向上取整（3.2→4.0），`floor` 向下取整（3.8→3.0）
- 第7题：sin(π/2) = 1.0
- 第8题：顺序结构程序四步：定义变量 → 输入数据 → 处理数据 → 输出结果
