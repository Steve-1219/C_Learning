## 📖 知识梳理

### 3.1.1 字符数据的输入与输出 

#### 一、输入输出的基本概念

输入输出是程序中最基本的操作之一，几乎每一个C程序都包含输入输出，没有输出的程序是没有意义的。关于程序的输入输出要注意以下几点：

| 要点           | 说明                                                         |
| :------------- | :----------------------------------------------------------- |
| **① 主体**     | 所谓输入输出是以**计算机主机为主体**而言的。从计算机向输出设备输出数据称为**输出**，从输入设备向计算机输入数据称为**输入** |
| **② 语言支持** | C语言本身**不提供**输入输出语句，输入和输出操作是由C函数库中的函数实现的。例如，`scanf` 函数和 `printf` 函数 |
| **③ 头文件**   | 在使用系统库函数时，应当在程序中使用预编译指令 `#include`。使用标准输入输出函数时要在文件开头加上：`#include <stdio.h>` |

#### 二、getchar 函数（字符输入函数）

`getchar` 函数的一般形式为：

```c
getchar()
```

| 特性         | 说明                                       |
| :----------- | :----------------------------------------- |
| **参数**     | 无参数                                     |
| **功能**     | 从计算机终端（一般是键盘）输入**一个字符** |
| **返回值**   | 将输入的字符作为函数值返回                 |
| **使用方式** | 可以赋给字符变量或整型变量                 |

**使用示例：**

```c
char c;
c = getchar();    // 从键盘输入一个字符，赋给字符变量c
```

> 💡 **名师点拨**：`getchar` 函数只能接收**一个字符**，如果输入多个字符就要使用多个 `getchar` 函数。

**重要说明：**

- `getchar` 函数不仅可以从输入设备获得**可显示的字符**，还可以获得在屏幕上**无法显示的字符**，如控制字符
- 用 `getchar` 函数得到的字符可以赋给一个字符变量或者整型变量
- 也可以作为表达式的一部分，如 `putchar(getchar())`

> 💡 **输入缓冲机制**：在使用键盘输入信息时，并不是敲一个字符该字符就立即送到计算机中。只有按了 **Enter 键**才把字符输入到计算机中，并按先后顺序分别赋给相应的变量。

**【例1】从键盘输入 KuKe 这4个字符，然后把它们输出到屏幕。**

```c
#include <stdio.h>
int main()
{
    char a, b, c, d;        // 定义字符变量 a, b, c, d
    a = getchar();          // 从键盘输入一个字符，赋给字符变量 a
    b = getchar();          // 从键盘输入一个字符，赋给字符变量 b
    c = getchar();          // 从键盘输入一个字符，赋给字符变量 c
    d = getchar();          // 从键盘输入一个字符，赋给字符变量 d
    putchar(a);             // 将变量 a 的值输出
    putchar(b);             // 将变量 b 的值输出
    putchar(c);             // 将变量 c 的值输出
    putchar(d);             // 将变量 d 的值输出
    putchar('\n');          // 换行
    return 0;
}
```

**运行结果：**

```
KuKe
KuKe
```

**程序分析：**

连续输入 `KuKe` 4个字符后，按 Enter 键，字符才送到内存中的存储单元，分别赋给变量 a、b、c、d。然后通过 `putchar` 函数依次输出。

#### 三、putchar 函数（字符输出函数）

`putchar` 函数的一般形式为：

```c
putchar(c)
```

| 特性       | 说明                                |
| :--------- | :---------------------------------- |
| **参数**   | 可以是字符常量或整型常量（ASCII码） |
| **功能**   | 输出**一个字符**                    |
| **返回值** | 输出的字符                          |

**使用示例：**

```c
putchar('A');      // 输出字符 'A'
putchar(66);       // 输出 ASCII 码 66 对应的字符 'B'
putchar('\101');   // 输出八进制数 101 对应的字符 'A'
putchar('\x41');   // 输出十六进制数 41 对应的字符 'A'
```

> 💡 **重要提示**：`putchar` 函数只能输出**字符**而不能输出整数。字符 `'B'` 的ASCII码是66，所以 `putchar(66)` 输出的是字符 `'B'`。


### 3.1.2 简单的格式输入与输出 

在C语言中，`printf` 函数和 `scanf` 函数是**格式化**的输出和输入函数，在进行输入输出时必须根据数据的类型指定输入输出数据的格式。

#### 一、printf 函数（格式输出函数）

`printf` 函数的作用是向终端（或系统隐含指定的输出设备）输出若干个任意类型的数据。

**1. printf 函数的一般格式**

```c
printf(格式控制, 输出表列)
```

- **格式控制**：用双撇号引起来的一个字符串，称"转换控制字符串"，简称"格式字符串"。它包括两个信息：

| 信息类型     | 说明                                                         | 示例                  |
| :----------- | :----------------------------------------------------------- | :-------------------- |
| **格式声明** | 由 `%` 和格式字符组成，如 `%d`、`%c`，将输出数据转换为指定的格式后输出 | `%d`、`%f`、`%c`      |
| **普通字符** | 在输出时**原样输出**的字符                                   | `"a="`、`","`、`"\n"` |

- **输出表列**：程序需要输出的一些数据，可以是常量、变量或表达式。

**2. 常用格式字符**

|        格式字符        | 说明                                                         |
| :--------------------: | :----------------------------------------------------------- |
|        `d`、`i`        | 以**十进制带符号**形式输出整数（正数不输出符号）             |
|          `o`           | 以**八进制无符号**形式输出整数（不输出前导符0）              |
|        `x`、`X`        | 以**十六进制无符号**形式输出整数（不输出前导符0x）。用 `x` 小写，用 `X` 大写 |
|          `u`           | 以**十进制无符号**形式输出整数                               |
|          `c`           | 以**字符**形式输出，只输出一个字符                           |
|          `f`           | 以**小数**形式输出单、双精度数，隐含输出 **6位小数**         |
|        `e`、`E`        | 以**指数**形式输出实数。用 `e` 时指数以"e"表示，用 `E` 时指数以"E"表示 |
|          `s`           | 输出**字符串**                                               |
| `ld`、`lo`、`lx`、`lu` | 用于**长整型**数据                                           |
|          `lf`          | 用于 **double** 型数据                                       |
|         `m.nf`         | 指定输出数据占 **m 列**，其中 **n 位小数**                   |

**3. 示例**

```c
int a = 1, b = 0;
printf("%d, ", b = a + b);
printf("%d", a = 2 * b);
```

**输出结果：**

```
1, 2
```

**解析：**

| 步骤 | 操作                                          |  结果   |
| :--: | :-------------------------------------------- | :-----: |
|  ①   | 第一条输出语句中 `a + b = 1 + 0 = 1` 赋给 `b` | `b = 1` |
|  ②   | 输出 `b` 的值                                 |   `1`   |
|  ③   | 第二条输出语句中 `2 * b = 2 * 1 = 2` 赋给 `a` | `a = 2` |
|  ④   | 输出 `a` 的值                                 |   `2`   |

#### 二、scanf 函数（格式输入函数）

**1. scanf 函数的一般格式**

```c
scanf(格式控制, 地址表列)
```

- **格式控制**：含义同 `printf` 函数
- **地址表列**：由若干个**地址**组成的表列，可以是变量的地址（`&变量名`），也可以是数组的地址

**2. 使用注意事项**

| 序号 | 注意事项                                                     | 正确示例                                             | 错误示例          |
| :--: | :----------------------------------------------------------- | :--------------------------------------------------- | :---------------- |
|  ①   | "格式控制"后面应为**变量地址**，而不是变量名                 | `scanf("%d", &a);`                                   | `scanf("%d", a);` |
|  ②   | 格式控制字符串中除格式说明外有**普通字符**，输入时在对应位置应输入**相同字符** | `scanf("a=%d", &a);` 输入：`a=5`                     | —                 |
|  ③   | 用 `%c` 格式输入时，**空格字符和转义字符**都作为有效字符输入 | `scanf("%c%c", &c1, &c2);` 输入：`a b`（空格被读入） | —                 |
|  ④   | 输入数值数据时，遇到**空格、回车、Tab键或非法字符**，认为该数据输入结束 | 输入 `123 456` 可读取两个整数                        | —                 |

**3. 输入格式示例**

```c
int a, b, c;
scanf("%d%d%d", &a, &b, &c);    // 输入：3 4 5 或 3[Enter]4[Enter]5[Enter]
scanf("%d,%d,%d", &a, &b, &c);  // 输入：3,4,5
scanf("a=%d,b=%d", &a, &b);     // 输入：a=3,b=4
```

> 💡 **提示**：在输入数值数据时，空格、回车、Tab键都作为数据结束的标志，而不是有效字符。

**4. 即学即练**

（单选）设有 `int a, b; float c, d;`，下列正确的是（ ）

A. `scanf("%d%d", a, b);`
B. `scanf("%f%f", c, d);`
C. `scanf("%f%f", &c, &d);`
D. `scanf("%d%f", &c, &d);`

**答案：** **C**

**精析：** `scanf` 函数的一般形式为 `scanf(格式控制, 地址表列)`，其中地址表列应当是**变量地址**（`&变量名`），而不是变量名。输入 `int` 型的格式字符应使用 `%d`，输入 `float` 型的格式字符应使用 `%f`。


## 💡 核心考点

1. **输入输出概念**：以计算机主机为主体，从计算机向外输出为输出，从外向内为输入
2. **C语言本身不提供输入输出语句**，由库函数实现，需包含 `<stdio.h>`
3. **`getchar` 函数**：无参数，输入**一个字符**，只能接收一个字符
4. **`putchar` 函数**：输出**一个字符**，参数可以是字符常量或整型常量
5. **`printf` 函数**：`printf(格式控制, 输出表列)`，格式字符串中包含格式声明和普通字符
6. **`scanf` 函数**：`scanf(格式控制, 地址表列)`，地址表列为变量地址（`&变量名`）
7. **格式字符**：`%d`（十进制整数）、`%f`（实数）、`%c`（字符）、`%s`（字符串）
8. **`scanf` 注意事项**：变量名前加 `&`，数值输入时空格/回车/Tab作为结束标志
9. **缓冲区机制**：按 Enter 键后才将字符送入计算机
