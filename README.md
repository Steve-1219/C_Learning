## 📚 项目简介

本项目是一套完整的C语言程序设计学习资料库，涵盖从基础语法到高级应用的全部知识点。每章均包含知识梳理、代码示例和同步训练，适合初学者系统学习，也适合备考复习。

## 🛠️ 推荐软件

为获得最佳学习体验，推荐使用以下软件工具：

| 软件 | 用途 | 推荐理由 |
| :---: | :--- | :--- |
| **VS Code** | 代码编辑器 | 轻量、插件丰富、支持语法高亮和调试 |
| **Code::Blocks** | 集成开发环境 | 开箱即用，适合C语言初学者 |
| **Dev-C++** | 集成开发环境 | 体积小、启动快，适合新手入门 |
| **CLion** | 集成开发环境 | JetBrains出品，功能强大（付费） |
| **GCC** | C语言编译器 | 开源免费，Linux/macOS自带，Windows可用MinGW |
| **Typora** | Markdown阅读器 | 实时渲染，适合阅读笔记文档 |

### 安装建议

**方案一：VS Code + GCC（推荐）**

```bash
# 1. 安装 VS Code
# 2. 安装 C/C++ 插件（搜索 "C/C++" 安装）
# 3. Windows 用户安装 MinGW-w64，macOS/Linux 自带 GCC

# 验证 GCC 是否安装成功
gcc --version

# 编译运行示例
gcc hello.c -o hello
./hello
```

**方案二：Code::Blocks（新手推荐）**

直接下载安装包，内置 GCC 编译器，无需额外配置即可运行C程序。
